① Header
상단 헤더 영역으로 좌측 뒤로가기 버튼과 중앙 화면명 "교환 신청함"으로 구성됨
← 뒤로가기 (마이페이지로 이동) | 중앙: 화면명
② 탭 바
2개 탭으로 구성: "받은 제안 (N)" 활성화 상태 / "보낸 제안 (N)"
각 탭에 대기중인 제안 수가 카운트 뱃지로 표시됨. 탭 전환 시 URL 변경
③ 받은 제안 탭 (내 교환글별 그룹핑)
내 교환글별로 그룹핑된 제안 카드 리스트. 각 그룹 상단에 내 교환글 요약 카드(이미지+상품명+받은 제안 수)가 표시되고, 아래에 해당 글에 대한 제안 목록이 최신순으로 정렬됨
정렬 순서: 대기중 → 수락됨 → 거절됨/만료
④ 제안 카드 구조
각 제안 카드 포함 요소:
• 제안자 프로필 (32px 아바타 + 닉네임 + 신뢰등급 Lv.N + 평점 ★)
• 제안 굿즈 (60px 이미지 + 상품명 + 상태등급 S/A/B/C + 작품·캐릭터)
• 첨부 메시지 (있는 경우만 회색 말풍선으로 표시)
• 제안 시간 ("2시간 전" 형식)
• 상태 뱃지 (대기중=Amber / 수락됨=Green / 거절됨=Gray / 만료=Gray)
• 액션 버튼 (대기중일 때만): [수락] 파란 버튼 + [거절] 회색 아웃라인 + [채팅] 아이콘
⑤ 인터랙션 동작
각 액션의 동작 정의:
• [수락] 탭 → 확인 모달 → 수락 처리 → 채팅방 자동 생성
• [거절] 탭 → 거절 사유 바텀시트 → 거절 처리
• [채팅] 아이콘 → 채팅방(CHAT02)으로 이동
• 제안자 정보/제안 굿즈 이미지 탭 → 교환 상세(EXCH02) 이동
