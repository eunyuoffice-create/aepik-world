  • 각 카드 구성:
    • └ 이벤트 타입 태그 (작은 배경, 흰 텍스트: 생일카페 Pink #EC4899 / 전광판 Purple #8B5CF6)

    • └ 리스트 뷰: 좌측 이미지 placeholder (80x80), 우측 하트 아이콘

[MapView] 지도 뷰

  • "이 지역 검색" 버튼 (중앙 상단, gray bg)
  • Bottom sheet: 선택된 이벤트 카드 (이미지 + 이름 + 시간 + 거리 "1.2km" + [상세 보기] 버튼)
  • 현재 위치 FAB: 지도 우하단, 흰 원형 + 위치 아이콘
  • Tap pin / bottom sheet [상세보기] → EVNT02 진입

[FAB] 이벤트 등록 FAB

[EventForm] 이벤트 등록 폼
  • 백 버튼 + "이벤트 등록" 제목
  • 필수(*) 필드: 이벤트 유형 / 작품/아이돌 / 캐릭터/멤버 / 이벤트명 / 시작~종료일 / 장소
  • 선택 필드: 운영 시간 / 상세 설명 / 대표 이미지(max 3) / 외부 링크
  • 이벤트 유형: dropdown (생일카페, 전광판, 팬사인회 등)
  • 작품/캐릭터: 검색 input (search icon + placeholder)
  • 날짜: date picker row (시작일 ~ 종료일)
  • 장소: 주소 input + 미니 지도 플레이스홀더
  • 대표 이미지: 3개 upload boxes (+)
  • Bottom: [등록하기] 풀너비 파란 버튼 → 바로 publish (승인 과정 없음, Phase 1)
