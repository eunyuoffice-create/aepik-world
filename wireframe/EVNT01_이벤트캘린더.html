EVNT01 — 이벤트 캘린더
라우트: /events  |  비로그인 접근: 가능  |  Phase 1.5
1
[Header] 앱 헤더
  • "이벤트" 제목 좌측 표시
  • 우측 아이콘: Search (SRCH01 이동)
  • 상단 고정 (스크롤 시 유지)
2
[Banner] 곧 다가올 이벤트 배너
  • D-7 이내 시작 이벤트 중 찜 수 상위 5건 노출
  • 배너 높이: 140px, 그래디언트 오버레이 + 배경 이미지
  • 구성: D-day 뱃지 (빨강/주황, 작은 pill) + 이벤트명 + 장소 (흰색 텍스트)
  • 자동 슬라이드: 5초 간격, 터치 스와이프 지원
  • 하단 인디케이터: ● ○ ○ (활성 = White)
  • 탭 시: EVNT02 (이벤트 상세) 이동
  • 데이터 없을 시: 배너 숨김
  • 배너/팝업 신청 스트립 (배너 바로 아래)
    • 구성: 아이콘(32px, Blue 배경) + 안내 텍스트 + 슬롯 상태 + [신청] 버튼
    • 안내 텍스트: "내 이벤트도 배너/팝업으로 홍보하기"
    • 슬롯 상태 표시 (API: GET /api/banner/slots):
      • 여유 (>50%): Green 도트 + "N슬롯 남음"
      • 마감 임박 (25~50%): Amber 도트 + "슬롯 거의 마감"
      • 매진 (0%): Red 도트 + "슬롯 마감" + 버튼 비활성(회색)
    • [신청] 버튼 탭 시: BANR01 (배너/팝업 신청) 이동
    • 매진 시: 버튼 "마감" (회색, 탭 불가)
    • 캘린더 뷰·리스트 뷰에서 노출, 지도 뷰에서는 미노출
3
[Tabs] 뷰 토글 탭
  • 3가지 뷰: 캘린더 | 리스트 | 지도
  • 활성 탭: Blue (#2F8EC1) 밑줄 (2px)
  • 비활성: Gray (#8EA2B7)
  • 탭 전환 시 필터 상태 유지
4
[FilterChips] 필터 바
  • 필터 칩: 작품 | 이벤트 유형 | 지역 (모두 드롭다운 ▾)
  • 비활성: Gray 배경, 활성: Blue 배경 + 흰색 텍스트
  • 가로 스크롤 가능
  • 필터 선택 → 목록 실시간 갱신
  • 초기화 버튼 추가 가능 (필터 활성화 시)
5
[MonthNav] 월 네비게이션
  • 형식: ‹ YYYY년 M월 › (좌우 화살표로 전후월 전환)
  • [오늘] 버튼: Blue 배경, EFF6FF border, Blue text
  • 클릭 시 현재 날짜로 이동
  • 모바일: 월 네비게이션만 노출
  • PC: 풀 월간 그리드 캘린더 표시 (반응형 분기)
6
[DateScroller] 날짜 스크롤러 (모바일)
  • 가로 스크롤 가능한 날짜 바
  • 각 날짜 셀 구성:
    • 요일 라벨 (월,화,수,...): 10px Gray #8EA2B7
    • 날짜 숫자: 32×32px 원형, 12px Bold
    • 오늘 날짜: Blue 배경 (#69BFEA) + 흰색 텍스트
    • 선택된 날짜: Blue 테두리 (1px)
    • 아래 카테고리 도트 (최대 4개): Purple/Pink/Amber/Cyan 4px
  • 클릭 → 선택 날짜로 이벤트 목록 갱신
  • 약 7개 날짜 한번에 노출, 좌우 스크롤
7
[DateHeader] 선택 날짜 헤더
  • 형식: "M월 D일 (요일)" (예: 3월 15일 (토))
  • 우측: 이벤트 수 뱃지 "N건" (Blue 배경, 10px White)
  • 배경: Light Gray #F7FAFD
8
[EventCards] 이벤트 카드 (리치 포맷)
  • 카드 레이아웃: 가로 (이미지 좌측 + 정보 우측)
  • 이미지: 72×72px, border-radius: 8px, Gray 배경
  • 정보 섹션:
    • 태그행: 이벤트 유형 태그 (생일카페/팬미팅/LED트럭 등) + D-day 뱃지
    • 유형 태그: 카테고리별 소프트 배경색 + 텍스트 (10px)
    • D-day: Red 배경 + White 텍스트 (9px Bold)
    • 제목: 13px Semibold, 1줄
    • 위치: MapPin 아이콘 + 텍스트 (11px Gray)
    • 시간: Clock 아이콘 + 텍스트 (11px Gray, HH:MM ~ HH:MM)
  • 우측 부동: 하트 아이콘 (찜 상태 변경)
  • 카드 탭 시: EVNT02 (이벤트 상세) 이동
  • 선택 날짜 기준 최대 10건 노출
9
[BottomNav] 하단 네비게이션
  • 5개 탭: 홈 | 교환 | 이벤트(활성) | 채팅 | 마이
  • 비활성: Gray #8EA2B7, 활성: Blue #2F8EC1
  • 채팅/마이: 비로그인 시 로그인 유도
  • 하단 고정
10
[FAB] 이벤트 등록 버튼
  • 위치: 우하단 고정 (bottom: 80px, right: 16px)
  • 크기: 48×48px, Blue 배경, + 아이콘
  • 탭 시: EVNT03 이벤트 등록 페이지로 이동
  • 비로그인 시: 로그인 유도 팝업
화면 상태
  • 기본 상태: 이벤트 데이터 있음, 배너 노출, 필터 가능
  • 다가올 이벤트 없음: 배너 숨김, 하단 텍스트 "다가올 이벤트가 없습니다"
  • 선택 날짜 이벤트 없음: 날짜 헤더만 표시, "이 날짜에 이벤트가 없습니다" 메시지
  • 필터 결과 없음: "검색 결과가 없습니다" 메시지 + 필터 초기화 버튼
  • 에러 상태: "데이터를 불러올 수 없습니다" + 재시도 버튼
  • 비로그인: 열람 가능, 이벤트 등록 시 로그인 유도
  • 지도 위치 권한 거부: "위치 권한을 허용해주세요" 메시지 + 설정 열기 버튼
