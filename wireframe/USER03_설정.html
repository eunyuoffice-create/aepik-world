USER03 — 설정
라우트: /my/settings, /my/keywords  |  로그인 필수  |  Phase 1
1
[Header] 설정 헤더
  • 좌측: 뒤로가기 화살표 (USER01 마이페이지로 복귀)
  • 중앙: "설정" 타이틀
  • 상단 고정
2
[Toggle] 알림 설정
  • 7개 토글 스위치:
    • 푸시 알림 (마스터, ON)
    • 교환 매칭 알림 (ON)
    • 키워드 알림 (ON)
    • 공구 알림 (ON)
    • 채팅 알림 (ON)
    • 이벤트/마케팅 알림 (OFF)
  • 푸시 알림 OFF 시 하위 항목 비활성화 (그레이 처리)
  • 변경 시 즉시 PATCH API 호출
3
[Time] 방해금지 시간
  • 시작/종료 시간 텍스트 표시 (예: 23:00 ~ 08:00)
  • 탭 시 시간 피커 바텀시트 표시
  • 해당 시간대에 FCM 발송 지연 (다음 활성 시간에 일괄)
4
[Category] 관심 카테고리
  • 현재 선택된 카테고리 태그 표시 (Purple/Pink/Amber/Cyan)
  • "변경" 텍스트 버튼 — 탭 시 카테고리 선택 바텀시트 (AUTH02 Step 4 패턴)
  • 선택 없음 시: 안내 메시지 표시
5
[Menu] 계정 관리
  • 5개 메뉴 항목:
    • 비밀번호 변경 — 바텀시트: 현재 비밀번호 + 새 비밀번호 (강도 바) + 확인
    • 연결된 소셜 계정 — 카카오 연동 상태 표시, 부제 "카카오 연동"
    • 키워드 알림 설정 — 등록 키워드 수 뱃지 (예: 3), 탭 시 Frame B 이동
    • 로그아웃 — 확인 팝업
    • 회원 탈퇴 — Danger 텍스트 (빨강), 탭 시 Frame C Step 1 진입
6
