GRPB01 공구 목록
공동구매 목록을 탐색하고 필터링하는 화면
1
[Header] 페이지 헤더
  • 좌측: 페이지 제목 "공구" (고정 텍스트)
  • 우측: 검색 아이콘 (Q)
    • 탭 시: 검색 페이지(SRCH01) 이동
  • 스크롤 시 상단 고정 (sticky)
2
[CategoryFilter] 카테고리 필터
  • 가로 스크롤 형태 (모바일에서 좌우 스와이프)
  • 필터 항목: 전체 / 애니/만화/소설 / 아이돌/아티스트 / 배우/무대 / 게임
  • 선택된 필터: Blue 배경 + 흰색 텍스트
  • 각 카테고리 탭 시: 선택 + 목록 필터링 (단일 선택)
3
[TabBar] 상태 탭 바
  • 모집중 (기본 활성) | 진행중 | 완료·무산
  • 모집중: status=recruiting인 공구만 표시
  • 진행중: status=pending_decision, confirmed, shipping인 공구만 표시
  • 완료·무산: status=completed, cancelled인 공구만 표시
  • 활성 탭: Blue 하단 보더 + 텍스트 컬러
  • 탭 탭 시: 목록 리로드
4
[FilterChips] 필터 칩 바
  • 가로 스크롤 칩 형태 (모바일에서 좌우 스와이프)
  • 필터 항목:
    • 작품 ▾ — 작품 DB 검색 선택
    • 굿즈 타입 ▾ — 포토카드/아크릴/캔뱃지/피규어/인형/엽서/의류/기타
    • 공구 유형 ▾ — 일반 / 세트
  • 칩 탭 시: 바텀시트 → 옵션 선택 → 칩에 선택값 표시
  • 선택된 칩: Blue 배경 + 흰색 텍스트
  • 초기화 버튼: 선택된 필터가 있을 때만 표시
5
[SortBar] 정렬 + 결과 수
  • 좌측: "전체 N개" — 필터 적용 결과 수
  • 우측: 정렬 드롭다운
    • 최신순 (기본) — created_at DESC
    • 인기순 — 조회수 + 참여 수 기반
    • 마감임박순 — deadline 기준 오름차순
  • 정렬 선택 변경 시 목록 즉시 리로드
6
[List] 공구 카드 리스트 (1열)
  • 1열 리스트 형태 (NOT 그리드 — 세로 스크롤만)
  • 카드: GroupBuyCard 컴포넌트
    • 이미지 (16:9 비율, object-fit: cover)
    • 태그: [공구] or [세트공구] + [카테고리]
    • 상품명 (최대 2줄, text-overflow: ellipsis)
    • 진행자 닉네임 + 신뢰배지 (Lv.1/Lv.2/Lv.3)
    • 참여현황 N/M명 (%) + 프로그레스바
      • 80% 이상 = 녹색 (#16A34A)
      • 미만 = 파란색 (#3B82F6)
    • 마감일 D-N (디데이 표시)
    • 가격 ₩15,000/1인 (1인당 가격)
  • 상태별 뱃지 표시:
    • 진행 결정 대기: "진행 결정 대기" 뱃지 (Amber #FCD34D)
    • 진행 확정: "진행 확정" 뱃지 (Green #86EFAC)
    • 배송중: "배송중" 뱃지 (Blue #BFDBFE)
    • 완료: 뱃지 없음 (Gray 처리)
    • 무산: "무산" 뱃지 (Red #FCA5A5) + 취소선 + opacity 0.6
  • 목표 미달 공구: "목표 미달" 뱃지 표시
  • 무한 스크롤 (Intersection Observer)
    • 하단 도달 시 다음 페이지 자동 로드
    • 로딩 중 스켈레톤 카드 표시
  • 카드 탭 시: 공구 상세(GRPB02) 이동
  • 카드 호버: shadow-md + translateY(-2px)
7
[FAB] 공구 등록 버튼
  • 우하단 고정 (bottom: 80px, right: 16px)
  • 56×56px, border-radius: 50%, Blue 배경
  • 아이콘: + (플러스)
  • 탭 시: 공구 등록(GRPB03) 이동
    • 비로그인: 로그인 유도 바텀시트
    • 미인증: 본인인증 유도 바텀시트
    • 미자격(거래 5회 미만/신뢰도 90% 미만): 등록 불가 안내
  • 스크롤 다운 시: 축소 or 반투명 (선택)
8
[BottomNav] 하단 네비게이션
  • 5개 탭: 홈(활성) | 공구 | 이벤트 | 채팅 | 마이
  • 활성 탭: Blue (#2563EB), 비활성: Gray 400
  • 공구는 별도 탭 (홈 탭과 분리됨)
  • 채팅/마이: 비로그인 시 로그인 유도
  • 채팅: 안 읽은 메시지 뱃지 (빨간 dot)
