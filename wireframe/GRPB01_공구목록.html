GRPB01 — 공구 목록
라우트: /groupbuy  |  접근: 전체  |  Phase 1
1
[Header] 페이지 헤더
  • 좌측: 페이지 제목 "공구" (고정 텍스트)
  • 우측 아이콘: Q (검색)
    • 탭 시: 검색 페이지(SRCH01) 이동
  • 상단 고정 (sticky) — 스크롤 시 유지
2
[CategoryFilter] 카테고리 필터
  • 가로 스크롤 형태 (모바일에서 좌우 스와이프)
  • 필터 항목: 전체 / 애니/만화/소설 / 아이돌/아티스트 / 배우/무대 / 게임
  • 선택된 필터: Blue 배경 + 흰색 텍스트
  • 각 카테고리 탭 시: 선택 적용 + 목록 필터링 (단일 선택)
3
[TabBar] 상태 탭 바
  • 3개 탭: 모집중 (기본 활성) | 진행중 | 완료·무산
  • 모집중: status=recruiting인 공구만 표시
  • 진행중: status=pending_decision, confirmed, shipping인 공구 표시
  • 완료·무산: status=completed, cancelled인 공구 표시
  • 활성 탭: Blue 하단 보더 + 텍스트 컬러
  • 탭 탭 시: 목록 리로드
4
[FilterChips] 필터 칩 바
  • 가로 스크롤 칩 형태 (모바일에서 좌우 스와이프)
  • 필터 항목:
    • 작품 — 작품 DB 검색 선택
    • 굿즈 타입 — 포토카드/아크릴/캔뱃지/피규어/인형/엽서/의류/기타
    • 공구 유형 — 일반 / 세트
  • 칩 탭 시: 바텀시트 → 옵션 선택 → 칩에 선택값 표시
  • 선택된 칩: Blue 배경 + 흰색 텍스트
5
[SortBar] 정렬 + 결과 수
  • 좌측: "전체 N개" — 필터 적용 결과 수
  • 우측: 정렬 드롭다운
    • 최신순 (기본) — created_at DESC
    • 인기순 — 조회수 + 참여 수 기반
    • 마감임박순 — deadline 오름차순
  • 정렬 선택 변경 시 목록 즉시 리로드
6
[List] 공구 카드 리스트 (1열)
  • 1열 리스트 형태 (세로 스크롤만, NOT 그리드)
  • 카드: GroupBuyCard 컴포넌트
    • 이미지 (16:9 비율, object-fit: cover)
    • 태그: [공구] 또는 [세트공구] + [카테고리]
    • 상품명 (최대 2줄, text-overflow: ellipsis)
    • 진행자 닉네임 + 신뢰배지 (Lv.1/Lv.2/Lv.3)
    • 프로그레스바: N/M명 (%) — 80% 이상 Green, 미만 Blue
    • 마감일: D-N (디데이 표시)
    • 가격: N/1인 또는 N/세트
  • 상태별 뱃지 표시:
    • 진행 결정 대기: Amber 뱃지
    • 진행 확정: Green 뱃지
    • 배송중: Blue 뱃지
    • 무산: Red 뱃지 + 취소선 + opacity 0.6
  • 무한 스크롤 (Intersection Observer) — 하단 도달 시 다음 페이지 로드
  • 로딩 중: 스켈레톤 카드 표시 (회색 박스 펄스 애니메이션)
  • 카드 탭 시: 공구 상세(GRPB02) 이동
  • 카드 호버: shadow-md + 약간의 translateY 이동
7
[FAB] 공구 등록 버튼
  • 우하단 고정 (bottom: 80px, right: 16px)
  • 크기: 48×48px, border-radius: 50%
  • 색상: Blue 배경 + 흰색 +
  • 탭 시: 공구 등록(GRPB03) 이동
    • 비로그인: 로그인 유도 팝업
    • 미인증: 본인인증 유도 팝업
    • 미자격 (거래 5회 미만/신뢰도 90% 미만): 등록 불가 안내
  • 스크롤 다운 시: 축소 또는 반투명 (옵션)
8
[BottomNav] 하단 네비게이션
  • 5개 탭: 홈 | 공구(활성) | 이벤트 | 채팅 | 마이
  • 활성 탭: Blue (#2563EB), 비활성: Gray 400
  • 공구: 별도 탭으로 홈과 분리됨
  • 채팅/마이: 비로그인 시 로그인 유도
  • 채팅: 안 읽은 메시지 뱃지 표시 (빨간 dot)
  • 하단 고정
화면 상태
  • 기본 상태: 필터 적용된 공구 목록 표시
  • 로딩 상태: 스켈레톤 카드 펄스 애니메이션
  • 빈 상태: "조건에 맞는 공구가 없어요" + [공구 등록하기] CTA
  • 에러 상태: "네트워크 오류" + [다시 시도] 버튼
  • 비로그인: 목록 조회 가능, FAB 탭 시 로그인 유도
