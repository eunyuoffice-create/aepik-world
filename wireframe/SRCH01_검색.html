SRCH01 검색
작품, 캐릭터, 굿즈를 통합 검색하고 필터링하는 화면
1
[Header] 검색 헤더 (입력 모드)
  • 좌측: 뒤로가기 버튼 (←)
  • 중앙: 검색 인풋
    • 44px 높이, 라운딩 10px, 파란색 테두리 2px
    • 플레이스홀더: "작품, 캐릭터, 굿즈 검색"
    • 자동 포커스 (진입 시 키보드 올라옴)
  • 우측: 입력 값 지우기 버튼 (X)
2
[ChipGroup] 최근 검색어
  • 헤더: "최근 검색어" 제목 + "전체 삭제" 링크
  • Pill 칩 형태 (배경 Gray 100 #F0F0F0)
  • 최대 10개 표시
  • 개별 X 버튼으로 삭제 가능
  • "전체 삭제" 탭 시 모든 검색어 제거
  • 탭 시: 해당 검색어로 검색 실행 → 결과 화면 전환
3
[ChipGroup] 인기 검색어
  • 헤더: "인기 검색어" + "24시간 갱신" 타임스탐프
  • Pill 칩 형태 (배경 Blue 50 #EFF6FF, 텍스트 Blue 900)
  • 순위 번호 표시 (1~5번)
  • 24시간마다 자동 갱신 (서버에서)
  • 탭 시: 해당 검색어로 검색 실행
-
[Dropdown] 자동완성 (입력 중)
  • 2자 이상 입력 시 드롭다운 표시
  • 최대 5개 추천 검색어
  • 일치하는 키워드는 Bold + Blue 강조
  • 최근/인기 검색어에서 우선 추천
  • 탭 시: 검색 실행
5
[Tabs] 거래 유형 필터 탭
  • 4개 탭: 전체 | 교환 | 공구 | 구함
  • 각 탭에 결과 건수 표시
  • 활성 탭: 파란색 텍스트 + 하단 인디케이터 (2px)
  • 탭 전환 시 인디케이터 슬라이드 애니메이션
  • 탭 선택 시: 결과 목록 즉시 필터링
6
[ChipBar] 필터 & 정렬 칩 바
  • 가로 스크롤 칩 바
  • 항목:
    • "필터" 칩 (필터 아이콘 + 텍스트) → 탭 시 바텀시트 올라옴
    • 선택된 카테고리 칩 (파란색 배경) + X로 삭제 가능
    • "최신순 ↓" 등 정렬 옵션
    • "거래 가능" 등 상태 필터
  • 칩 탭 시: 필터값 변경 → 결과 즉시 업데이트
8
[CardList] 검색 결과 카드 (리스트형)
  • 리스트형 카드 (그리드 아님)
  • 구조: 좌측 이미지(80×80) + 우측 정보
    • 이미지: 1:1 비율, object-fit: cover
    • 태그: [거래유형] + [카테고리] (필터 칩과 동일 스타일)
    • 상품명 (1~2줄)
    • 시간 + 조회수 (Gray 400)
  • 무한 스크롤 (Intersection Observer)
  • 카드 탭 시: 교환상세(EXCH02) 또는 공구상세(GRPB02) 이동 (거래유형별)
9
[BottomSheet] 필터 바텀시트
  • 배경: 어두운 오버레이 (rgba 0,0,0,0.4)
  • 시트: 하단 border-radius 16px 16px 0 0
  • 드래그 핸들 (40×4px, 상단)
  • 헤더: "필터" 제목 + "초기화" 링크
  • 필터 그룹 (각 padding 16px):
    • 카테고리 — 4개 칩 (Purple selected)
    • 굿즈 타입 — 5개 칩
    • 상품 상태 — 4개 칩 (S/A/B/C급)
    • 거래 방법 — 3개 칩 (택배/직거래/둘다)
    • 정렬 — 3개 칩 (최신/인기/조회)
    • 추가 옵션 — 2개 체크박스 (차액협의, 거래가능)
  • 하단 버튼 (2개):
    • "초기화" — Ghost 스타일 (회색 테두리)
    • "N건 보기" — Primary Blue 스타일
검색 결과 없음 상태
  • 중앙 정렬된 콘텐츠
  • 큰 검색 아이콘 (40px, 회색)
  • "검색 결과가 없어요" (14px, 600 weight)
  • "다른 키워드로 검색해 보세요" (12px, Gray 500)
  • 선택 해제 후 새로 검색하기 유도

화면 상태 체크리스트
초기 진입: 검색 인풋 포커스 + 최근/인기 검색어 표시
입력 중: 2자 이상 시 자동완성 드롭다운 표시
검색 완료: 결과 탭 + 필터 칩 + 카드 목록
필터 활성: 바텀시트 올라옴 (z-index 40)
빈 결과: 검색 아이콘 + 안내 메시지 + 다시 검색 유도
로딩: 무한 스크롤 하단 로딩 시 스켈레톤 카드 표시
필터 선택: 칩에 배경색 변경 + "N건 보기" 버튼 활성화
