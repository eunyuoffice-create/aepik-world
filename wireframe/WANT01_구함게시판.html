화면 기능 설명
1
[Header] 앱 헤더
  • "구함" 타이틀
  • 검색 아이콘 (SRCH01으로 이동)
2
[Tab] 탭 바
  • 전체 (기본값, 모든 사용자 구함글 표시)
  • 내 구함 (로그인 사용자의 구함글만, 비로그인 시 로그인 유도)
  • 2px 파란 인디케이터 표시
3
[Filter] 필터 바
  • 작품 / 캐릭터 / 굿즈 타입 / 상태 필터
  • 각 필터는 바텀시트 형식으로 다중 선택 가능
  • 적용된 필터에는 bold 또는 하이라이트 표시
  • 초기화 버튼으로 모든 필터 리셋
4
[Sort] 정렬 바
  • 전체 구함글 N개 표시
  • 정렬 옵션: 최신순, 만료임박순
  • 기본값: 최신순
5
[List] 구함 카드 리스트
  • 좌측: 레퍼런스 이미지 (80x80, 둥근모서리 8px, 회색 플레이스홀더)
  • 우측: 구함 정보
    • 상태 뱃지: 구함중(green #22C55E) / 매칭됨(blue #3B82F6) / 만료(gray #9CA3AF)
    • 카테고리 태그: 애니 Purple / 아이돌 Pink / 배우 Amber / 게임 Cyan
    • 제목 (14px, bold, 2줄 max)
    • 작품명 · 캐릭터명 (회색)
    • 굿즈 타입 (소형 회색)
    • 등록일 · D-day (소형 회색)
    • 우측 하단: "매칭 N건" (파란색, bold)
  • 끌어올리기 상태면 우상단에 "UP" 뱃지 (Amber bg, 소형)
  • 무한 스크롤로 추가 로딩
6
[BottomSheet] 구함 상세 바텀시트
  • 구함글 선택 시 하단에서 위로 슬라이드 (60-70% 높이)
  • 핸들 바 (4px, 회색)
  • 닫기 버튼 (우상단 X)
  • 구함 제목 (18px, bold)
  • 작품명 · 캐릭터명 · 굿즈 타입 태그들
  • 레퍼런스 이미지 영역 (플레이스홀더)
  • 상세 설명 텍스트
  • 등록일 / 만료일 (D-day 뱃지 포함)
  • 교환 조건 섹션
  • 매칭된 교환글 섹션 (최대 5건 미리보기, 교환글 카드)
  • 액션 버튼:
    • 타인의 구함: "교환 제안하기"(primary) + "채팅 문의"(secondary)
    • 본인의 구함: "수정"(secondary) + "끌어올리기"(secondary) + "삭제"(secondary)
7
[BottomSheet] 구함글 등록 바텀시트
  • FAB 버튼(+) 또는 화면 상단 CTA로 진입 (75-80% 높이)
  • 비로그인 시 로그인 유도 팝업
  • 핸들 바 + 닫기 버튼
  • 입력 필드:
    • 구함 제목 (input, placeholder 안내)
    • 작품 선택 (dropdown + 검색)
    • 캐릭터 선택 (dropdown + 검색)
    • 굿즈 타입 (chip selector: 포토카드/아크릴스탠드/피규어/봉제인형/포스터/기타, 단일 선택)
    • 상세 설명 (textarea)
    • 레퍼런스 이미지 (최대 3장, 업로드 영역)
    • 교환 조건 (toggle/checkbox)
    • 만료 기간 (chip selector: 7일/14일/30일/60일, 30일 기본값)
  • "구함글 등록" 풀-너비 파란 버튼
  • 등록 즉시 매칭 알림 발송 (매칭된 교환글이 있으면)
8
[FAB] 플로팅 액션 버튼
  • 우하단 56px 파란 원형 버튼
  • + 아이콘
  • 클릭 시 구함글 등록 바텀시트 오픈 (Frame 3)
  • 비로그인 시 로그인 유도 팝업
9
[Navigation] 하단 네비게이션
  • 5탭: 홈 / 교환 / 이벤트 / 채팅 / 마이
  • 각 아이콘 20x20px (Lucide SVG)
  • 구함은 하단탭에 미포함 (진입 경로: 마이 → 내활동 또는 검색에서)
  • 활성 탭: 아이콘 + 텍스트 파란색
  • 비활성 탭: 아이콘 + 텍스트 회색
10
[State] 화면 상태
  • 정상: 구함글 목록 표시
  • 빈 상태 (전체 탭): "등록된 구함글이 없습니다" + 아이콘
  • 빈 상태 (내 구함 탭): "작성한 구함글이 없습니다" + FAB 유도
  • 로딩: 스켈레톤 카드 3-4개
  • 에러: "데이터를 불러올 수 없습니다" + 재시도 버튼
  • 비로그인 (내 구함 탭): "로그인 후 내 구함글을 확인하세요" + 로그인 버튼
11
