CHAT02 채팅 대화방
교환/공구 상대와 실시간으로 대화하며 거래 프로세스를 진행하는 채팅 화면
1
[Sub Header] 채팅 헤더
  • 좌측: 뒤로가기 아이콘 (`ChevronLeft`) — CHAT01로 복귀
  • 상대 프로필: 32px 아바타 + 닉네임 (14px SemiBold) + 온라인 도트
    • 공구 채팅방: 공구명 + "참여자 N/M명" 표시
  • 상품명 12px `#45617E` — 탭 시 교환/공구 상세로 이동
  • 우측: 더보기 아이콘 (`MoreVertical`) > 신고/알림 끄기/나가기
  • 높이 56px, 하단 보더 1px `#EEF3F8`
2
[Banner] 거래 상태 배너
  • 채팅방 상단 고정, 현재 거래 진행 상태 표시
  • 8개 상태 변형:
    • 교환 협의 중 — 배경 `#EAF7FF`, Blue
    • 교환 확정 요청 — 배경 `#FEF3C7`, Amber
    • 교환 확정 완료 — 배경 `#D1FAE5`, Green
    • 발송 완료(내가) — 배경 `#EAF7FF`, Blue
    • 양쪽 발송 완료 — 배경 `#D1FAE5`, Green
    • 교환 완료 — 배경 `#EEF3F8`, Gray
    • 공구 모집중 — 배경 `#EAF7FF`, Blue
    • 공구 진행확정 — 배경 `#D1FAE5`, Green
  • 높이 auto (최소 44px), 탭 시 거래 타임라인 바텀시트 열기
3
[Messages] 메시지 영역
  • 보낸 메시지 (우측): 배경 `#69BFEA`, white 텍스트, 라운딩 16/16/4/16, 최대 너비 70%
  • 받은 메시지 (좌측): 배경 `#EEF3F8`, `#163958` 텍스트, 라운딩 16/16/16/4, 32px 아바타
  • 시스템 메시지: 중앙 정렬, 배경 `#EEF3F8`, 라운딩 999px, 12px `#45617E`
  • 이미지 메시지: 최대 200px 썸네일, 라운딩 12px, 탭 시 전체화면 뷰어
  • 날짜 구분선: "2026년 3월 6일 목요일" 형식, 좌우 점선
  • 메시지 그룹핑: 동일 발신자 연속 시 첫 메시지만 아바타+닉네임, 5분 이내 연속 시 마지막만 타임스탬프
  • 메시지 롱프레스: 복사 / 삭제(내 메시지, 5분 이내)
4
[Trade Card] 교환 확정 카드
  • 인라인 카드: 보더 1px `#DEE8F2`, 라운딩 12px, 패딩 16px
  • 상품 이미지 60x60px + 상품명 + 교환 조건 요약 ("내 OO <> 상대 OO")
  • 확정 버튼 (Primary) + 거절 버튼 (Ghost)
    • 양쪽 모두 확정 클릭 시 교환 확정 완료
    • 확정됨 상태: 버튼 비활성 + "확정 완료" Green 뱃지
5
[Trade Card] 운송장 입력 카드
  • 교환 확정 후 표시, 보더 1px `#DEE8F2`, 라운딩 12px
  • 택배사 선택 (셀렉트): CJ대한통운/한진/롯데/우체국/기타
  • 운송장 번호 인풋: 숫자 키보드
  • "발송 완료" Primary 버튼
  • 입력 완료 상태: 택배사 + 운송장 표시 + "배송 조회" 링크
6
[Trade Card] 수취 확인 / 교환 완료 카드
  • "상품을 수령하셨나요?" 안내 + "수취 확인" Primary 버튼
  • 한쪽만 확인 시 5일 후 자동 완료 안내 텍스트
  • 완료 후: 시스템 메시지 "교환이 완료되었습니다" + 후기 작성 CTA
    • CTA: Green 배경, 별 아이콘 + "거래 후기 작성하기"
7
[Input Bar] 메시지 입력 바
  • 좌측: + 버튼 > 이미지 첨부 바텀시트 (카메라/앨범, 최대 3장)
  • 중앙: 인풋 필드 (라운딩 20px, 배경 `#EEF3F8`, 멀티라인 최대 5줄)
  • 우측: 전송 버튼 (`Send`) — 비활성 `#DEE8F2`, 활성 `#69BFEA`
  • 높이 auto (최소 48px), safe-area 대응
  • 거래 완료 후: 입력 비활성 → "거래가 완료되어 메시지를 보낼 수 없어요"
