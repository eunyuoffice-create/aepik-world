1
[Header] 앱 헤더
  • 좌측: `ChevronLeft` 뒤로가기
  • 중앙: "배너/팝업"
  • 우측: 없음
2
[Section] 현재 진행 중 배너/팝업
  • 섹션 헤더: "지금 진행 중" + 전체 건수 뱃지 (파란 pill)
  • 가로 스크롤 카드 리스트 (최대 10건, 최신순)
  • 카드 구성:
    • 이미지 썸네일 (카드 너비 x 56px, 통이미지)
    • 이벤트명 (1줄, ellipsis)
    • 기간: "3/8 ~ 3/14" (시작일~종료일)
    • 유형 뱃지: "메인 배너" / "피드 배너" / "팝업" (pill)
    • 카테고리 컬러 도트 (Purple/Pink/Amber/Cyan)
  • 카드 탭 시: 해당 배너/팝업 풀뷰 이미지 (바텀시트 또는 모달)
  • 진행 중 없을 시: "현재 진행 중인 배너/팝업이 없어요" (빈 상태)
3
[CardList] 슬롯 현황 카드 (유형별)
  • 3개 카드 세로 나열: 메인 배너 / 피드 배너 / 팝업
  • 각 카드 구성:
    • 유형 아이콘 (36px, 컬러 배경) + 유형명 + 노출 위치 설명
    • 이미지 규격: "1200x300px 통이미지" 등
    • 슬롯 프로그레스 바: 사용/전체 (예: 5/8)
    • 잔여 텍스트: "3슬롯 남음" (Green) / "1슬롯 남음" (Amber) / "마감" (Red)
    • 가격 표시 (Phase 1.5~): "5,000~10,000원/일"
  • 신청 가능 시: [신청하기] Primary 버튼 (활성) -> BANR02 이동
  • 마감 시: [마감] 버튼 (회색, 비활성) + "다음 신청: MM/DD~" 안내
  • 신청 버튼 탭: BANR02 이동, type 파라미터 전달 (main_banner / feed_banner / popup)
  • 통이미지 정책: 배너/팝업은 모두 이미지 1장으로만 구성, 텍스트 오버레이 없음
4
[Info] 다음 신청 기간 안내
  • 마감된 유형이 있을 때만 노출 (모든 유형 여유 시 미노출)
  • 파란 배경 안내 카드: `Clock` 아이콘 + "다음 신청 기간"
  • 마감된 유형별 다음 신청 가능일 표시
  • 예: "팝업 : 3/15(월)부터 신청 가능"
  • 알림 토글: "신청 가능할 때 알려드릴까요?" -> FCM 등록/해제
  • 전체 마감 시: 안내 카드 강조 (파란 테두리 추가)
5
[Link] 내 신청 내역 바로가기
  • "내 배너/팝업 관리 >" 링크 -> USER02 (내 활동 관리) 이동
  • 최근 신청 1건 미리보기 (있을 경우):
    • 이미지 썸네일 + 유형 + 상태 뱃지
    • 상태: 대기중 / 검수중 / 승인 / 수정요청 / 반려 / 게시중 / 종료
  • 신청 내역 없을 시: 미리보기 영역 미노출, 링크만 표시
