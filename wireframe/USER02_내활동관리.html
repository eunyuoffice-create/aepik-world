USER02 — 내 활동 관리
라우트: /my/*  |  로그인 필수  |  Phase 1
1
[Header] 헤더
  • 좌측: 뒤로가기 화살표 (마이페이지 USER01로 복귀)
  • 중앙: "내 활동" 제목
  • 상단 고정
2
[TabBar] 탭 바
  • 6개 탭: 교환글 / 공구 / 동행글 / 내역 / 후기 / 찜
  • 가로 스크롤 가능
  • 활성 탭: 파란색 하단 인디케이터 + Bold
  • 각 탭 우측에 카운트 뱃지 (예: "교환글 3")
  • 탭 탭 시: 해당 탭 콘텐츠 전환
3
[Filter] 상태 필터 칩
  • 교환글 탭: 전체 / 교환중 / 제안 대기 / 완료
  • 공구 탭: 전체 / 모집중 / 진행 결정 대기 / 진행확정 / 배송중 / 완료 / 무산
  • 선택된 칩: Primary(Blue) 배경 + 흰 텍스트
  • 탭 시: 해당 상태로 목록 필터링
4
[Card] 교환글/공구 카드
  • 가로 레이아웃: 이미지(80×80) + 정보 구역
  • 상품명: 14px Bold, 최대 2줄 (ellipsis)
  • 작품명 · 캐릭터명 또는 카테고리: 12px Gray 500
  • 상태 뱃지: 교환중(Blue) / 제안 N건(Amber) / 완료(Green) / 무산(Red)
  • 메타 정보: 등록일 / 마감일 (D-N)
  • 공구 카드: 프로그레스 바 + "N/M명" 참여 현황
  • 우측: 더보기(⋯) 아이콘 → 액션시트 (끌어올리기/수정/삭제)
  • 카드 탭 시: 상세 페이지 이동 (EXCH02 / GRPB02)
5
[Card] 동행글 카드
  • 가로 레이아웃: 이미지(80×80) + 정보 구역
  • 상품명: 14px Bold, 최대 2줄 (ellipsis)
  • 작품명 · 캐릭터명: 12px Gray 500
  • 상태 뱃지: 활성(Blue) / 만료(Gray)
  • 등록일 · 만료예정일 D-N: 10px Gray
  • 매칭 알림 뱃지: "매칭 N건" (Blue bg)
  • 우측: 더보기(⋯) 아이콘 → 액션시트 (수정/삭제/기간연장)
  • 기간연장: 만료 7일 전부터 활성
  • 카드 탭 시: 동행글 상세 페이지 이동 (WANT02)
6
[Card] 받은 후기 카드
  • 감정 아이콘: (긍정) / (부정) 18px
  • 작성자 닉네임: 12px Bold
  • 후기 텍스트: 12px Gray, 최대 2줄 (ellipsis)
  • 작성 날짜: 12px Gray 400 (YYYY-MM-DD)
  • 관련 거래: 11px Gray (예: "교환: 하이큐 히나타 아크릴")
  • 부정() 후기만: "이의 제기" 링크(Blue) 표시
  • 이의 제기: 거래 완료일로부터 72시간 이내만 활성
  • 링크 탭 시: 이의 제기 바텀시트 오픈
7
[Summary] 후기 요약 카드
  • 원형 게이지: 긍정 비율을 Blue arc로 시각화
  • 중앙 %: 긍정 비율 퍼센트 (예: "83%")
  • 우측 통계: 전체 N건 / 긍정 N건 / 부정 N건
  • 컬러: 긍정(Blue #2F8EC1) / 부정(Red #EA5D86)
  • 후기 목록 상단 고정
8
[SubTab] 공구 서브탭
  • 진행 중인 공구: 내가 진행자인 공구
  • 참여한 공구: 내가 참여자인 공구
  • 활성 탭: Blue 하단 인디케이터
  • 탭 탭 시: 역할별로 목록 전환
9
[Filter] 기간 & 유형 필터
  • 내역 탭에서 사용
  • 기간: 최근 1개월(기본) / 3개월 / 6개월 / 1년 / 전체
  • 유형: 전체 / 교환 / 공구
  • 이중 필터로 조합 사용
10
[Card] 거래 내역 카드
  • 좌측: 36px 원형 아바타
  • 상대방 닉네임: 12px Bold
  • 거래 유형 뱃지: 교환(Blue) / 공구(Purple)
  • 상품명: 14px Bold
  • 완료일: 12px Gray 400
  • 후기 상태: "후기 작성하기" 링크(Blue) 또는 "작성 완료"(Gray)
  • 후기 미작성 시 링크 탭 → 후기 작성 바텀시트
11
[Category] 찜 카테고리 탭
  • 전체(기본) / 교환글 / 공구 / 이벤트
  • 활성 탭: Blue 하단 인디케이터 + Bold
  • 탭 탭 시: 카테고리별 목록 필터링
12
[Card] 찜 카드 (2열 그리드)
  • ProductCard와 동일 포맷 (교환글/공구)
  • 카드 우측 상단: 채워진 빨간 하트(♥) 아이콘
  • 하트 탭 시: 찜 해제 + Optimistic UI 적용
  • 하단 토스트: "찜이 해제되었습니다" + "되돌리기" 링크 (5초)
  • 카드 탭: 상세 페이지 이동 (EXCH02 / GRPB02)
화면 상태
  • 로딩 중: 스켈레톤 카드 3개 펄스 애니메이션
  • 빈 상태: 일러스트 + "등록한 [항목명]이 없어요" + CTA 버튼
    • 교환글 빈: "교환글 등록하기" → EXCH03
    • 공구 빈: "공구 둘러보기" → GRPB01
    • 내역 빈: "교환 매물 둘러보기" → EXCH01
    • 후기 빈: "아직 받은 후기가 없어요"
    • 찜 빈: "매물 둘러보기" → HOME01
  • 에러 상태: "연결이 불안정합니다" + 재시도 버튼
  • 삭제 실패: 토스트 "진행 중인 교환 제안이 있어 삭제할 수 없습니다"
