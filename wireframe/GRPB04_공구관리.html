GRPB04 — 공구 관리
라우트: /groupbuy/[id]/manage  |  진행자 전용  |  Phase 1
1
[Header] 헤더
  • 좌측: 뒤로가기 버튼
  • 중앙: "공구 관리" 제목
  • 우측: 알림 아이콘 (공지 설정)
  • 상단 고정
2
[Card] 공구 요약 카드
  • 상품명: 공구의 정식 명칭
  • 태그: [공구] 또는 [세트공구] 구분
  • 참여 현황: 프로그레스바로 N/M명 표시 (예: 18/30)
  • 마감일: D-N 형식
  • 상태 뱃지: 모집중(파랑) / 마감(주황) / 구매진행중(파랑) / 배송중(파랑) / 완료(회색)
3
[Stepper] 상태 스텝 인디케이터
  • 현재 진행 단계 하이라이트: 1.모집 → 2.결정 → 3.확정 → 4.배송 → 5.완료
  • 무산 경로는 분기로 표시
  • 현재 상태: 파랑(활성), 이전 상태: 회색(비활성)
  • 진행 결정 단계에서 "무산" 옵션 선택 가능
4
[Form + List] 공지사항 섹션
  • 공지 작성: Textarea 입력 필드 (500자 제한) + 등록 버튼
  • 기존 공지 목록: 최신순 정렬
    • 작성일시 (M/D 형식)
    • 공지 내용 (최대 2줄 미리보기)
    • 더보기 링크
  • 등록 후 즉시 목록 상단에 반영
5
[Tabs] 참여자 필터 탭
  • 전체 (N명)
  • 미입금 (N명)
  • 입금완료 (N명)
  • 배송완료 (N명)
  • 활성 탭: 파랑 배경, 비활성: 회색 배경
6
[Table] 참여자 목록 테이블
  • 컬럼 (일반 공구 기준):
    • 닉네임: 프로필 이미지 + 닉네임
    • 참여수량: 구매 수량 (개수)
    • 입금상태: 미입금 / 입금확인 (토글)
    • 배송상태: 대기 / 발송완료 / 수취확인
    • 액션: 채팅 바로가기, 상세 정보 보기
  • 입금 확인 토글: 클릭 시 상태 즉시 변경, 파랑(입금) / 회색(미입금)
  • 배송상태 변경: 드롭다운 또는 모달에서 운송장 번호 입력
  • 채팅 바로가기: CHAT02 화면으로 이동
7
[ButtonBar] 일괄 액션 바
  • 상태별로 표시되는 버튼이 달라집니다:
    • 모집중 상태: [조기 마감] (Amber) · [일괄 발송처리] (비활성)
    • 진행 결정 대기: [진행 확정] (Blue) · [무산 처리] (Red Outline)
    • 진행 확정: [일괄 발송처리] (Blue) · [공구 취소] (Red Outline)
    • 배송중/완료: [일괄 발송처리] (미발송 건만 활성)
  • 조기 마감: 마감일 전에 수동 마감 → 진행 결정 단계로 이동
  • 일괄 발송처리: 선택한 참여자들의 배송상태 일괄 변경
  • 진행 확정 / 무산 처리: 확인 모달 필수
화면 상태
  • 모집중 상태: 조기 마감 버튼 활성, 일괄 발송처리 비활성
  • 진행 결정 대기: 진행 확정/무산 처리 버튼 활성, 자동 무산 타이머 표시
  • 진행 확정 상태: 일괄 발송처리 활성, 공구 취소 가능
  • 배송중/완료: 미발송 건이 있을 때만 일괄 발송처리 활성
  • 진행자 미접근: 권한 없음 오류 메시지
