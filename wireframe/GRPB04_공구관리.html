2) 목표 미달 + 최소 충족: "목표는 미달했지만 최소 인원이 확보되었습니다. 공구를 진행하시겠습니까?"
3) 목표 미달 + 최소 미충족: "최소 인원이 미달되었습니다. 공구를 무산 처리합니다." (버튼 없음, 자동 무산)

7

  • 진행 결정 대기: [진행 확정](파랑색) + [무산 처리](빨강색 테두리)
  • 진행 확정: [배송 시작] + [운송장 입력]
  • 배송 중: [배송 완료] + [미배송 확인]
주요 기능
  • 진행자만 접근 가능한 관리 화면
  • 마감일 도달 시 자동으로 "진행 결정" 단계로 전환
  • 목표 인원 달성 여부에 따라 진행/무산 판단
  • 최소 인원 조건 미충족 시 자동 무산 처리
  • 진행 결정 단계에서 총대의 수동 진행/무산 선택 가능
  • 미결정 상태 5일 경과 시 자동 무산 처리
  • 참여자별 입금 확인 상태 토글
  • 배송상태 관리 및 운송장 번호 입력
  • 참여자 목록 필터링 (입금상태, 배송상태)
  • 공지사항 작성 및 발송
