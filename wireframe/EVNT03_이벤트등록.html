EVNT03 이벤트 등록
이벤트 생성 폼: 카테고리, 유형, 작품, 캐릭터, 기본/선택 정보 입력
1
[Header] 상단 헤더
  • 좌측: 뒤로가기 (작성 중 이탈 시 "작성 중인 내용이 있습니다" 확인)
  • 중앙: "이벤트 등록" 제목 (수정 모드: "이벤트 수정")
2
[Chips] 카테고리 선택
  • 4대 카테고리: 애니/만화/소설(Purple #8B5CF6) | 아이돌/아티스트(Pink #EC4899) | 배우/무대(Amber #F59E0B) | 게임(Cyan #06B6D4)
  • 필수 단일 선택 (처음 진입 시 첫 번째 선택됨)
  • 선택: Blue 배경 + Bold 텍스트, 미선택: Gray 테두리
  • 선택 후 작품 DB 필터링 (카테고리별로 작품 검색)
3
[Dropdown] 이벤트 유형
  • 7가지 옵션: 생일카페 / 데뷔 기념 / 팬미팅 / 팬이벤트 / 출판 기념 / 콘서트/공연 / 기타
  • 필수 선택
4
[SearchInput] 작품/아이돌 검색
  • 작품 DB에서 키워드 검색 (디바운스 300ms)
  • 선택한 카테고리 내에서만 검색 결과 표시
  • 자동완성 드롭다운 (최대 10개)
  • 필수 선택
5
[SearchInput] 캐릭터/멤버 검색
  • 작품 선택 후 해당 작품의 캐릭터/멤버 목록 검색
  • 자동완성 드롭다운
  • 필수 선택
6
[TextInput] 이벤트명
  • 최대 50자 제한
  • 플레이스홀더: "이벤트명을 입력해주세요"
  • 필수 선택
  • 예: "히나타 쇼요 생일 축하 생카", "세븐틴 민규 팬미팅"
7
[DatePicker] 이벤트 기간
  • 시작일 ~ 종료일 두 개 DatePicker 필드
  • 캘린더 아이콘 + 날짜 선택
  • 필수 선택
  • 유효성: 종료일 >= 시작일
8
[MapPicker] 장소
  • 주소 검색 인풋 + 카카오맵 통합
  • 지도 내 핀 드래그로 미세 조정 가능
  • 선택 시 주소 텍스트 표시
  • 필수 선택
9
[OptionalSection] 선택 입력
  • 운영 시간: 시작 시간 ~ 종료 시간 (두 개 시간 필드, HH:mm)
  • 상세 설명: Textarea (최대 1000자, 글자 수 카운터)
  • 대표 이미지: 이미지 업로드 (최대 3장, JPG/PNG, 10MB 이하)
    • 첫 번째 이미지 = 썸네일
    • Sharp로 max 800px 리사이즈
  • 외부 링크: URL 인풋 필드 (예: 티켓팅 페이지, 상세 페이지)
10
[CTA] 등록 버튼
  • 필수 항목 모두 완료 시: Blue 배경, "등록하기" 텍스트
  • 미완료 시: Gray 배경 + 비활성 (필수 필드 빨간 테두리 + 에러 텍스트)
  • 탭 시: API 호출 → 로딩 스피너 → 성공 → EVNT01(이벤트 캘린더) 이동 또는 이벤트 상세 페이지(EVNT02)
  • 수정 모드: "수정하기" 텍스트
  • 에러: 토스트 메시지 "등록에 실패했습니다. 다시 시도해주세요."
화면 상태
기본 상태: 빈 폼, 첫 카테고리 선택됨
입력 완료: 모든 필수 필드 작성 → 버튼 활성(Blue)
수정 모드: 기존 이벤트 데이터 프리필, 제목 "이벤트 수정"
로딩: 제출 중 스피너, 폼 비활성
필드 에러: 빨간 테두리 + 하단 에러 텍스트 (예: "필수 항목입니다")
이탈 확인: 작성 중 뒤로가기 시 다이얼로그
