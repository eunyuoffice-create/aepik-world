3 태그 & 뱃지
거래 유형 태그(교환/공구/구함/세트공구), 카테고리 태그(4대 팬덤), 상태 뱃지(7종), 상품 등급(S/A/B/C), 신뢰 등급(Lv.1~4), 필터 칩

4 팝업 & 오버레이

5 카드 & 빈 상태 & 스켈레톤
상품 카드(12px 라운딩, 이미지 1:1), 빈 상태(라인 아이콘+메시지+CTA), 스켈레톤 로딩(shimmer 애니메이션)
카드 호버: shadow-md + translateY(-2px) 200ms. 빈 상태: 항상 CTA 버튼 포함.
6 네비게이션
탑 바, 서브 헤더(뒤로가기), 바텀 네비게이션(5탭), 탭 바, FAB
바텀 네비게이션 (공통)
- 5탭: 홈(/), 교환(/exchange), 이벤트(/events), 채팅(/chat), 마이(/my)
- 높이: 56px (safe-area 별도), 배경 #FFF, 상단 보더 1px #F3F4F6
- 아이콘: Lucide SVG 20x20px, 라벨 10px Medium, 간격 2px
- 비활성: #9CA3AF (Gray 400), 활성: #2563EB (Blue 600)
- 채팅 탭: 안읽음 카운트 뱃지 (빨간 도트/숫자, 아이콘 우상단)
- 768px 이상: display:none (사이드 네비 전환)
- 적용 대상: 메인 탭 화면 (HOME01, EXCH01, GRPB01, USER01, EVNT01 등)
- 미적용: 상세/등록/설정 등 서브 화면 (앱 헤더 뒤로가기 사용)
FAB: 56px, radius 16px, Blue 배경, shadow. 우하단 고정 (bottom 80px, right 16px).
7 검색 (Search)
검색 바, 자동완성 드롭다운, 최근/인기 검색어, 검색 결과 헤더, 거래 유형 탭, 필터 & 정렬 바, 빈 결과 상태
요소스펙비고
검색 바높이 44px, 라운딩 10px홈: 배경 Gray 100 / 포커스: Blue 보더 2px + ring
자동완성최대 5개, 보더 1px, 라운딩 10px, shadow-md2자 이상 입력 시 노출, 디바운스 300ms
최근 검색어pill 칩, 배경 Gray 100최대 10개, 개별/전체 삭제 가능
인기 검색어pill 칩, 배경 Blue 50, 텍스트 Blue상위 10개, 24시간 기준 갱신
결과 탭전체/교환/공구/구함각 탭에 건수 표시
필터 칩pill, 선택 시 Blue 보더+배경가로 스크롤, 필터 아이콘 포함
빈 결과검색 아이콘(40px) + 안내 텍스트대안 검색어 또는 CTA 제공
검색 바는 홈 탑바와 전용 검색 페이지 두 곳에서 사용. 홈에서 탭 → 검색 페이지로 이동(push). 검색어 입력 중 X 버튼으로 클리어.
7-3 필터 바텀시트

공구 상태: 5단계 수직 타임라인으로 시간 흐름 표현.
현재 단계: 파란색(active) 강조, 완료 단계는 초록 체크마크.
D+3 경고 타이머: 주황색 배경, 진행바로 남은 시간 시각화.
타이머 텍스트: "2일 15시간" 또는 "72h 36m" 형식 (설정 가능).
자동 무산: D+5 도달 시 시스템 자동 무산 처리, 사전 알림 필수.
알림 사용 원칙
미읽 알림: 파란 점 + 배경색 강조 + 굵은 텍스트.
읽음 알림: 일반 텍스트, 배경색 없음.
카테고리 필터: 전체 / 교환 / 공구 / 시스템 4가지.
최대 표시 알림: 최근 100개, 이전 알림은 아카이브 또는 삭제.
푸시 알림(FCM): 앱 background에서 도착 시 배너 띄우기.
알림 클릭: 해당 거래/공구/채팅으로 직진, 자동 읽음 처리.
채팅 사용 원칙
발신/수신 버블: 자동 구분, 우측/좌측 정렬.
메시지 시간: 각 메시지 또는 시간대별 그룹헤더로 표시.
이미지 메시지: 최대 너비 200px, 전체화면 뷰어 지원.
거래 확인 카드: 양측 수락 필요한 경우, 인라인 카드 + 수락/거절 버튼.
읽음 상태: 상대 아바타 하단 초록 점 또는 "읽음" 표시 (선택적).
메시지 저장 및 검색: 추후 단계 고려.
차단된 유저: 메시지 송수신 불가, 명확한 안내 필수.
지도/위치 사용 원칙
카카오맵 API 연동: 배포 지역 한정(한국).
위치 검색 자동완성: 500ms 디바운스, 최대 5개 결과.
선택된 위치: 주소 + 지하철 정보 카드 표시.
위치 변경: 검색 재입력 또는 지도 직접 핀 드래그.
거래 알림: "우리 동네", "인접 동네" 필터링 (향후).
개인정보: 정확한 주소는 거래 확정 후에만 공개.
캘린더 사용 원칙
이벤트 표시: 활동 있는 날짜 하단 컬러 점 (카테고리별).
오늘 표시: 파란 원형, 항상 현재 달에서 강조.
뷰 모드: 월별(그리드) / 리스트(시간순) / 지도(위치).
이벤트 카테고리: 애니/만화/소설(Purple) / 아이돌/아티스트(Pink) / 배우/무대(Amber) / 게임(Cyan).
이벤트 추가: 날짜 탭 → "새로운 일정" 또는 거래 진행 시 자동 추가.
알림: 이벤트 당일 아침 또는 설정된 시간 이전 알림.
뱃지/닷 사용 원칙
빨간 점(8px): 일반 알림 표시, 개별 카운트 불필요.
숫자 뱃지: 정확한 카운트 필요 시 (미읽 메시지, 제안 개수).
최대 표시: 100 이상일 경우 "99+" (overflow 처리).
배치: 아이콘/버튼/탭 우상단 absolute 위치.
색상: 빨강 기본, 상황에 따라 주황색(경고) 또는 파랑(정보) 가능.
애니메이션: 첫 나타남 시 scale-in, 업데이트 시 fade, 제거 시 fade-out.
아바타 사용 원칙
사이즈별 용도: 24px(댓글) / 32px(알림) / 40px(채팅) / 48px(모달) / 64px(프로필).
온라인 상태: 초록 점(10px) 우하단, hover 시 상태(온라인/자리비움/오프라인) 팝오버.
신뢰도 테두리: 2px 테두리, 등급별 색상(빨강/주황/파랑/초록).
아바타 그룹: 3명 이상일 때 좌측 -8px 겹침, 흰 보더로 구분.
기본 아바타: person 아이콘 silhouette, 로드 완료 시 이미지 교체.
캐싱: 네트워크 효율을 위해 아바타 이미지 캐싱 권장.
버튼 사용 원칙
화면당 Primary CTA는 최대 1~2개로 제한.
파괴적 액션(삭제, 신고)은 Danger 또는 확인 모달 필수.
폼 하단 CTA는 풀폭(Full Width) Large 사용.
인풋 사용 원칙
필수 항목: 레이블 우측 붉은 별(*) 표시.
에러 메시지: 인풋 바로 아래 빨간 텍스트.
셀렉트: 옵션 5개 이하 → 라디오, 5개 초과 → 드롭다운.
긴 텍스트: textarea 사용 (최소 높이 100px).
팝업 사용 구분
확인 모달: 되돌릴 수 없는 액션 (수락, 삭제, 탈퇴)
바텀시트: 선택지 제공 (사유 선택, 정렬, 필터)
토스트: 성공/에러 피드백 (저장 완료, 네트워크 오류)
스낵바: 되돌리기 가능한 액션 (삭제 후 되돌리기)
태그 & 뱃지 사용 원칙
카드 내 태그: 최대 2개 (거래유형 + 카테고리).
상태 뱃지: 1개만 표시. 여러 상태일 때 우선순위 높은 것 표시.
필터 칩: 선택 시 Primary 색상, 비선택 시 Gray.
피트 칩은 가로 스크롤 가능.
검색 사용 원칙
검색 바: 홈 탑바에 항상 노출, 탭 시 전용 검색 페이지로 이동.
자동완성: 최대 5개, 검색어 강조(Bold Blue).
최근 검색어: 최대 10개, FIFO. 로컬 저장.
인기 검색어: 서버 집계, 24시간 기준 갱신.
빈 결과: 항상 대안 안내 (다른 키워드 제안 또는 전체 목록 CTA).
필터: 카테고리(4대), 거래 유형(교환/공구/구함), 정렬(최신/인기) 지원.
필터 바텀시트 사용 원칙
진입: 검색 결과 화면의 "필터" 칩 탭 시 바텀시트 올라옴.
필터 그룹: 카테고리(4대 팬덤) > 굿즈 타입 > 상품 상태(S/A/B/C) > 거래 방법 > 정렬 > 추가 옵션 순서.
칩 선택: 복수 선택 가능 (정렬만 단일 선택).
선택 상태: Blue 보더+배경 (카테고리는 해당 카테고리 컬러 사용).
"N건 보기": 실시간 결과 건수 갱신, 0건이어도 버튼 활성 상태 유지.
적용 후: 필터 칩 바에 선택된 필터가 칩으로 표시됨.
초기화: 모든 필터 해제, 기본값(최신순)으로 복귀.
Desktop(768px+): 중앙 모달 전환, 최대 너비 480px.
접근성 체크리스트
모든 버튼: aria-label 또는 텍스트 포함
인풋: label과 연결 (for/id)
모달: 열릴 때 포커스 트랩, ESC로 닫기
터치 타겟: 최소 44x44px
색상 대비: WCAG AA (4.5:1)
