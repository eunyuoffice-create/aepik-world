2 인풋 & 폼 (Inputs)
텍스트 인풋, 텍스트 영역, 셀렉트, 체크박스, 라디오, 토글 지원. 모든 인풋은 높이 44px, 보더 8px 라운딩.
상태보더배경
기본#E5E7EB#FFF
포커스#3B82F6 + ring 2px#FFF

필수 항목: 레이블 옆 붉은 별(*) 표시. 에러 시 하단에 빨간 메시지. 글자 수 힌트는 하단 우측.
3 태그 & 뱃지
거래 유형 태그(교환/공구/구함/세트공구), 카테고리 태그(4대 팬덤), 상태 뱃지(7종), 상품 등급(S/A/B/C), 신뢰 등급(Lv.1~4), 필터 칩
태그 라운딩: 999px (pill). 패딩: 4px 10px. 폰트: 12px SemiBold.
4 팝업 & 오버레이
확인 모달, 알림 모달, 바텀시트, 토스트, 스낵바 5종
컴포넌트라운딩그림자진입 모션
모달16pxLevel 2250ms scale+opacity
바텀시트16px 16px 0 0Level 2300ms translateY
토스트12pxLevel 3300ms translateY+opacity
스낵바8pxLevel 1200ms opacity
모달: 배경 딤(#000, 50% opacity). 바텀시트: 드래그 핸들 포함, 하단에서 올라옴. 토스트: 3초 후 자동 사라짐.
5 카드 & 빈 상태 & 스켈레톤
상품 카드(12px 라운딩, 이미지 1:1), 빈 상태(라인 아이콘+메시지+CTA), 스켈레톤 로딩(shimmer 애니메이션)
카드 호버: shadow-md + translateY(-2px) 200ms. 빈 상태: 항상 CTA 버튼 포함.
6 네비게이션
탑 바, 서브 헤더(뒤로가기), 바텀 네비게이션(5탭), 탭 바, FAB
바텀 네비: 비활성 Gray 400, 활성 Blue. 5탭(홈/교환/이벤트/채팅/마이). FAB: 56px, radius 16px, Blue 배경, shadow.
버튼 인터랙션
호버: 배경색 10% 어둡게 + translateY(-1px), 150ms ease-out
프레스: 배경색 15% 어둡게 + scale(0.98)
디스에이블: 클릭 불가, cursor: not-allowed
포커스: Blue ring 2px (키보드 탐색 시)
인풋 인터랙션
포커스: 보더 Blue + 링 2px, 즉시 반응
에러: 보더 Red + 하단 에러 메시지, 입력 중 실시간 검증
클리어 버튼: 입력값 있을 때 우측 X 아이콘 표시
글자 수 카운터: 실시간 갱신
모달 인터랙션
열기: 배경 딤 페이드인 + 모달 center scale(0.95→1)+opacity, 250ms
닫기: 역순 애니메이션, 200ms
배경 딤 탭: 모달 닫기 (확인 모달은 비활성)
ESC 키: 모달 닫기
바텀시트 인터랙션
열기: 하단에서 슬라이드업, 300ms ease-out
닫기: 하단으로 슬라이드다운 또는 배경 딤 탭
드래그: 핸들 드래그로 닫기 (아래로 100px+)
Desktop: 중앙 모달로 전환 (768px+)
토스트 & 스낵바
토스트: 하단에서 올라옴, 3초 유지 후 자동 사라짐
스낵바: 하단 고정, 액션 버튼 있으면 5초 유지
스와이프 또는 탭으로 수동 닫기 가능
스켈레톤 & 로딩
shimmer 애니메이션: 1.5s ease-in-out, 좌→우 그라데이션
실제 콘텐츠 크기와 동일한 placeholder
데이터 로드 완료 시 fade-in 전환 (200ms)
버튼 사용 원칙
화면당 Primary CTA는 최대 1~2개로 제한.
파괴적 액션(삭제, 신고)은 Danger 또는 확인 모달 필수.
폼 하단 CTA는 풀폭(Full Width) Large 사용.
인풋 사용 원칙
필수 항목: 레이블 우측 붉은 별(*) 표시.
에러 메시지: 인풋 바로 아래 빨간 텍스트.
셀렉트: 옵션 5개 이하 → 라디오, 5개 초과 → 드롭다운.
긴 텍스트: textarea 사용 (최소 높이 100px).
팝업 사용 구분
확인 모달: 되돌릴 수 없는 액션 (수락, 삭제, 탈퇴)
바텀시트: 선택지 제공 (사유 선택, 정렬, 필터)
토스트: 성공/에러 피드백 (저장 완료, 네트워크 오류)
스낵바: 되돌리기 가능한 액션 (삭제 후 되돌리기)
태그 & 뱃지 사용 원칙
카드 내 태그: 최대 2개 (거래유형 + 카테고리).
상태 뱃지: 1개만 표시. 여러 상태일 때 우선순위 높은 것 표시.
필터 칩: 선택 시 Primary 색상, 비선택 시 Gray.
피트 칩은 가로 스크롤 가능.
접근성 체크리스트
모든 버튼: aria-label 또는 텍스트 포함
인풋: label과 연결 (for/id)
모달: 열릴 때 포커스 트랩, ESC로 닫기
터치 타겟: 최소 44x44px
색상 대비: WCAG AA (4.5:1)
