HOME01 — 메인 홈
라우트: /  |  비로그인 접근: 가능  |  Phase 1
1
[Header] 앱 헤더
  • 좌측: 애픽 월드 로고 (브랜드 컬러 Blue #3B82F6)
  • 우측 아이콘:
    • 알림 아이콘 → NOTI01 이동, 미읽은 알림 시 뱃지 표시
    • ☰ 메뉴 아이콘 → 사이드 메뉴 슬라이드
  • 상단 고정 (스크롤 시 유지)
2
[SearchBar] 검색바
  • 플레이스홀더: "상품명, 작품, 캐릭터 검색"
  • 좌측 아이콘
  • 홈에서는 입력 폼이 아닌 탭 영역으로 동작
  • 탭 시 검색 페이지(SRCH01) 이동
3
[BannerCarousel] 배너 캐러셀
  • 배너 이미지 규격: 1200×300px (4:1 비율)
  • 자동 슬라이드: 5초 간격, 터치 스와이프 지원
  • 하단 인디케이터: ● ○ ○ (활성 = Blue)
  • 배너 유형: 생일/데뷔 축하 (사용자 제작), 프로모션
  • 탭 시: 배너 주인공 관련 매물 목록 페이지 이동
  • 배너 없을 시: 기본 서비스 소개 배너 표시
4
[Section] 다가오는 이벤트
  • 섹션 헤더: " 다가오는 이벤트" + [더보기] → EVNT01
  • 카드: EventMiniCard (가로 스크롤)
  • 카드 구성:
    • 날짜 (M/D 요일)
    • 이벤트명
    • 장소
    • 카테고리 태그 (애니·만화·소설/아이돌·아티스트/배우·무대/게임)
  • 카드 탭 시: 이벤트 상세(EVNT02) 이동
  • 최대 6개 표시 (오늘 이후, 날짜순)
  • Phase 1.5 기능 (Phase 1에서는 이 섹션 미노출)
5
[Grid] 인기 교환
  • 섹션 헤더: " 인기 교환" + [더보기] → EXCH01
  • 2열 그리드, 최대 4개 카드 표시
  • 카드: ProductCard 컴포넌트
    • 이미지 (1:1 비율, object-fit: cover)
    • 태그: [거래유형] + [카테고리]
    • 상품명 (최대 2줄, text-overflow: ellipsis)
    • 작품명 · 캐릭터명 (Gray 400)
    • ★ 평점 · 거래 N회
  • 카드 탭 시: 교환 상세(EXCH02) 이동
  • 정렬: 최근 7일 조회수 + 찜 수 기준
6
[HScroll] 진행 중 공구
  • 섹션 헤더: " 진행 중 공구" + [더보기] → GRPB01
  • 카드: GroupBuyCard (가로 스크롤)
  • 카드 구성:
    • 이미지 (16:9)
    • 태그: [일반공구] 또는 [세트공구]
    • 상품명 (1줄)
    • 프로그레스 바: (N/M)%
    • 참여 현황: N/M명 · 마감 D-N
  • 프로그레스 바 색상: 80% 이상 Green, 미만 Blue
  • 카드 탭 시: 공구 상세(GRPB02 또는 GRPB03) 이동
  • 최대 6개 표시 (마감 임박순)
7
[List] 구함글 모아보기
  • 섹션 헤더: " 구함글 모아보기" + [더보기] → WANT01
  • 리스트 형태 (최대 3개 표시)
  • 카드 구성:
    • [구함] 태그 (Green 배경)
    • 구함글 제목 (1줄)
    • 작성 시간 (N분 전, N시간 전)
  • 카드 탭 시: 구함글 상세(WANT02) 이동
  • 정렬: 최신순
8
[FAB] 플로팅 액션 버튼
  • 위치: 우하단 고정 (bottom: 80px, right: 16px)
  • 크기: 56×56px, border-radius: 50%
  • 색상: Blue 배경 + 흰색 + 아이콘
  • 탭 시: 옵션 팝업
    • 교환글 등록 → EXCH03
    • 공구 등록 → GRPB (new)
    • 구함글 등록 → WANT (new)
  • 비로그인 시: 로그인 유도 팝업
9
[BottomNav] 하단 네비게이션
  • 5개 탭: 홈 | 교환 | 이벤트 | 채팅 | 마이
  • 비활성: Gray 400, 활성: Blue (#3B82F6)
  • 채팅/마이: 비로그인 시 로그인 유도
  • 채팅: 안 읽은 메시지 수 뱃지 표시
  • 하단 고정
화면 상태
  • 로딩 상태: 각 섹션 스켈레톤 UI 표시 (카드 형태 회색 박스 펄스 애니메이션)
  • 빈 상태: 해당 섹션 미노출 (데이터 없는 섹션은 숨김)
  • 에러 상태: "데이터를 불러올 수 없습니다" + 재시도 버튼
  • 비로그인: 모든 섹션 열람 가능, FAB 탭 시 로그인 유도
