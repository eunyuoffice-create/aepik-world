1
[Header] 앱 헤더
  • 좌측: 애픽 월드 로고 (브랜드 컬러 Blue #69BFEA)
  • 우측 아이콘:
    • 알림 아이콘 → NOTI01 이동, 미읽은 알림 시 뱃지 표시
    • ☰ 메뉴 아이콘 → 사이드 메뉴 슬라이드
  • 상단 고정 (스크롤 시 유지)
2
[SearchBar] 검색바
  • 플레이스홀더: "상품명, 작품, 캐릭터 검색"
  • 좌측 아이콘
  • 홈에서는 입력 폼이 아닌 탭 영역으로 동작
  • 탭 시 검색 페이지(SRCH01) 이동
3
[BannerCarousel] 배너 캐러셀
  • 배너 이미지 규격: 1200×600px (2:1 비율)
  • 자동 슬라이드: 5초 간격, 터치 스와이프 지원
  • 하단 인디케이터: ● ○ ○ (활성 = Blue)
  • 배너 유형: 생일/데뷔 축하 (사용자 제작), 프로모션
  • 탭 시: 배너 주인공 관련 매물 목록 페이지 이동
  • 배너 없을 시: 기본 서비스 소개 배너 표시
4
[Section] 다가오는 이벤트
  • 섹션 헤더: " 다가오는 이벤트" + [더보기] → EVNT01
  • 카드: EventMiniCard (가로 스크롤, 160px 고정 너비)
  • 카드 구성:
    • 배너 이미지 (가로, 160×80px, object-fit: cover)
    • 날짜 (M/D 요일) — Blue #2F8EC1
    • 이벤트명 (1줄, ellipsis)
    • 장소
    • 카테고리 태그 (애니·만화·소설/아이돌·아티스트/배우·무대/게임)
  • 카드 탭 시: 이벤트 상세(EVNT02) 이동
  • 최대 6개 표시 (오늘 이후, 날짜순)
  • Phase 1.5 기능 (Phase 1에서는 이 섹션 미노출)
5
[Grid] 인기 교환
  • 섹션 헤더: " 인기 교환" + [더보기] → EXCH01
  • 2열 그리드, 최대 4개 카드 표시
  • 카드: ProductCard 컴포넌트
    • 이미지 (1:1 비율, object-fit: cover)
    • 태그: [거래유형] + [카테고리]
    • 상품명 (최대 2줄, text-overflow: ellipsis)
    • 작품명 · 캐릭터명 (Gray 400)
    • ★ 평점 · 거래 N회
  • 카드 탭 시: 교환 상세(EXCH02) 이동
  • 정렬: 최근 7일 조회수 + 찜 수 기준
6
[HScroll] 진행 중 공구
  • 섹션 헤더: " 진행 중 공구" + [더보기] → GRPB01
  • 카드: GroupBuyCard (가로 스크롤)
  • 카드 구성:
    • 이미지 (1:1 비율, object-fit: cover)
    • 태그: [일반공구] 또는 [세트공구]
    • 상품명 (1줄)
    • 프로그레스 바: (N/M)%
    • 참여 현황: N/M명 · 마감 D-N
  • 프로그레스 바 색상: 80% 이상 Green, 미만 Blue
  • 카드 탭 시: 공구 상세(GRPB02 또는 GRPB03) 이동
  • 최대 6개 표시 (마감 임박순)
7
[List] 동행 모아보기
  • 섹션 헤더: " 동행 모아보기" + [더보기] → WANT01
  • 리스트 형태 (최대 3개 표시)
  • 카드 구성:
    • [동행] 태그 (Green 배경)
    • 동행글 제목 (1줄)
    • 작성 시간 (N분 전, N시간 전)
  • 카드 탭 시: 동행글 상세(WANT02) 이동
  • 정렬: 최신순
8
[InfoStrip] 서비스 소개 · 운영정책 진입
  • 모바일 홈 하단에 `/about` 바로가기 스트립 노출
  • 문구: 운영자 개발 배경 + 유료 운영 이유 간단 요약
  • 링크: 서비스 소개/운영정책 페이지(STAT01 /about)
  • PC는 우측 패널에 동일 진입 카드 제공
9
[FAB] 플로팅 액션 버튼
  • 위치: 우하단 고정 (bottom: 80px, right: 16px)
  • 크기: 56×56px, border-radius: 50%
  • 색상: Blue 배경 + 흰색 + 아이콘
  • 탭 시: 옵션 팝업
    • 교환글 등록 → EXCH03
    • 공구 등록 → GRPB (new)
    • 동행글 등록 → WANT (new)
  • 비로그인 시: 로그인 유도 팝업
10
[BottomNav] 하단 네비게이션
  • 5개 탭: 홈 | 교환 | 이벤트 | 채팅 | 마이
  • 비활성: Gray 400, 활성: Blue (#69BFEA)
  • 채팅/마이: 비로그인 시 로그인 유도
  • 채팅: 안 읽은 메시지 수 뱃지 표시
  • 하단 고정
11
[Popup] 이벤트 팝업 (접속 시)
  • 노출 시점: 홈 진입 시 자동 노출 (1일 1회/팝업)
  • 슬롯: 최대 3건/일, 여러 건일 경우 순차 노출
    • [닫기] 시 다음 미확인 팝업 표시
    • 모든 팝업 확인 완료 시 팝업 없음
  • 팝업 구성:
    • 이벤트 포스터 이미지 (320x400px, 세로형)
    • 이벤트명 텍스트
    • 기간 + 장소 메타 정보
    • 카테고리 태그
    • [이벤트 보러가기] CTA → EVNT02 이동
  • 닫기: 우상단 X 버튼 (반투명 원형)
  • "오늘 하루 안 보기" 체크박스
    • 체크 시 localStorage에 날짜+팝업ID 저장
    • 당일 해당 팝업 재노출 안 됨
  • 슬롯 상태 뱃지 (팝업 하단, 실시간):
    • "이번 주 슬롯 여유" (Green) — 잔여 >50%
    • "이번 주 슬롯 마감 임박" (Amber) — 잔여 25~50%
    • "이번 주 슬롯 마감" (Red) — 잔여 0%
  • "나도 배너/팝업 올리기" 링크 → BANR01 이동
  • 배경: 반투명 딤 (rgba(0,0,0,0.5)), 딤 탭 시 닫기
  • 팝업 용도: 생일카페, 데뷔 기념, 콘서트, 팬미팅, 전시 등 팬 이벤트 홍보
  • 게시 기간: 최소 1일 ~ 최대 14일 (신청 시 선택)
  • 현재 슬롯 마감 시: 달력에서 매진 표시, 신청 불가
  • Phase 1.5 기능 (Phase 1에서는 팝업 미노출)
12
[DesktopLayout] PC 메인 레이아웃
  • 전환 기준: 768px 이상
  • 구조: TopBar + 3열 레이아웃 (좌측 사이드 / 메인 콘텐츠 / 우측 패널)
  • TopBar: 카테고리 탭 + 통합 검색 + 알림/채팅/프로필 아이콘
  • 좌측 사이드: 홈/교환/이벤트/동행/공구 + 내 활동 바로가기
  • 메인 콘텐츠: 배너(2:1) + 다가오는 이벤트 스와이퍼 + 인기 교환 4열 + 진행 중 공구 4열
  • 우측 패널: 이벤트 배너 안내 + 동행 구함 목록
  • 모바일 BottomNav는 PC에서 TopBar/사이드 네비로 대체
화면 상태
  • 로딩 상태: 각 섹션 스켈레톤 UI 표시 (카드 형태 회색 박스 펄스 애니메이션)
  • 빈 상태: 해당 섹션 미노출 (데이터 없는 섹션은 숨김)
  • 에러 상태: "데이터를 불러올 수 없습니다" + 재시도 버튼
  • 비로그인: 모든 섹션 열람 가능, FAB 탭 시 로그인 유도
