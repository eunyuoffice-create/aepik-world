1
[Header] 페이지 헤더
  • 좌측: 뒤로가기 아이콘 (<) → 작성 정보 유실 경고 모달 후 GRPB01 이동
  • 중앙: 페이지 제목 "공구 등록" (고정 텍스트)
  • 우측: 닫기 아이콘 (×) → 작성 정보 유실 경고 모달 후 GRPB01 이동
  • 스크롤 시 상단 고정 (sticky)
2
[RadioCard] 공구 유형 선택
  • 형태: 큰 카드 2개 (2열 또는 모바일 수직 스택)
  • 선택: 라디오 버튼 (하나만 선택)
  • 옵션:
    • 일반 공구 (기본 선택) — 단일 상품 공동 구매
    • 세트 공구 — 세트 구매 후 캐릭터 분배
  • 선택된 카드: Blue 테두리 (#2563EB) + 배경 강조 (#EFF6FF)
  • 선택 변경 시 폼 필드 동적 갱신 (세트 공구 시 캐릭터 슬롯 섹션 표시/숨김)
3
[ChipGroup] 카테고리 선택
  • 형태: 4개 칩 (라디오 그룹, 단일 선택)
  • 옵션: 애니/만화/소설 | 아이돌/아티스트 | 배우/무대 | 게임
  • 기본 선택: 애니/만화/소설
  • 선택된 칩: Blue 배경 (#2563EB) + 흰색 텍스트
  • 비선택 칩: Gray 배경 (#F5F5F5) + Dark 텍스트 (#555)
4
[Form] 기본 정보
  • 상품명 (필수)
    • Text input, 최대 100자
    • 예: "원신 카즈하 아크릴 스탠드 공구"
  • 상품 이미지 (선택)
    • 최대 10장 (S3 + Sharp 리사이즈 1200×1200px)
    • 회색 점선 박스 + "+ 사진 추가" 텍스트
    • 업로드 후 썸네일 그리드 (3열) 표시
    • 각 이미지: 삭제 버튼(X) 오버레이, 드래그로 순서 변경 가능
  • 작품 (선택)
    • Text input with search icon, Autocomplete 드롭다운
    • 세트 공구 선택 시 필수
  • 굿즈 타입 (선택)
    • Dropdown: 포토카드/아크릴/캔뱃지/피규어/인형/엽서/의류/기타
5
[Form] 가격 정보
  • 일반 공구 (type: regular)
    • 1인당 가격 (필수) — Number input with ₩
    • 배송비 포함 (선택) — Toggle switch (기본: OFF)
  • 세트 공구 (type: set)
    • 캐릭터당 가격 (필수) — Number input with ₩
    • 배송비 포함 (선택) — Toggle switch (기본: OFF)
  • 토글 ON 시: 배송비 이미 포함된 가격으로 처리
6
[Form] 모집 정보
  • 목표 인원 (필수)
    • Number input, 최소 2명 이상
    • 예: 30
  • 최소 인원 (선택)
    • Number input, 목표 인원 이하
    • 예: 20
  • 마감일 (필수)
    • Date picker
    • 최소: 현재 + 1일 이후
    • 예: 2026-03-13
  • 배송 예정일 (필수)
    • Date picker
    • 마감일 이후 설정 필수
    • 예: 2026-04-01
7
[Form] 캐릭터 슬롯 & 상세 설명
  • 캐릭터 슬롯 설정 (세트 공구 시에만 표시)
    • 작품 선택 시 해당 작품의 캐릭터 목록 자동 로드
    • 각 행: [체크박스] + [캐릭터명] + [선점 가능 수량 input]
    • 선택된 캐릭터만 공구에 포함
    • 최소 2개 캐릭터 선택 필수
    • 버튼: "[+] 캐릭터 직접 추가" — 작품에 없는 캐릭터 수동 입력
  • 상세 설명
    • Textarea, 마크다운 지원
    • Placeholder: "공구 상세 내용을 입력하세요..."
    • 최대 3000자
    • 배송 방법, 픽링 일정, 추가 금액 등 안내
8
[ButtonBar] 하단 CTA 버튼
  • Step 1 (기본/가격): [미리보기] | [다음]
  • Step 2 (모집/캐릭터): [이전] | [등록하기]
  • [미리보기] — Outline 스타일, 공구 정보 프리뷰 (GRPB02 preview mode)
  • [등록하기] — Blue 배경, 흰색 텍스트, 클릭 시 확인 모달
    • 모달: "공구를 등록하시겠습니까?" + [취소] [등록하기]
    • 등록 중: 버튼 비활성화 + 로딩 스피너
    • 성공: GRPB02(공구 상세) 이동
  • 진행자 자격 미달 시: 안내 메시지 + 버튼 비활성화
    • "진행자 자격(교환 5회 이상, 긍정 90% 이상)이 없습니다."
  • Sticky 위치 (폼 스크롤 시에도 항상 표시)
화면 상태
  • 기본 상태: 공구 유형 일반 공구 선택, 카테고리 애니/만화/소설 선택, 폼 입력 준비
  • 세트 공구 모드: 캐릭터 슬롯 섹션 표시, 작품 필수, 캐릭터 최소 2개
  • 검증 상태: 필수 필드 미입력 시 "다음" 또는 "등록하기" 버튼 비활성화
  • 로딩 상태: 등록 중 버튼 비활성화 + 스피너
  • 진행자 미달: "진행자 자격이 없습니다" 경고 메시지 표시
  • 작성 유실 경고: 뒤로가기/닫기 탭 시 확인 모달 "작성 중인 정보가 유실됩니다"
