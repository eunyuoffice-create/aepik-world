EXCH01 교환 목록
교환글을 탐색하고 필터링하는 메인 목록 화면
1
[Header] 페이지 헤더
  • 좌측: 페이지 제목 "교환" (고정 텍스트)
  • 우측: 검색 아이콘 (🔍)
    • 탭 시: 검색 페이지(SRCH01) 이동
  • 스크롤 시 상단 고정 (sticky)
2
[FilterChips] 필터 바
  • 가로 스크롤 칩 형태 (모바일에서 좌우 스와이프)
  • 필터 항목:
    • 작품 ▾ — 작품 DB 검색 선택
    • 캐릭터 ▾ — 작품 선택 시 활성화
    • 굿즈 타입 ▾ — 포토카드/아크릴/캔뱃지/피규어/인형/엽서/의류/기타
    • 상품 상태 ▾ — S급/A급/B급/C급
    • 거래 방법 ▾ — 택배/직거래/둘 다
    • 차액 여부 ▾ — 차액 가능/불가
  • 칩 탭 시: 바텀시트 → 옵션 선택 → 칩에 선택값 표시 + ✕
  • 선택된 칩: Blue 배경 + 흰색 텍스트
  • 초기화 버튼: 선택된 필터가 1개 이상일 때만 표시
3
[SortBar] 정렬 + 결과 수
  • 좌측: "전체 N개" — 필터 적용 결과 수
  • 우측: 정렬 드롭다운
    • 최신순 (기본) — created_at DESC
    • 인기순 — 조회수 + 찜 수 기반
    • 가까운순 — 위치 권한 허용 시만 활성화
  • 선택 변경 시 목록 즉시 리로드
4
[Grid] 교환글 그리드
  • 2열 그리드 (모바일) / 4열 (데스크톱)
  • 카드: ProductCard 컴포넌트
    • 이미지 (1:1 비율, object-fit: cover)
    • 이미지 우상단 뱃지 (선택):
      • 제안 받은 매물 (본인 글): "제안 N건" 파란색 뱃지
      • 끌어올리기된 매물: "🔥 UP" 주황색 뱃지
    • 태그: [거래유형] + [카테고리]
    • 상품명 (최대 2줄, text-overflow: ellipsis)
    • 작품명 · 캐릭터명 (Gray 400)
    • ★ 평점 · 거래 N회
  • 교환 완료 매물: 이미지 위 "교환완료" 뱃지 + 전체 반투명(opacity 0.5)
  • 무한 스크롤 (Intersection Observer)
    • 하단 도달 시 다음 페이지 자동 로드
    • 로딩 중 스켈레톤 카드 2개 표시
  • 카드 탭 시: 교환 상세(EXCH02) 이동
  • 카드 호버: shadow-md + translateY(-2px)
5
[FAB] 교환글 등록 버튼
  • 우하단 고정 (bottom: 80px, right: 16px)
  • 56×56px, border-radius: 50%, Blue 배경
  • 아이콘: + (또는 ✏️ 편집 아이콘)
  • 탭 시: 교환글 등록(EXCH03) 이동
    • 비로그인: 로그인 유도 바텀시트
    • 미인증: 본인인증 유도 바텀시트
  • 스크롤 다운 시: 축소 or 반투명 (선택)
6
[BottomNav] 하단 네비게이션
  • 5개 탭: 홈 | 교환(활성) | 이벤트 | 채팅 | 마이
  • 활성 탭: Blue (#2563EB), 비활성: Gray 400
  • 채팅/마이: 비로그인 시 로그인 유도
  • 채팅: 안 읽은 메시지 뱃지 (빨간 dot)
