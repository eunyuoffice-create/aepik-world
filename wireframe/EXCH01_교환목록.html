EXCH01 교환 목록
교환글을 탐색하고 필터링하는 메인 목록 화면 (모바일/PC + 필터/팝오버 + 내 글 액션 팝업 플로우)
1
[Header]페이지 헤더
  • 좌측: 페이지 제목 "교환"
  • 우측: 검색 아이콘 탭 시 SRCH01 이동
  • 스크롤 시 상단 고정(sticky)
2
[FilterChips]필터 바
  • 가로 칩: 작품/캐릭터/굿즈 타입/상태/거래 방법/차액
  • 선택 칩은 Blue 배경 + 값 표시(예: 하이큐 ✕)
  • 필터가 하나 이상 선택되면 `초기화` 표시
  • 칩 탭 시 바텀시트(모바일) 또는 팝오버(PC) 열림
3
[SortBar]정렬 + 결과 수
  • 좌측: 전체 N개
  • 우측: 최신순/인기순/가까운순
  • 변경 즉시 목록 재조회
4
[Grid]교환글 그리드
  • 모바일 2열 / PC 4열
  • 카드 정보: 이미지, 거래유형/카테고리, 상품명, 작품·캐릭터, 평점·거래횟수
  • 이미지 우상단 상태 뱃지
    • 내 글: `내 글`
    • 제안 N건(본인 글)
    • UP(끌어올리기)
  • 받은 제안 이력이 1건 이상(상태 무관)이면 `수정/삭제` 비활성 + 안내 문구 노출
  • 교환 완료는 오버레이 + 카드 반투명
  • 카드 탭 시 EXCH02 이동
5
[FAB]교환글 등록 버튼
  • 우하단 고정
  • 탭 시 EXCH03 이동
  • 비로그인 시 로그인 유도
6
[BottomNav]하단 네비게이션
  • 홈/교환/이벤트/채팅/마이
  • 교환 탭 활성 상태
7
[BottomSheet]필터 선택 바텀시트
  • 칩 탭 시 바텀시트 노출 + 배경 dim
  • 검색 + 옵션 리스트 + 초기화/적용
  • 적용 시 칩/목록/건수 동기 갱신
8
[DesktopLayout]PC 교환 목록 레이아웃
  • 768px+에서 적용
  • TopBar + 필터 바 + 정렬 + 4열 그리드
  • 모바일과 동일한 카드 정보/상태 노출
9
[DesktopFilterPopover]PC 필터 선택 팝오버
  • 칩 하단 팝오버 오픈
  • 검색 + 옵션 + 초기화/적용
  • 외부 클릭/닫기 시 미적용
10
[MyPostActions]내 글 카드 액션 케이스
  • 본인 글 카드에는 `내 글` 뱃지 + 더보기 메뉴 표시
  • 더보기 메뉴: `수정 / 끌어올리기 / 삭제`
  • 단, 받은 제안 이력 1건 이상(상태 무관)이면 `수정/삭제` 비활성 + 정책 안내 노출
  • 받은 제안 이력 0건일 때만 `수정` 탭 시 EXCH03 수정 모드(`/exchange/[id]/edit`) 이동
  • 모바일/PC 동일 정책 적용
11
[EditFlow]수정 액션 분기
  • `proposal_history_count=0`이면 `수정` 탭 시 EXCH03 수정 모드 이동
  • `proposal_history_count≥1`이면 `수정` 비활성 + 안내 토스트 노출
  • 잠금 안내 문구: "받은 제안 이력이 있어 수정할 수 없어요."
12
[BumpConfirm]끌어올리기 확인 팝업
  • `끌어올리기` 탭 시 확인 팝업 노출 (취소/끌어올리기)
  • 확인 시 bump 처리 후 성공 토스트 + 카드 `UP` 뱃지/정렬 갱신
  • 무료 횟수/유료 전환 안내를 팝업 본문에 함께 노출 (Phase 1.5 추가 500원/회)
13
[DeletePolicy]삭제 잠금/확인 정책
  • `proposal_history_count=0`에서만 `삭제` 탭 시 확인 팝업 노출 (취소/삭제)
  • `proposal_history_count≥1`이면 `삭제` 비활성 + 잠금 안내 토스트/문구 노출
  • 잠금 안내: "받은 제안 이력이 있어 삭제할 수 없어요. 분쟁 방지를 위해 마감으로 종료해 주세요."
14
[DesktopMyPostActions]PC 내 글 액션 팝업
  • PC는 바텀시트 대신 중앙 모달로 `끌어올리기/삭제` 확인
  • 단, 삭제 확인 모달은 받은 제안 이력 0건에서만 노출
  • 모바일과 동일한 정책/문구/예외 조건 유지
  • 키보드 접근성(`Enter` 확인, `Esc` 닫기) 지원
화면 상태
  • 필터 미적용(초기): 선택 칩 없음, 초기화 숨김
  • 필터 적용: 선택 칩/초기화 노출, 조건 기반 결과
  • 필터 선택 상태: 바텀시트 오픈, 옵션 선택/적용
  • PC 기본 상태: 4열 목록 + 상단 필터/정렬 고정
  • PC 필터 선택 상태: 칩 하단 팝오버 오픈
  • 내 글 상태: `내 글`/`제안 N건` 표시 + 더보기 메뉴에서 수정/삭제(이력 0건)
  • 수정 잠금 상태: 받은 제안 이력이 있으면 `수정` 비활성 + 안내 문구 노출
  • 삭제 잠금 상태: 받은 제안 이력이 있으면 `삭제` 비활성 + 마감 유도 안내 노출
  • 끌어올리기 확인 상태: 확인 팝업에서 취소/진행 분기
  • 삭제 확인 상태: 받은 제안 이력 0건에서만 삭제 확인 팝업 노출
  • 로딩: 스켈레톤 카드 표시
  • 빈 상태: 결과 없음 + 등록 CTA
  • 에러: 네트워크 오류 + 다시 시도
