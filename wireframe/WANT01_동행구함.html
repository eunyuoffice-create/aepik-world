WANT01 동행 구함
이벤트 현장 동행자 모집 & 매칭 플랫폼
1
[Header] 앱 헤더
  • 타이틀: "동행 구함"
  • 우측: 검색 아이콘 (Search icon)
  • 배경: 흰색, 하단 경계선 (#EEF3F8)
2
[Tab] 탭 바
  • 탭 1: "전체" (기본 활성)
  • 탭 2: "내 동행" (사용자가 신청한 동행 목록)
  • 활성 탭: 파란색 하단 경계선 (#2F8EC1) 강조
  • 비활성 탭: 회색 텍스트 (#8EA2B7)
3
[Filter] 필터 바
  • 필터 칩: 전체 / 카테고리 / 작품 / 이벤트 / 지역
  • 활성: 파란 배경 + 흰 텍스트 (#2F8EC1)
  • 비활성: 회색 배경 (#F7FAFD) + 어두운 텍스트
  • 가로 스크롤 가능 (overflow-x: auto)
4
[Sort] 정렬 + 결과 수
  • 좌: "전체 24개" (검색 결과 수)
  • 우: "최신순 ▾" (정렬 드롭다운)
  • 선택 옵션: 최신순 / 인기순 / 마감임박순
5
[Card] 동행 카드 리스트
  • 카드 구조:
    • 좌: 썸네일 이미지 (80x80)
    • └ 좌상단 카테고리 뱃지 (애니/아이돌/배우/게임)
    • └ 중앙: 모집상태 뱃지 (초록색 "모집중")
    • └ 제목 (최대 2줄)
    • └ 연결 이벤트 + 날짜 (Calendar icon)
    • └ 작품 · 캐릭터
    • └ 모집 현황 (예: 1/3명 신청)
    • └ D-day 뱃지 (우상단, 빨간 배경)
6
[FAB] 등록 버튼 (플로팅)
  • 위치: 우하단 (bottom: 72px, right: 16px)
  • 배경: 파란색 (#2F8EC1)
  • 아이콘: 플러스 기호 (+)
  • 클릭 시: 동행글 등록 바텀시트 오픈
  • 비로그인 시: 로그인 유도 팝업 (AUTH01)
  • 로그인 + 미인증 시: 본인인증 유도 바텀시트 (JIT)
7
[BottomSheet] 동행글 등록 바텀시트
  • 큰 사이즈 바텀시트 (스크롤 가능, 약 90% 높이)
  • mode=create|edit 공용 폼 (수정 시 기존 값 prefill)
  • 필드 목록:
    • 연결 이벤트 검색 (dropdown / search)
    • 제목 (텍스트 input)
    • 카테고리 (4개 칩 선택)
    • 작품 & 캐릭터 (dropdown 또는 search)
    • 일시 (날짜 picker + 시간대 칩: 오전/오후/저녁/종일)
    • 장소 (텍스트 input)
    • 모집 인원 (stepper: 1~5명)
    • 설명 (textarea)
    • 사진 첨부 (max 3장, 이미지 업로드 버튼)
  • 하단 버튼: "동행 등록"/"수정 저장" (mode 기반)
  • 수정 잠금: 신청 이력 1건 이상이면 핵심 조건(일시/장소/모집인원) 비활성 + 안내 문구 노출
8
[Nav] 하단 네비게이션
  • 5개 탭: Home / Search / Companion(활성) / Notify / My
  • 활성 탭: 파란 아이콘 + 파란 텍스트 (#2F8EC1)
  • 비활성 탭: 회색 아이콘 + 회색 텍스트
화면 상태
상태 설명
정상 동행글 목록 표시, 필터/정렬 적용 가능
빈 상태 검색 결과 없음 또는 필터 적용 후 결과 0개 시
로딩 초기 진입 또는 필터 변경 시
에러 서버 오류 또는 네트워크 문제
비로그인 게시글 조회는 가능, "내 동행" 탭/FAB/신청 시 로그인 유도
미인증 로그인 + 미인증 상태에서 FAB/신청 시도 시 본인인증 유도 바텀시트
수정 잠금 신청 이력 1건 이상인 내 동행글은 핵심 조건 수정 불가, 상세 설명만 수정 가능
연관 화면
이동 대상 조건 화면코드
동행 상세 동행 카드 탭 → 동행 상세 페이지 이동 WANT02
이벤트 상세 동행글의 "연결 이벤트" 클릭 → 해당 이벤트 상세 페이지 이동 EVNT02
이벤트 목록 필터 "이벤트" 칩 클릭 → 이벤트 필터 조회로 이동 EVNT01
검색 Header의 검색 아이콘 클릭 → 검색 페이지 이동 SRCH01
메인 홈 하단 네비게이션 "Home" 탭 클릭 HOME01
회원가입/로그인 비로그인 상태에서 "내 동행"/FAB/신청 시도 AUTH01~02
마이페이지 하단 네비게이션 "My" 탭 클릭 USER01
채팅 하단 네비게이션 "채팅" 탭 클릭 CHAT01
