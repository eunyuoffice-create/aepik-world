1
[Header] 앱 헤더
  • 좌측: 뒤로가기 버튼 (이전 화면 이동)
  • 중앙: "공구 상세" 타이틀
  • 우측 아이콘:
    • 공유(Share) 아이콘 → 링크 공유 팝업 (카톡, URL 복사 등)
    • 신고(Report) 아이콘 → 신고 모달
2
[ImageCarousel] 이미지 갤러리
  • 상품 이미지 스와이프 기능 (좌우)
  • 하단 페이지 인디케이터: N/M 형식
  • 1:1 비율 이미지 영역
  • 이미지 없는 경우: 회색 배경 플레이스홀더
3
[Tags] 공구 타입 & 카테고리 태그
  • 첫 번째 태그: [일반공구] 또는 [세트공구] (Blue 배경)
  • 두 번째 태그: 카테고리 (애니/만화/소설 / 아이돌/아티스트 / 배우/무대 / 게임)
  • 각 카테고리별 색상: Purple / Pink / Amber / Cyan
4
[ProductInfo] 공구 기본 정보
  • 공구명 (Bold, Large 텍스트)
  • 가격:
    • 일반 공구: ₩N/1인
    • 세트 공구: ₩N/캐릭터
  • 배송비 표시: "배송비 포함" 또는 "배송비 별도 ₩N"
5
[OrganizerCard] 진행자 정보
  • 프로필 이미지 (48×48px, 원형)
  • 닉네임
  • 신뢰 배지 (Lv.1~3)
  • 거래 횟수 (예: 거래 12회)
  • 탭 시: 진행자 프로필 페이지(USER02) 이동
6
[ParticipationInfo] 참여 현황 (일반 공구)
  • 프로그레스 바: 큰 크기, Blue 채우기 (80% 이상일 경우 Green)
  • 참여 현황: N/M명 (X%) 표시
  • 최소 인원: "최소 20명" 형식
  • 마감일: "YYYY-MM-DD (D-N)" 형식
  • 배송 예정일: "YYYY-MM-DD" 또는 "미정"
6-B
[CharacterSlots] 캐릭터 선점 현황 (세트 공구)
  • 각 행 구성:
    • 캐릭터명
    • 프로그레스 바 (선점 현황)
    • 선점 상태: "선점 가능", "N/M 남음", 또는 "완료" 뱃지
  • 마감된 캐릭터는 회색 처리 (비활성화)
  • 마감일 & 배송 예정일 하단 표시
7
[Description] 상세 설명
  • 진행자가 작성한 공구 상세 내용
  • Plain text 또는 마크다운 포맷 지원
  • 최대 높이 제한 후 "더보기" (scrollable)
8
[ParticipantsList] 참여자 목록
  • 현재 참여자 프로필 아바타 나열 (원형, 32×32px)
  • 최대 4명 표시 후 "+N명 더보기" 링크
  • "+N명 더보기" 탭 시: 전체 참여자 목록 모달
9
[BottomCTA] 하단 CTA 바 (Fixed)
  • 좌측: 찜 버튼 (♡ 또는 ♥ 토글)
  • 우측: 주 액션 버튼 (Full-width)
    • 일반 공구: [참여하기] (Blue)
    • 세트 공구: [캐릭터 선점하기] (Blue)
    • 마감된 공구: [마감된 공구] (Disabled Gray)
    • 이미 참여 중: [참여 취소] (Outline)
    • 본인 글: [관리하기] + [수정] (잠금 시 수정 비활성)
  • 비로그인: 로그인 유도 모달
  • 미인증: 본인인증 유도 모달
  • 수정 잠금 안내: "참여 이력이 있어 핵심 조건을 수정할 수 없어요. 상세 설명이나 공지로 안내해 주세요."
모달 & 바텀시트
  • 참여 확인 바텀시트 (일반 공구):
    • 상품명 + 단가
    • 참여 수량 스피너 (1~N개)
    • 총 결제금액 계산
    • [참여 확정] 버튼
    • 안내: "결제는 진행자와 직접 정산합니다"
  • 캐릭터 선점 바텀시트 (세트 공구):
    • 선점 가능 캐릭터만 활성화
    • 캐릭터별 수량 선택 (체크박스 + 스피너)
    • 선택 시 총 결제금액 자동 계산
    • [선점 확정] 버튼
  • 공유 팝업: 카톡, URL 복사, 기타 SNS
  • 신고 모달: 신고 사유 선택 + 상세 입력
