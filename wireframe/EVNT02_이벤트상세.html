[Header] 헤더
  • 좌측: 뒤로가기 버튼 (ChevronLeft SVG) → EVNT01
  • 중앙: 스크롤 시 제목 표시 (처음엔 투명)
  • 우측: 공유 (Share2 SVG) + 더보기 (MoreVertical SVG)
    • 더보기 옵션: 신고/수정(본인만)/삭제(본인만)
2
[HeroImage] 대표 이미지
  • 전체 너비, 높이 200px, 회색 플레이스홀더
  • 캐러셀 형태 (다중 이미지 지원)
  • 하단 페이지 지시 점 (첫 번째 활성)
  • 이미지 없을 시: 카테고리 그라데이션 + 타입 아이콘
3
[BasicInfo] 기본 정보
  • 타입 태그: 범주 컬러 배경 (예: "생일카페" pink bg #FDF2F8, text #DB2777)
  • 이벤트명: 20px bold, 검은색
  • 대상: 작품 > 캐릭터 형식, 파란색, 탭 가능
  • 카테고리 뱃지: Purple pill (예: "애니/만화/소설")
  • 등록자: 아바타(24px 원) + 닉네임 + 작성일
  • 찜 수: Heart icon + 숫자
4
[ScheduleInfo] 일정 정보
  • Calendar icon + 날짜 범위 (예: "2026.03.15 ~ 2026.03.22")
  • 상태 뱃지:
    • 진행 중: 초록 배경 #DCFCE7, 텍스트 #16A34A
    • D-N (대기 중): 파란 배경 #EAF7FF, 텍스트 #2F8EC1 (예: D-8)
    • 종료: 회색 배경 #EEF3F8, 텍스트 #45617E
  • Clock icon + 운영 시간 (예: "10:00 ~ 20:00")
  • [캘린더 추가] 버튼: .ics 다운로드 또는 Google/Apple Calendar 연동
5
[LocationInfo] 장소 정보
  • MapPin icon + 주소
  • 주소 우측 Copy 버튼
  • 카카오맵 미니맵: 160px 높이, 반경 12px, 중앙에 핀 마커
  • 탭 시: 카카오맵 앱/웹으로 길찾기 연결
6
[Description] 상세 설명
  • 자유 텍스트, 3줄에서 expand/collapse 가능
  • "더보기" 텍스트 링크 (파란색)
  • 외부 링크: ExternalLink icon + URL (파란색, 밑줄)
7
[RelatedItems] 관련 교환/공구/동행
  • 헤더: "이 이벤트 관련 교환/공구" + "더보기 >" 링크
  • 수평 스크롤 카드 (max 4개)
  • 각 카드: 이미지 플레이스홀더(120x120) + 제목 + 타입 태그(교환/공구)
  • 관련 항목 없을 시: 숨김
  • "더보기" 탭: EXCH01/GRPB01로 필터링된 목록
  • 추가 헤더: "이 이벤트 관련 동행" + "동행 더보기 >"
  • 동행 카드 탭: WANT02, "동행 더보기" 탭: WANT01(event_id 필터)
8
[ActionBar] 액션 바
  • 고정 하단, 흰 배경, 상단 경계선 1px solid #EEF3F8, padding 12px 16px
  • 좌측: Heart icon + "찜" 텍스트 (탭 가능)
  • 우측: [동행 찾기] 보조 버튼 (WANT01, event_id 쿼리) + [공유하기] 파란 버튼
  • 본인 이벤트일 시: [수정]/[삭제] 버튼으로 변경
  • 종료된 이벤트: share 버튼 비활성(회색)
