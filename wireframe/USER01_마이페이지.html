USER01 — 마이페이지
라우트: /my, /my/edit, /user/[userId]  |  비로그인 접근: /my 제한, /user 가능  |  Phase 1
1
[Header] 앱 헤더 (마이페이지 메인)
  • 좌측: 페이지 제목 "마이페이지"
  • 우측 아이콘: 설정 톱니바퀴 아이콘 (파란 배경)
  • 설정 아이콘 탭 시: USER03 설정 화면 이동
2
[ProfileCard] 프로필 카드
  • 72px 원형 프로필 이미지 (미등록 시 기본 아바타)
  • 닉네임 18px Bold + 본인인증 체크 뱃지 (16px, 파란색)
  • 신뢰 지표: "교환 N회 · 긍정 N% · 가입일 YYYY.MM"
  • 최애 표시: "최애: [캐릭터/아이돌명]" (있을 때만 표시)
  • "프로필 편집" Secondary Small 버튼 → 모드 B 이동
3
[QuickCount] 퀵 카운트
  • 가로 3등분 카드 (탭 가능)
  • 항목: "내 교환글 N" / "내 공구 N" / "내 동행글 N"
  • 카드 탭 시: USER02 해당 탭으로 이동
4
[MenuList] 메뉴 리스트
  • 4개 그룹 구조 (내 활동 / 알림 / 지원 / 정보)
  • 내 활동 그룹:
    • 교환 내역 → USER02 교환내역 탭
    • 받은 후기 → USER02 후기 탭
    • 찜 목록 → USER02 찜 탭 (찜 카운트 뱃지 표시)
    • 보낸/받은 교환 제안 → EXCH05
  • 알림 그룹: 키워드 알림 설정 → USER03 (등록 키워드 N개 뱃지)
  • 지원 그룹: 공지사항, 고객센터 → STAT01
  • 정보 그룹: 이용약관, 개인정보처리방침, 앱 버전 (현재 버전 표시)
5
[Button] 로그아웃
  • Ghost 텍스트 버튼 "로그아웃" (메뉴 하단)
  • 탭 시 확인 팝업: "로그아웃 하시겠습니까?" / [취소] [로그아웃]
  • 확인 시: JWT 토큰 삭제, AUTH01로 이동
6
[Header] 편집 헤더
  • 좌측: 뒤로가기 화살표 (마이페이지로 복귀)
  • 중앙: "프로필 편집"
  • 우측: "저장" 텍스트 버튼 (변경 사항 있을 때만 활성)
7
[ImageEdit] 프로필 이미지 변경
  • 100px 원형 이미지, 중앙 정렬
  • 우하단 28px 원형 버튼에 카메라 아이콘
  • 탭 시 액션시트: 카메라 촬영 / 앨범에서 선택 / 기본 이미지로 변경
  • 업로드 제한: 5MB, JPG/PNG/WebP, 리사이즈 400x400
8
[Input] 닉네임 변경
  • 인풋 필드: 현재 닉네임 pre-fill, 44px 높이
  • 글자 수 카운터: 우측 하단 "N/12"
  • 실시간 중복 확인: debounce 500ms, 사용 가능(초록) / 이미 사용 중(빨강)
  • 규칙: 2~12자, 한글/영문/숫자, 특수문자/공백 불가
  • 금칙어 검사: 서버사이드 필터링
  • 변경 제한: 30일에 1회만 변경 가능
9
[Input] 최애 표시 설정
  • 라벨: "최애 표시 (프로필에 공개)"
  • 자유 입력, 최대 20자
  • 예시 placeholder: "예: 영지 (블루아카이브)"
  • 인풋 우측 X 버튼으로 비우기
  • 빈 값 저장 시 프로필에서 최애 행 숨김
10
[Tags] 관심 카테고리 변경
  • 현재 선택된 4개 카테고리 태그 표시 (Purple/Pink/Amber/Cyan)
  • "변경" 버튼 탭 시 카테고리 선택 바텀시트 (AUTH02 Step 4와 동일)
  • 4개 카테고리 카드, 복수 선택 가능
11
[Header] 타인 프로필 헤더
  • 좌측: 뒤로가기 화살표
  • 중앙: 닉네임 표시
  • 우측: 더보기 메뉴 (점 3개) → 신고하기 옵션
12
[ProfileCard] 타인 프로필 영역
  • 80px 원형 프로필 이미지
  • 닉네임 18px Bold + 인증 뱃지 (해당 시)
  • 최애 표시 (있을 시 표시)
  • 신뢰 지표: "교환 N회 · 긍정 N% · 가입일 YYYY.MM"
  • 가입일 라벨: "YYYY년 MM월 가입"
13
[Tabs+Grid] 등록 매물 탭 및 그리드
  • 탭 바: "교환글 N" / "공구 N" / "동행글 N"
  • 각 탭 내 2열 그리드 카드 (ProductCard)
  • 활성 매물만 표시 (완료/삭제 제외)
  • 빈 상태: "등록된 [교환글/공구/동행글]이 없습니다"
  • 카드 탭 시: 해당 상세 화면 (EXCH02/GRPB02/WANT01) 이동
14
[Reviews] 받은 후기
  • 후기 카드: 긍정/부정 아이콘 + 작성자 텍스트 + 날짜
  • 긍정 아이콘: 파란 배경 + + 기호
  • 부정 아이콘: 빨강 배경 + - 기호
  • 정렬: 최신순
화면 상태
  • 로딩 중: 프로필 카드 + 메뉴 스켈레톤 UI (회색 박스 펄스 애니메이션)
  • 에러: "연결이 불안정합니다" + 재시도 버튼
  • 비로그인 (/my 접근): 로그인 유도 바텀시트 (AUTH01-D)
  • 타인 프로필 빈 상태: 매물 0건 시 "등록된 매물이 없습니다"
  • 탈퇴 사용자: "탈퇴한 사용자입니다" 안내
  • 정지 사용자: "이용이 제한된 사용자입니다" 안내
