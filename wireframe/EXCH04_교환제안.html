EXCH04 교환 제안
교환 상세(EXCH02)에서 진입하는 전용 제안 작성 화면. 내 교환글 매칭/새 제안 등록 분기를 모두 제공한다.
1
[Header]헤더
  • 뒤로가기/닫기 액션 제공
  • 작성 중 이탈 시 확인 다이얼로그
2
[TargetCard]대상 교환글 요약
  • 상대 교환글 썸네일 + 제목 + 핵심 조건 요약 표시
  • 내가 어떤 글에 제안 중인지 항상 고정 노출
3
[Tabs]제안 방식 탭
  • 내 교환글 매칭(`listing`) / 새 제안 등록(`manual`) 2탭
  • 탭 전환 시 입력 폼 영역이 분기된다.
4
[ListSelect]내 교환글 선택
  • 내 등록글 목록에서 1개 선택
  • 이미 해당 글에 제안한 항목은 `제안 중` 뱃지 + 선택 비활성
  • 내 교환글이 없으면 Empty + EXCH03 이동 CTA 노출
5
[PCLayout]PC 3열 레이아웃
  • 좌: 대상/내 교환글 목록, 중: 입력 폼, 우: 제안 요약 + 제출 버튼
  • 모바일과 동일한 데이터/검증 규칙을 유지
6
[TradeStructure]거래 구조/수량
  • 구조: `1:1` / `1:N` / `N:1`
  • 제공 수량/희망 수량 필수 입력
7
[CashDiff]차액 제안
  • 없음 / `+금액` / `-금액` 선택
  • 제안 단위로 저장되며 최종 확정은 채팅에서 협의
8
[InstantMeetup]오늘 직교환
  • 당일만 ON 가능
  • 이벤트/시간대/대략 위치(역/랜드마크) 입력
  • 정확 주소 입력은 금지하고 채팅에서 최종 조율
  • 수락 후 채팅 상단에 관련 배너 노출
9
[MessageField]메시지 입력
  • 선택 입력, 최대 100자
  • 조건 보충/시간 협의 메모 용도
10
[PrimaryCTA]제안 발송 CTA
  • 필수 입력 충족 시 활성
  • 성공 시 EXCH05 받은/보낸 제안함 반영
화면 상태
기본 상태: listing 탭 기본 활성 + 내 교환글 목록 노출
manual 상태: 사진/상태/수량 입력 폼 노출
내 교환글 없음: Empty + EXCH03 이동 CTA
제안 제한 상태: 동일 글 동시 pending 3건 초과 시 CTA 비활성 + 안내 문구
재시도 제한 상태: 동일 글 기준 24시간 3회 / 7일 6회 초과 시 요청 차단
오늘 직교환 비활성: 당일 외 날짜에는 토글 비활성
제출 중: 버튼 로딩 + 중복 제출 방지
오류 상태: 유효성/서버 에러 메시지 + 재시도
