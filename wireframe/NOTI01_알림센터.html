알림 센터 (NOTI01)
푸시 알림 및 인앱 알림 조회 & 관리 화면
1
[Header] 알림 헤더
  • 좌측: ChevronLeft 뒤로가기 버튼 (USER03 설정 페이지에서 진입)
  • 중앙: "알림" 텍스트 (16px, 600 weight)
  • 우측: Settings 아이콘 (설정 이동) + Bell 아이콘에 미읽음 수 뱃지
    • 뱃지: 빨간색 배경 (#EF4444), 흰색 텍스트, 원형 (18px)
    • 미읽음 수 3개 이상은 "99+"로 표시
2
[Action] 모두 읽음 버튼
  • 우측 상단 텍스트 버튼 (12px, 600 weight, 파란색 #2563EB)
  • 탭 시 전체 알림을 읽음 상태로 변경
  • API: PATCH /api/notifications/read-all
3
[Filter] 필터 탭바
  • 5개 탭: 전체 / 교환 / 공구 / 이벤트 / 시스템
  • 활성 탭: 파란색 텍스트 + 하단 2px 파란색 바
  • 비활성: #999 회색 텍스트
  • 수평 스크롤 가능 (모바일)
  • 탭 선택 시 알림 목록 즉시 필터링 (API: GET /api/notifications?type=exchange)
4
[List] 알림 카드 리스트
  • 각 아이템 구조:
    • 좌측: 알림 유형 아이콘 원형 (32px, 카테고리별 색상)
    • 중앙: 제목(1줄, 13px, 600w) + 본문(2줄 ellipsis, 12px, #666) + 시간(11px, #999)
    • 우측: 관련 이미지 썸네일 (40px, border-radius 8px)
  • 미읽음 상태: 배경 #F0F7FF + 좌측 파란 점(6px) 표시
  • 읽음 상태: 배경 #fff
  • 아이템 탭 시 해당 상세 페이지 딥링크 진입 + 읽음 처리 (PATCH /api/notifications/{id}/read)
5
[Divider] 날짜 그룹핑 섹션
  • 알림을 날짜별로 그룹핑: "오늘" / "어제" / "이번 주" / "이전 알림"
  • 섹션 헤더: 배경 #F9FAFB, padding 8px 16px, 텍스트 12px / #999
  • 한국 시간대 기준 자정 기준으로 자동 분류
6
[Interaction] 필터 탭 활성화
  • 필터 탭 선택 시 목록이 즉시 업데이트
  • 선택된 탭은 파란색, 하단 바 표시
  • API 쿼리: type=exchange / type=group_buy / type=event / type=system
7
[Interaction] 스와이프 삭제
  • 알림 아이템 좌 스와이프 → 빨간 "삭제" 버튼 노출
  • 버튼 너비 56px, 높이 아이템 높이와 동일
  • 삭제 확인 없이 즉시 삭제 (Optimistic UI)
  • API: DELETE /api/notifications/{id}
8
[Empty] 빈 상태
  • 센터 정렬된 Bell 아이콘 (64px, #D1D5DB)
  • 제목: "아직 알림이 없어요" (14px, 600w, #9CA3AF)
  • 설명: "교환, 공구, 이벤트 알림이 여기에 표시됩니다" (12px, #D1D5DB)
9
[Legend] 알림 유형 범례
  • 10개 알림 유형별 아이콘 + 색상 표시
  • 교환 매칭(Blue Repeat → EXCH02) / 교환 제안(Blue Send → EXCH05)
  • 키워드 매칭(Purple Search → EXCH02) / 구함 매칭(Green Target → WANT01)
  • 공구(Purple Package → GRPB02) / 채팅(Blue MessageCircle → CHAT02)
  • 이벤트(Amber Calendar → EVNT02) / 배너(Pink Palette → BANR01)
  • 신고(Red Shield → USER02) / 시스템(Gray Info → 공지)
  • 2열 그리드 레이아웃, 각 항목 11px 폰트
10
[Nav] 하단 네비게이션
  • 5탭: 홈(Home) / 교환(Repeat) / 이벤트(Calendar) / 채팅(MessageCircle) / 마이(User)
  • Lucide SVG 아이콘 20×20px, 라벨 10px
  • 알림 화면은 헤더에서만 진입 가능 (탭 활성 상태 없음)
  • HOME01 레퍼런스 참고
-
[States] 화면 상태
  • 로딩 중: 목록 상단 스켈레톤 로더 (회색 bar 4개, 8px high)
  • 빈 상태: Item 9 (Empty state) 참고
  • 에러 상태: 상단 빨간 토스트 "알림 로드 실패. 다시 시도해주세요." (dismiss 버튼)
  • 네트워크 오프라인: 회색 배너 "오프라인 상태입니다" (상단)
