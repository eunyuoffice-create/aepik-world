STAT01 — 정적 페이지
라우트: /terms, /privacy, /support, /about  |  비로그인: 가능
1
[Header] 정적 페이지 공통 헤더
  • 좌측: 뒤로가기 화살표 ()
  • 중앙: 페이지 제목 (이용약관 / 개인정보처리방침 / 고객센터 / 서비스 소개)
  • 상단 고정
  • 뒤로가기 탭 시: 이전 화면 또는 HOME01 이동
2
[Document] 이용약관/개인정보처리방침 본문
  • 목차 섹션: 클릭 가능한 섹션 링크 (제1장 총칙, 제2장 서비스 이용 등)
  • 본문: 법적 약관 텍스트, 섹션별 구조화
  • 스크롤 가능한 콘텐츠 영역
  • 하단: 시행일 표시
  • /terms 경로: 이용약관
  • /privacy 경로: 개인정보처리방침
  • AUTH02(회원가입 약관 동의)에서 [약관 보기] 탭 시 이 화면으로 이동
3
[Tabs] 고객센터 탭 바
  • 2개 탭: FAQ | 1:1 문의
  • 활성 탭: Blue 하단선, 비활성: Gray 텍스트
  • 탭 전환 시 해당 컨텐츠 렌더링
4
[SearchBar] FAQ 검색 바
  • 플레이스홀더: "FAQ 검색어 입력"
  • 좌측 아이콘
  • 입력 시 실시간 검색 결과 필터링
5
[Filter] FAQ 카테고리 칩
  • 카테고리: 전체(기본 선택) / 교환 / 공구 / 계정 / 이벤트 / 기타
  • 활성 칩: Blue 배경 + 흰 텍스트, 비활성: 회색 보더 + 흰 배경
  • 칩 탭 시 해당 카테고리 FAQ만 필터링
  • 가로 스크롤 가능
6
[Accordion] FAQ 아코디언 리스트
  • 각 항목 구성:
    • 카테고리 태그 (색상: 교환-Purple, 공구-Blue, 계정-Amber, 기타-Pink 등)
    • 질문 텍스트
    • 화살표 아이콘 (접힘 상태)
  • 탭 시 아코디언 펼침, 답변 텍스트 표시
  • 다른 항목 탭 시 이전 항목 자동 접힘
  • 스크롤 가능한 리스트
7
[Form] 1:1 문의 폼
  • 필드 구성:
    • 문의 유형: 셀렉트 (선택해주세요 > 교환 관련 / 공구 관련 / 계정 관련 / 기타)
    • 제목: 텍스트 인풋
    • 내용: 텍스트 에어리어 (min-height: 100px)
  • 모든 필드: 1px #E5E7EB 보더, 8px border-radius, 14px 폰트
  • 비로그인 시: 폼 표시 전 로그인 유도 모달
8
[ImageUpload] 이미지 첨부 영역
  • 레이아웃: 점선 보더 박스 (1px dashed #D1D5DB), 배경 #FAFAFA
  • 아이콘: 카메라 ()
  • 텍스트: "이미지를 선택하세요" + "최대 3장" (힌트)
  • 탭 시: 디바이스 카메라 또는 갤러리 열림
  • 선택 후: 이미지 프리뷰 표시
9
[Button] 문의 제출 버튼
  • 유형: Primary Button (Blue #3B82F6 배경 + 흰 텍스트)
  • 크기: Large, Full Width (48px height)
  • 텍스트: "문의 제출"
  • 탭 시: 폼 유효성 검사 → 성공 시 제출 완료 메시지 + 내 문의 내역에 추가
10
[List] 내 문의 내역
  • 섹션: "내 문의 내역" 헤더
  • 항목 구성:
    • 상태 뱃지: [접수] (Blue bg) / [답변완료] (Green bg)
    • 문의 제목
    • 작성일 · 상태 메타 (회색 텍스트)
  • 항목 탭 시: 문의 상세 페이지 (별도)
  • 빈 상태: "제출한 문의가 없습니다" 메시지
  • 비로그인: 이 섹션 미표시
11
[Hero] 서비스 소개 히어로
  • 배경: 연한 회색 (#F9FAFB)
  • 콘텐츠:
    • 로고: "애픽 월드" (32px 폰트, Blue #2563EB)
    • 태그라인: "팬이 만드는 굿즈 세상" (12px, Gray)
12
[Grid] 4가지 기능 카드 (2x2)
  • 카드 배열: 2열 그리드, gap 12px
  • 각 카드:
    • 아이콘 (SVG, 20px)
    • 기능명: 교환 / 공구 / 구함 / 이벤트
    • 배경: 연한 회색 + 보더 1px #E5E7EB
  • 카드 탭 시: 해당 기능 페이지로 이동 (EXCH01 / GRPB01 / WANT01 / EVNT01)
13
[Section] 지원 카테고리 섹션
  • 색상 도트 4개 (28px, 원형):
    • Purple #8B5CF6 - 애니/만화/소설
    • Pink #EC4899 - 아이돌/아티스트
    • Amber #F59E0B - 배우/무대
    • Cyan #06B6D4 - 게임
  • 도트 하단: 카테고리명 텍스트 (10px, Gray)
  • 섹션 헤더: "지원 카테고리"
14
[CTA] 시작하기 버튼
  • 유형: Primary Button (Blue #3B82F6)
  • 크기: Large, Full Width
  • 텍스트: "지금 시작하기"
  • 탭 시:
    • 로그인 상태: HOME01 이동
    • 비로그인: AUTH01(로그인) 이동
화면 상태
  • 이용약관/개인정보처리방침: 로딩 중 스켈레톤, 콘텐츠 로드 후 표시
  • FAQ 검색: 검색 결과 없음 시 "검색 결과가 없습니다" 메시지
  • 1:1 문의 — 비로그인: 폼 상단에 로그인 유도 배너 (파란 배경 + CTA)
  • 내 문의 내역 — 빈 상태: "제출한 문의가 없습니다" + 빈 상태 아이콘
  • 에러 상태: API 실패 시 "데이터를 불러올 수 없습니다" + 재시도 버튼
