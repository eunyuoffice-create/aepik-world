# 애픽 (Aepik) — 픽셀 아트 UI 컨셉 가이드 v1.0

> **"덕질의 시작, 8비트로 Pick!"**
>
> 레트로 게임 감성의 픽셀 아트 UI로 오타쿠/덕후 타겟층의 감성을 자극하는 디자인 시스템

## 1. 컨셉 방향

### 왜 픽셀 아트인가?

| 이유 | 설명 |
|------|------|
| **타겟 감성 적중** | 애니·게임 팬은 도트 그래픽에 친숙하고 호감도 높음 (스타듀밸리, 언더테일, 포켓몬 등) |
| **차별화** | 번개장터·당근마켓 등 경쟁사와 완전히 다른 비주얼 → 한눈에 기억됨 |
| **커뮤니티 바이럴** | "이 앱 UI 봐" 같은 자발적 공유 유도 → 트위터/인스타 스크린샷 확산 |
| **1인 개발 유리** | 픽셀 아트는 사진급 일러스트보다 제작 난이도 낮음, 에셋 재활용 용이 |
| **브랜드 아이덴티티** | 독보적인 시각 정체성 확립 가능 |

### 레퍼런스

| 레퍼런스 | 참고 포인트 |
|---------|-----------|
| **스타듀밸리** | 따뜻한 색감, 아이콘 디자인, 인벤토리 UI |
| **무한의 목장** | 한국형 픽셀 게임 UI, 카드 레이아웃, 뱃지 시스템 |
| **포켓몬 시리즈 (GBA)** | 메뉴 구조, 상태 표시, 도감 UI |
| **언더테일** | 텍스트 표현, 대화 UI, 흑백+컬러 포인트 |
| **Habitica** | 픽셀 RPG + 실용 앱의 결합 사례 |
| **NH카드 올바른 HOMETOWN 광고** | 픽셀 아트로 실용적 정보 전달한 마케팅 사례 |

### 컨셉 키워드

## 2. 컬러 팔레트

### 메인 컬러

픽셀 아트에 적합한 제한된 팔레트. 16~32색 기반으로 설계.

#### Primary (브랜드 컬러)

| 이름 | HEX | 용도 |
|------|-----|------|
| **Aepik Coral** | `#FF6B6B` | 메인 브랜드 컬러. CTA 버튼, 로고, 강조 |
| **Aepik Coral Dark** | `#CC4455` | Coral의 그림자/호버 |
| **Aepik Coral Light** | `#FF9B9B` | Coral의 하이라이트 |

#### Secondary (보조 컬러)

| 이름 | HEX | 용도 |
|------|-----|------|
| **Sky Blue** | `#4ECDC4` | 교환 매칭, 성공 알림, 링크 |
| **Golden** | `#FFD93D` | 별점, 인증 뱃지, 끌어올리기 |
| **Lavender** | `#A78BFA` | 공구, 특별 이벤트, 프리미엄 |

#### Background & Neutral

| 이름 | HEX | 용도 |
|------|-----|------|
| **Cream** | `#FFF8E7` | 메인 배경 (따뜻한 크림색) |
| **Warm White** | `#FFFDF5` | 카드 배경 |
| **Sand** | `#F5E6C8` | 보조 배경, 구분선 |
| **Brown Dark** | `#5D4037` | 텍스트, 테두리 (픽셀 아트의 아웃라인) |
| **Brown Medium** | `#8D6E63` | 보조 텍스트, 비활성 |
| **Brown Light** | `#D7CCC8` | 비활성 배경, 플레이스홀더 |

#### Status Colors

| 이름 | HEX | 용도 |
|------|-----|------|
| **HP Green** | `#66BB6A` | 성공, 교환 완료, 긍정 후기 |
| **Danger Red** | `#EF5350` | 에러, 신고, 부정 후기 |
| **Warning Amber** | `#FFA726` | 경고, 마감 임박 |
| **Info Blue** | `#42A5F5` | 정보, 안내 |

### 컬러 적용 원칙

- 배경은 항상 따뜻한 톤 (Cream/Sand 계열) — 차가운 흰색 사용 금지
- 아웃라인은 `Brown Dark` 로 통일 — 픽셀 아트의 핵심
- 그림자는 각 컬러의 한 단계 어두운 톤으로 (드롭 쉐도우 대신 픽셀 그림자)
- 그라데이션 사용 금지 — 순수 플랫 컬러만 (픽셀 아트 원칙)

## 3. 타이포그래피

### 픽셀 폰트

| 폰트 | 용도 | 비고 |
|------|------|------|
| **DungGeunMo (둥근모꼴)** | 한글 본문 + 제목 | 무료, 한글 픽셀 폰트의 대표 |
| **Galmuri11** | 한글 본문 대안 | 11px 기반, 가독성 우수, SIL 라이선스 |
| **Galmuri9** | 작은 텍스트, 뱃지, 태그 | 9px 기반, 콤팩트 |
| **Press Start 2P** | 영문 제목, 로고 타입 | Google Fonts, 클래식 아케이드 |
| **Silkscreen** | 영문 본문, 버튼 | Google Fonts, 가독성 좋은 픽셀 |

### 폰트 사이즈 체계

```
제목 (H1):     DungGeunMo 24px  — 페이지 타이틀
부제목 (H2):   DungGeunMo 20px  — 섹션 제목
소제목 (H3):   DungGeunMo 16px  — 카드 제목, 리스트 제목
본문:          Galmuri11 14px   — 일반 텍스트
캡션:          Galmuri9 12px    — 보조 텍스트, 태그, 뱃지
버튼:          DungGeunMo 14px  — 버튼 라벨
```

### 폰트 적용 원칙

- 모든 텍스트에 `image-rendering: pixelated` 적용
- `font-smooth: never` / `-webkit-font-smoothing: none` — 안티앨리어싱 끄기
- 폰트 사이즈는 짝수 px만 사용 (픽셀 스케일링 깨짐 방지)
- 줄 간격(line-height)은 폰트 높이의 1.5배

## 4. 픽셀 아트 UI 컴포넌트

### 4.1 버튼

```
┌─────────────────────┐
│  ┌─────────────────┐ │   ← Brown Dark 아웃라인 (2px)
│  │   교환하기 ♥    │ │   ← Aepik Coral 배경
│  └─────────────────┘ │
│    ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓   │   ← 하단 픽셀 그림자 (Coral Dark)
└─────────────────────┘

[호버] 배경 → Coral Light, 커서 → 픽셀 포인터
[비활성] 배경 → Brown Light, 텍스트 → Brown Medium
```

#### 버튼 종류

| 종류 | 배경색 | 용도 |
|------|--------|------|
| Primary | Aepik Coral | CTA (교환하기, 공구 참여 등) |
| Secondary | Sky Blue | 보조 행동 (채팅하기, 더보기) |
| Outline | 투명 + Brown Dark 테두리 | 취소, 뒤로가기 |
| Golden | Golden | 끌어올리기, 프리미엄 |
| Danger | Danger Red | 삭제, 신고 |

## 7. 애니메이션 & 인터랙션

### 픽셀 스타일 애니메이션 원칙

| 원칙 | 설명 |
|------|------|
| **스프라이트 기반** | CSS 트랜지션 대신 프레임 단위 변화 (step() 이징) |
| **저프레임** | 부드러운 60fps 대신 8~12fps (레트로 느낌) |
| **과장된 동작** | 미세한 변화보다 확실한 변화 (바운스, 깜빡임) |
| **이즈 함수** | `steps(N)` 또는 `cubic-bezier` 없이 단계적 변화 |

### 주요 애니메이션

| 요소 | 애니메이션 | CSS |
|------|-----------|-----|
| 페이지 전환 | 와이프(좌→우) 또는 페이드(3단계) | `animation: pixelFade steps(3)` |
| 버튼 클릭 | 2px 아래로 눌림 + 그림자 사라짐 | `transform: translateY(2px)` |
| 카드 등장 | 아래에서 위로 팝업 (4단계) | `animation: slideUp steps(4)` |
| 매칭 알림 | 화면 흔들림 + 별 이펙트 | `animation: shake steps(4)` |
| 로딩 | 8프레임 회전 스프라이트 | 별/하트 아이콘 회전 |
| 스크롤 | 패럴랙스 없음 — 한 줄씩 스크롤 | 자연 스크롤 유지 |
| 탭 전환 | 선택 탭 바운스 (2프레임) | `animation: bounce steps(2)` |
| 좋아요(찜) | 하트 팝 (3프레임: 작게→크게→원래) | `animation: heartPop steps(3)` |

### CSS 유틸리티

```css
/* 픽셀 렌더링 */
.pixel {
  image-rendering: pixelated;
  image-rendering: crisp-edges;
  -webkit-font-smoothing: none;
  -moz-osx-font-smoothing: unset;
}

/* 픽셀 아웃라인 */
.pixel-border {
  border: 2px solid #5D4037;
  box-shadow:
    2px 2px 0px #5D4037,   /* 우하단 그림자 */
    -1px -1px 0px #FFFDF5; /* 좌상단 하이라이트 */
}

/* 픽셀 버튼 그림자 */
```

## 11. 접근성 고려사항

| 항목 | 대응 |
