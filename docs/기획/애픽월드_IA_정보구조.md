# 애픽 월드 (Aepik World) — Information Architecture (IA)

> **버전**: v1.3
> **기반 문서**: PRD v5.0, 유저 플로우 v1.4
> **작성일**: 2026-03-06
> **용도**: HTML 와이어프레임 구축 시 페이지 구조 및 라우트 설계 참고

## 1. 사이트맵 (전체 구조)

```
애픽 월드 (Aepik World)
│
├── `Home` 홈 (/)
│     ├── 배너 캐러셀 (생일/데뷔 축하)
│     ├── 다가오는 이벤트 (가로 스크롤)
│     ├── 인기 교환 (그리드)
│     ├── 진행 중 공구 (가로 스크롤)
│     └── 구함글 모아보기 (리스트)
│
├── `Repeat` 교환 (/exchange)
│     ├── 교환 목록 (/exchange)
│     ├── 교환 상세 (/exchange/[id])
│     └── 교환글 등록 (/exchange/new)
│           ├── Step 1: 보유 굿즈
│           ├── Step 2: 원하는 굿즈
│           └── Step 3: 거래 방식
│
├── `Package` 공구 (/group-buy)
│     ├── 공구 목록 (/group-buy)
│     ├── 공구 상세 (/group-buy/[id])
│     │     └── (세트 공구) 캐릭터 선점 UI
│     └── 공구 등록 (/group-buy/new)
│           ├── 유형 선택 (일반/세트)
│           ├── Step 1: 기본 정보
│           ├── Step 2: (세트) 캐릭터 설정
│           └── Step 3: 확인
│
├── `Search` 구함 (/wanted) [WANT01]
│     ├── 구함글 목록 (/wanted) — 전체 / 내 구함 탭, 필터·정렬, 카드 리스트
│     ├── 구함 상세 바텀시트 — 매칭 교환글 노출 + 교환 제안 / 채팅 문의
│     └── 구함글 등록 바텀시트 — 인라인 폼 (별도 페이지 이동 없음)
│
├── `Calendar` 이벤트 (/events)
│     ├── 이벤트 캘린더 (/events)
│     │     ├── 캘린더 뷰
│     │     ├── 리스트 뷰
│     │     └── 지도 뷰
│     ├── 이벤트 상세 (/events/[id])
│     └── 이벤트 등록 (/events/new)
│
├── `MessageCircle` 채팅 (/chat) [CHAT01~02]
│     ├── 채팅 목록 (/chat) [CHAT01]
│     │     ├── 필터 탭 (전체/교환/공구)
│     │     └── 채팅방 아이템 (아바타/닉네임/상품명/최신메시지/안읽은수)
│     └── 채팅방 (/chat/[roomId]) [CHAT02]
│           ├── 거래 상태 배너 (8개 상태)
│           ├── 메시지 영역 (텍스트/이미지/시스템)
│           ├── 교환 확정 카드
│           ├── 운송장 입력 카드
│           ├── 수취 확인 / 교환 완료 카드
│           └── 메시지 입력 바
│
├── `Bell` 알림 (/notifications) [NOTI01]
│     ├── 필터 탭바 (전체/교환/공구/이벤트/시스템)
│     ├── 알림 카드 리스트 (유형별 아이콘 + 제목 + 본문 + 시간 + 썸네일)
│     ├── 날짜 그룹핑 (오늘/어제/이번 주/이전)
│     ├── 모두 읽음 처리 + 스와이프 삭제
│     └── 빈 상태 + 알림 유형 범례
│
├── `Search` 검색 (/search)
│     ├── 검색 입력 + 자동완성
│     └── 검색 결과 (필터 + 정렬)
│
├── `User` 마이페이지 (/my) [USER01]
│     ├── A. 마이페이지 메인 (/my)
│     │     ├── 프로필 카드 (이미지/닉네임/인증뱃지/신뢰지표/최애)
│     │     ├── 퀵 카운트 (교환글/공구/구함글)
│     │     ├── 메뉴 리스트 (내 활동/알림/지원/정보)
│     │     └── 로그아웃
│     ├── B. 프로필 편집 (/my/edit)
│     │     ├── 프로필 이미지 변경
│     │     ├── 닉네임 변경 (30일 1회)
│     │     ├── 최애 표시 설정
│     │     └── 관심 카테고리 변경
│     ├── 내 활동 관리 [USER02]
│     │     ├── A. 내 교환글 (/my/exchanges)
│     │     ├── B. 내 공구 (/my/group-buys)
│     │     ├── C. 내 구함글 (/my/wanted)
│     │     ├── D. 교환 내역 (/my/history)
│     │     ├── E. 받은 후기 (/my/reviews)
│     │     └── F. 찜 목록 (/my/wishlist)
│     └── 설정 (/my/settings) [USER03]
│           ├── A. 설정 메인
│           │     ├── 알림 설정 (7개 토글 + 방해금지)
│           │     ├── 관심 카테고리
│           │     ├── 계정 관리 (비밀번호/소셜연동/로그아웃/탈퇴)
│           │     └── 정보 (약관/정책/라이선스/버전)
│           ├── B. 키워드 알림 설정 (/my/keywords)
│           └── C. 탈퇴 프로세스 (안내→사유→본인확인→완료)
│
├── `Palette` 배너/팝업 신청 (/banner/apply) [BANR01]
│     ├── 신청 유형 선택 (메인 배너 / 피드 배너 / 팝업, 복수 선택)
│     ├── 대상 정보 (작품/캐릭터 + 이벤트 종류)
│     ├── 슬롯 달력 (날짜별 잔여 슬롯 도트 표시)
│     ├── 이미지 업로드 + 저작권 가이드라인
│     ├── 비용 안내 (Phase 1 무료 → Phase 1.5 유료)
│     └── 내 배너/팝업 관리 (마이페이지 연계)
│
├── `FileText` 정적 페이지 [STAT01]
│     ├── 이용약관 (/terms) — 목차 앵커 + 장별 문서 뷰
│     ├── 개인정보처리방침 (/privacy) — 목차 앵커 + 항목별 문서 뷰
│     ├── 고객센터 (/support) — 탭 (FAQ / 1:1 문의)
│     │     ├── FAQ: 검색 + 카테고리 칩 (전체/교환/공구/계정/이벤트/기타) + 아코디언
│     │     └── 1:1 문의: 카테고리/제목/내용/이미지 폼 + 내 문의 내역
│     └── 서비스 소개 (/about) — 히어로 + 기능 카드 + 카테고리 쇼케이스 + CTA
│
├── `Lock` 인증
│     ├── 로그인 (/login) [AUTH01]
│     │     ├── A. 로그인 메인 (카카오/이메일)
│     │     ├── B. 이메일 로그인
│     │     ├── C. 비밀번호 찾기
│     │     └── D. 로그인 유도 바텀시트
│     ├── 회원가입 (/signup) [AUTH02]
│     │     ├── Step 1: 약관 동의
│     │     ├── Step 2: 계정 정보 (이메일 가입만)
│     │     ├── Step 3: 프로필 설정
│     │     ├── Step 4: 관심 카테고리 (선택)
│     │     └── 가입 완료
│     └── 본인인증 (모달/바텀시트, Just-in-time)
│
├── `User` 사용자 프로필 (타인) [USER01-C]
│     └── 사용자 프로필 (/user/[userId])
│           ├── 프로필 영역 (이미지/닉네임/인증뱃지/신뢰지표/가입일)
│           ├── 등록 매물 탭 (교환글/공구/구함글)
│           ├── 받은 후기
│           └── `AlertTriangle` 신고하기 (바텀시트)
│
└── `Shield` 어드민 (/admin) [별도 레이아웃]
      ├── 대시보드 (/admin)
      ├── 회원 관리 (/admin/users)
      ├── 매물 관리 (/admin/listings)
      ├── 공구 관리 (/admin/group-buys)
      ├── 신고 관리 (/admin/reports)
      ├── 작품/캐릭터 관리 (/admin/works)
      ├── 이벤트 관리 (/admin/events)
      ├── 배너 검수 (/admin/banners)
      ├── 후기 관리 (/admin/reviews)
      ├── 공지 관리 (/admin/notices)
      └── 통계 (/admin/stats)
```

## 2. 페이지별 상세 IA

### 2-1. 홈 (/)

```
[Header]
  ├── 로고 (애픽 월드)
  ├── 알림 아이콘 (`Bell`) + 뱃지
  └── 메뉴 아이콘 (☰)

[검색바]
  └── 플레이스홀더: "상품명, 작품, 캐릭터 검색"

[배너 캐러셀]
  ├── 배너 이미지 (1200×300)
  ├── 배너 제목 + 설명
  └── 인디케이터 (● ○ ○)

[다가오는 이벤트 섹션]
  ├── 섹션 헤더: "`Calendar` 다가오는 이벤트" + [더보기]
  └── 가로 스크롤 카드
        ├── 날짜
        ├── 이벤트명
        ├── 장소
        └── 카테고리 태그

[인기 교환 섹션]
  ├── 섹션 헤더: "`Flame` 인기 교환" + [더보기]
  └── 2열 그리드 카드
        ├── 상품 이미지 (1:1)
        ├── 거래유형 태그 + 카테고리 태그
        ├── 상품명 (2줄)
        ├── 작품 · 캐릭터
        └── ★ 평점 · 거래 횟수

[진행 중 공구 섹션]
  ├── 섹션 헤더: "`Package` 진행 중 공구" + [더보기]
  └── 가로 스크롤 카드
        ├── 상품 이미지
        ├── 공구유형 태그 (일반/세트)
        ├── 상품명
        ├── 참여현황 (N/M명)
        └── 프로그레스 바 (%)

[구함글 섹션]
  ├── 섹션 헤더: "`Search` 구함글 모아보기" + [더보기]
  └── 리스트 카드
        ├── [구함] 태그 + 구함글 제목
        └── 작성자 · 시간

[Bottom Navigation]
  ├── 홈 (`Home`)
  ├── 교환 (`Repeat`)
  ├── 이벤트 (`Calendar`)
  ├── 채팅 (`MessageCircle`)
  └── 마이 (`User`)
```

### 2-2. 교환 목록 (/exchange)

```
[Header]
  ├── 페이지 제목: "교환"
  └── 필터 아이콘

[필터 바] (가로 스크롤 칩)
  ├── 작품 ▾
  ├── 캐릭터 ▾
  ├── 굿즈 타입 ▾
  ├── 상품 상태 ▾
  ├── 거래 방법 ▾
  └── 차액 여부 ▾

[정렬]
  └── 최신순 / 인기순 / 가까운순

[교환글 그리드] (2열)
  └── 상품 카드 (동일 구조)

[FAB] (+) → 교환글 등록
```

### 2-3. 교환 상세 (/exchange/[id])

```
[Header]
  ├── ← 뒤로가기
  └── 공유 / 신고

[이미지 슬라이더]
  ├── 상품 이미지 (스와이프)
  └── 인디케이터 (● ○ ○)

[거래유형 태그] + [카테고리 태그]

[상품 정보]
  ├── 상품명
  ├── 작품 · 캐릭터
  └── 등록일 · 조회수

[상세 정보 박스]
  ├── 상태: S급 (미개봉)
  ├── 희망 교환: OO 캐릭터 OO 굿즈
  ├── 차액: ±O,OOO원 가능
  └── 거래 방식: 택배/직거래

[(직거래 시) 지도]
  └── 카카오맵 — 거래 희망 위치

[판매자 정보 박스]
  ├── 프로필 이미지 + 닉네임
  ├── 신뢰 등급 + 뱃지
  ├── ★ 평점 · 거래 횟수
  └── 최근 활동

[하단 고정 CTA]
  ├── [`Heart` 찜]
  ├── [`MessageCircle` 채팅]
  └── [`Repeat` 교환 제안]
```

### 2-4. 교환글 등록 (/exchange/new)

```
[Header]
  ├── ← 뒤로가기
  └── 페이지 제목: "교환글 등록"

[스텝 인디케이터]
  └── ① 보유 굿즈 → ② 원하는 굿즈 → ③ 거래 방식

--- Step 1: 보유 굿즈 ---

[사진 업로드]
  └── 최대 5장, 드래그 정렬

[작품 선택]
  └── 검색 인풋 + 자동완성 + "직접 추가" 버튼

[캐릭터 선택]
  └── 작품 선택 후 드롭다운 활성화

[굿즈 타입 선택]
  └── 포토카드 / 아크릴 / 캔뱃지 / 피규어 / ...

[상품 상태]
  └── S급 / A급 / B급 / C급 라디오

[보유 수량]
  └── 수량 입력 (기본 1)

[다음 버튼]

--- Step 2: 원하는 굿즈 ---

[원하는 작품/캐릭터]
  └── 검색 선택

[원하는 굿즈 타입]
  └── 드롭다운

[원하는 수량]

[차액 협의]
  ├── 체크박스: "차액 협의 가능"
  └── (체크 시) 희망 차액 금액 인풋 (+/- 선택)

[다음 버튼]

--- Step 3: 거래 방식 ---

[거래 방법]
  └── 택배 / 직거래 / 둘 다 가능

[(직거래 선택 시)]
  ├── 지도 — 위치 선택 (카카오맵)
  └── 설명 텍스트 인풋

[추가 조건]
  └── 텍스트 인풋 (선택)

[등록하기 버튼]
```

### 2-5a. 구함 게시판 (/wanted) [WANT01]

```
[Header]
  ├── "구함" 타이틀
  └── `Search` 검색 아이콘 → SRCH01

[탭 바]
  ├── 전체 (기본 활성) — 모든 유저의 구함글
  └── 내 구함 — 본인 작성 구함글 (로그인 필요)

[필터 바]
  ├── 작품 ▾ / 캐릭터 ▾ / 굿즈 타입 ▾ / 상태 ▾
  └── 상태 옵션: 구함중 / 매칭됨 / 만료

[정렬 + 결과 수]
  ├── 전체 N개
  └── 최신순 / 만료임박순

[구함 카드 리스트]
  ├── 좌측: 레퍼런스 이미지 (80×80) or 기본 아이콘
  ├── 상태 뱃지 (구함중:Green / 매칭됨:Blue / 만료:Gray)
  ├── 카테고리 태그 (4대 카테고리 컬러)
  ├── 구함 제목 (14px, 최대 2줄)
  ├── 작품명 · 캐릭터명 · 굿즈 타입
  ├── 등록일 · D-day (남은일수)
  ├── 매칭 N건 표시
  └── 끌어올리기 UP 뱃지

[구함 상세 바텀시트] (카드 탭 시)
  ├── 구함 제목 (18px)
  ├── 작품/캐릭터/굿즈 태그
  ├── 레퍼런스 이미지 · 상세 설명
  ├── 등록일 / 만료일 (D-day)
  ├── 교환 조건
  ├── 매칭된 교환글 섹션 (최대 5건) → EXCH02
  └── 액션: 교환 제안하기 / 채팅 문의 (본인 글: 수정/끌어올리기/삭제)

[구함글 등록 바텀시트] (FAB 탭 시)
  ├── 구함 제목 (필수)
```

## 6. 데이터 흐름 요약

```
[클라이언트]                     [서버]                      [DB/외부]
    │                              │                            │
    ├─ 상품 카드 렌더링 ────────→ GET /api/listings ────────→ PostgreSQL
    ├─ 교환글 등록 ─────────────→ POST /api/exchange ────────→ PostgreSQL
    │                              │                            │
    │                              ├─ 매칭 알고리즘 실행 ──────→ PostgreSQL
    │                              └─ 매칭 결과 ──────────────→ FCM (푸시)
    │                              │                            │
    ├─ 채팅 메시지 ─────────────→ Firebase Realtime DB ─────→ Firebase
    ├─ 이미지 업로드 ───────────→ POST /api/upload ──────────→ S3 + CloudFront
    ├─ 본인인증 ────────────────→ PortOne API ─────────────→ 통신사
    └─ 지도 ────────────────────→ 카카오맵 API ────────────→ 카카오
```

---

> **이 문서의 활용**: 사이트맵(§1)으로 전체 구조를 파악하고, 페이지별 IA(§2)로 각 화면의 구성 요소를 확인하며, 페이지 목록(§4)과 컴포넌트 목록(§5)을 기반으로 HTML 와이어프레임을 체계적으로 구축할 수 있습니다.
