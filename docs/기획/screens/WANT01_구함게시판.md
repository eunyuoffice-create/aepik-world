## 화면 개요
> 원하는 굿즈를 '구함' 형태로 게시하고, 관련 교환글을 매칭받는 게시판. 교환글 등록자에게 구함글이 노출되어 역제안을 유도하며, 매칭 알림으로 원하는 굿즈를 빠르게 찾을 수 있다.

## 진입 경로
- 홈 > 구함 바로가기
- 교환 목록 > 구함 탭 (또는 바로가기)
- 마이페이지 > 내 구함글
- 알림 > 구함 매칭 알림 탭
- 검색 결과 > 구함 탭

## 화면 구성 요소

### (1) Header
- 좌측: 페이지 제목 "구함"
- 우측: 검색 아이콘 (`Search`) — 탭 시 검색 페이지(SRCH01) 이동

### (2) 탭 바 (내 구함 / 전체)
- 2개 탭: **전체** / **내 구함**
- 전체: 모든 유저의 구함글 목록 (기본 활성)
- 내 구함: 본인 작성 구함글만 표시 (로그인 필요)
- 탭 하단 2px 인디케이터 (Blue #3B82F6)

### (3) 필터 바 (FilterChips)
- 가로 스크롤 칩 형태
- 필터 항목: 작품 ▾ / 캐릭터 ▾ / 굿즈 타입 ▾ / 상태 ▾
- 상태 옵션: 구함중 / 매칭됨 / 만료
- 칩 탭 시 바텀시트 열림 (COMM02 필터 바텀시트 패턴)
- 선택된 필터: 파란 배경(#3B82F6) + 흰 텍스트 + X 제거 버튼
- 필터 초기화: 우측 끝 "초기화" 텍스트 버튼

### (4) 정렬 + 결과 수
- 좌측: 전체 N개
- 우측: 정렬 드롭다운 — 최신순(기본) / 만료임박순

### (5) 구함글 목록
- 카드 리스트 (세로 나열)
- **구함 카드** 구성:
  - 좌측: 대표 이미지 또는 레퍼런스 이미지 (80x80px, 라운딩 8px) / 없으면 기본 아이콘
  - 우측 정보:
    - 상태 뱃지 (구함중: Green / 매칭됨: Blue / 만료: Gray)
    - 카테고리 태그 (카테고리 컬러)
    - 구함 제목 (14px, 600, 최대 2줄)
    - 작품명 · 캐릭터명 (12px, #6B7280)
    - 굿즈 타입 (10px, #9CA3AF)
    - 등록일 · 남은일수 (예: "D-15") (10px, #9CA3AF)
  - 우측 끝: 매칭 교환글 수 표시 (매칭 N건)
- **내 구함 탭**: 추가로 "끌어올리기" 버튼, 삭제/수정 메뉴
- 끌어올리기된 글: `Flame` 아이콘 + "UP" 뱃지
- 카드 탭 시: 구함 상세 바텀시트 열림
- 무한 스크롤 (Intersection Observer)
- 스켈레톤 UI: 로딩 시 카드 형태 회색 박스 펄스 애니메이션

### (6) 구함 상세 바텀시트
- 카드 탭 시 화면 하단에서 슬라이드 업
- 내용:
  - 구함 제목 (18px, 700)
  - 작품명 · 캐릭터명 · 굿즈 타입
  - 레퍼런스 이미지 (있는 경우)
  - 상세 설명 (원하는 버전, 상태 조건 등)
  - 등록일 / 만료일 (D-day 표시)
  - 교환 조건 (보유 굿즈 / 차액 가능 여부)
  - **매칭된 교환글 섹션**: 관련 교환글 카드 (최대 5건) + "더보기"
    - 매칭 기준: 작품+캐릭터 일치 (Phase 1)
    - 카드 탭 시 EXCH02 교환 상세 이동
  - **액션 버튼**: 
    - 타인 글: "교환 제안하기" (해당 교환글로 이동) / "채팅으로 문의" 
    - 본인 글: "수정" / "끌어올리기" / "삭제"

### (7) FAB (Floating Action Button)
- 우하단 고정 (bottom: 80px, right: 16px)
- 아이콘: `Edit` (+) — 구함글 등록
- 비로그인 시: 로그인 유도 팝업

### (8) 구함글 등록 바텀시트 (인라인)
- FAB 탭 시 바텀시트 슬라이드 업 (별도 페이지 이동 없음)
- 필드:
  - 구함 제목 (필수, placeholder: "찾고 있는 굿즈를 한 줄로 설명해주세요")
  - 작품 선택 (필수, 드롭다운/검색)
  - 캐릭터/멤버 선택 (필수, 드롭다운)
  - 굿즈 타입 (필수, 칩 선택: 포토카드/아크릴 스탠드/피규어/봉제인형/포스터/기타)
  - 상세 설명 (선택, textarea: 원하는 버전, 상태 조건 등)
  - 레퍼런스 이미지 (선택, 최대 3장)
  - 교환 조건 (선택: 보유 굿즈 목록 / 차액 가능 여부)
  - 만료일 설정 (기본 30일, 7/14/30/60일 선택)
- CTA: "구함글 등록" 버튼 (Primary)
- 등록 즉시 목록 반영 + 매칭 교환글 알림 발송

### (9) 하단 네비게이션 (BottomNav)
- 5개 탭: 홈(`Home`) | 교환(`Repeat`) | 이벤트(`Calendar`) | 채팅(`MessageCircle`) | 마이(`User`)
- 현재: 별도 활성 탭 없음 (구함은 하단 탭에 포함되지 않으므로)
- 또는 홈에서 진입 시 홈 활성, 마이페이지에서 진입 시 마이 활성 (진입 경로에 따라)

## 데이터 필드

### API: GET /api/wanted

**Request (Query Parameters):**

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| page | number | X | 페이지 번호 (기본 1) |
| limit | number | X | 페이지당 개수 (기본 20) |
| work_id | string | X | 작품 필터 |
| character_id | string | X | 캐릭터 필터 |
| goods_type | string | X | 굿즈 타입 필터 |
| status | string | X | 상태 (seeking/matched/expired) |
| sort | string | X | 정렬 (latest/expiring_soon) |
| mine | boolean | X | 내 구함글만 (true/false) |

**Response:**

| 필드 | 타입 | 설명 |
|------|------|------|
| items | array | 구함글 목록 |
| items[].id | string | 구함글 ID |
| items[].title | string | 구함 제목 |
| items[].work | object | 작품 정보 {id, name} |
| items[].character | object | 캐릭터 정보 {id, name} |
| items[].goods_type | string | 굿즈 타입 |
| items[].category | string | 카테고리 (anime_manga_novel/idol_artist/actor_stage/game) |
| items[].description | string | 상세 설명 (nullable) |
| items[].reference_images | string[] | 레퍼런스 이미지 URL (nullable) |
| items[].trade_condition | object | 교환 조건 {has_offer, cash_possible} |
| items[].status | string | 상태 (seeking/matched/expired) |
| items[].expires_at | string | 만료일 |
| items[].matched_count | number | 매칭된 교환글 수 |
| items[].user | object | 등록자 {id, nickname, trust_level} |
| items[].created_at | string | 등록일 |
| items[].is_bumped | boolean | 끌어올리기 여부 |
| total | number | 전체 개수 |
| has_next | boolean | 다음 페이지 여부 |

### API: POST /api/wanted (구함글 등록)

**Request Body:**

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| title | string | O | 구함 제목 (max 50자) |
| work_id | string | O | 작품 ID |
| character_id | string | O | 캐릭터 ID |
| goods_type | string | O | 굿즈 타입 |
| description | string | X | 상세 설명 (max 500자) |
| reference_images | file[] | X | 레퍼런스 이미지 (최대 3장) |
| has_offer | boolean | X | 보유 굿즈 교환 의향 |
| cash_possible | boolean | X | 차액 가능 여부 |
| expire_days | number | X | 만료일 (7/14/30/60, 기본 30) |

## 화면 상태

| 상태 | 조건 | 표시 |
|------|------|------|
| 정상 | 구함글 1건 이상 | 구함글 목록 |
| 빈 상태 (전체) | 필터 결과 없음 or 구함글 없음 | "조건에 맞는 구함글이 없어요" + CTA "구함글 등록하기" |

| 로딩 | 데이터 페칭 중 | 스켈레톤 UI |
| 에러 | API 실패 | "잠시 후 다시 시도해주세요" + 재시도 버튼 |
| 비로그인 (내 구함) | 미로그인 상태에서 "내 구함" 탭 | "로그인하고 구함글을 관리하세요" + 로그인 버튼 |
