## 화면 개요

> 진행 중인 교환/공구 채팅방을 시간순으로 확인하고 빠르게 대화에 진입하는 목록 화면. 최신 메시지 미리보기, 안 읽은 메시지 수 뱃지, 거래 상태 태그 등을 통해 한눈에 대화 현황을 파악할 수 있다.

---

## 진입 경로

- 하단 네비게이션 > 채팅 탭
- 교환 상세 > "채팅하기" CTA
- 교환 제안 수락 시 자동 채팅방 생성 > 알림 탭 > 채팅 목록
- 공구 참여 완료 시 채팅방 입장
- 푸시 알림 > 채팅 메시지 알림 탭

---

## 화면 구성 요소

### 1 Header

- 좌측: 페이지 제목 "채팅"
- 우측: 검색 아이콘(`Search`) -- 채팅방 내 검색 (닉네임/상품명)
- 스타일: 높이 48px, 하단 보더 1px `#F3F4F6`

### 2 필터 탭 (FilterTabs)

- 가로 탭 형태: **전체** | **교환** | **공구**
- 활성 탭: Blue 텍스트 + 하단 Blue 보더 2px
- 비활성 탭: Gray 400 텍스트
- 각 탭 우측에 안 읽은 채팅방 수 표시 (예: "교환 3")
- 전체 탭에는 총 안 읽은 수 합산 표시

### 3 채팅방 리스트

각 채팅방 아이템의 구성:

| 요소 | 스펙 |
|------|------|
| **아바타** | 좌측 48px 원형, 상대방 프로필 이미지 (기본 아바타 fallback) |
| **온라인 닷** | 아바타 우하단 10px 원형 `#10B981`, 상대방 접속 중일 때만 표시 |
| **거래 유형 태그** | 닉네임 우측 pill 뱃지 — 교환(`#FFF7ED`/`#EA580C`) 또는 공구(`#EFF6FF`/`#2563EB`) |
| **닉네임** | 14px SemiBold `#111827` |
| **상품명** | 닉네임 하단, 12px `#6B7280`, 최대 1줄 ellipsis |
| **최신 메시지** | 13px Regular `#6B7280`, 최대 1줄 ellipsis |
| **이미지 메시지** | 텍스트 대신 `Image` 아이콘 + "사진" 표시 |
| **시스템 메시지** | 텍스트 대신 "교환 확정이 요청되었습니다" 등 상태 안내 |
| **타임스탬프** | 우상단, 11px `#9CA3AF` — 오늘이면 "오후 3:42", 올해면 "3/6", 이전 연도면 "2025.12.31" |
| **안 읽은 수 뱃지** | 우측 하단, 숫자 뱃지 (COMM02 2.18 카운트 뱃지 스펙) — 배경 `#EF4444`, 텍스트 white 10px Bold, 100+ = "99+" |
| **거래 상태 인디케이터** | 상품명 좌측에 컬러 도트 — 진행중(`#10B981` Green), 확정대기(`#F59E0B` Amber), 완료(`#9CA3AF` Gray) |

#### 채팅방 아이템 레이아웃

- 높이: auto (최소 72px)
- 패딩: 16px
- 하단 보더: 1px `#F3F4F6`
- 안 읽은 채팅방: 배경 `#F8FAFF` (연한 블루)
- 읽은 채팅방: 배경 `#FFFFFF`
- 탭 시: CHAT02 채팅 대화방으로 이동

#### 정렬

- 최신 메시지 시간순 (최신이 위)
- 고정된 채팅방이 최상단 (Phase 2)

### 4 빈 상태 (Empty State)

- 조건: 채팅방이 하나도 없는 경우
- 아이콘: `MessageCircle` 라인 아이콘 48px, opacity 0.4
- 메시지: "아직 채팅이 없어요"
- 부가 텍스트: "교환 제안이 수락되거나 공구에 참여하면\n여기에서 대화를 시작할 수 있어요"
- CTA: "교환 둘러보기" 버튼 (Secondary) > EXCH01 이동

### 5 하단 네비게이션 (BottomNav)

- 5개 탭: 홈(`Home`) | 교환(`Repeat`) | 이벤트(`Calendar`) | **채팅(`MessageCircle`)** | 마이(`User`)
- 채팅 탭 활성 상태 (Blue)
- 채팅 탭에 안 읽은 메시지 총 수 뱃지 (빨간 숫자 뱃지)

## 인터랙션

| 액션 | 동작 |
|------|------|
| 채팅방 탭 | CHAT02 채팅 대화방으로 push 전환 |
| 채팅방 좌 스와이프 | 액션 메뉴 노출: 알림 끄기 / 나가기 |
| "나가기" 탭 | 확인 모달: "채팅방을 나가시겠어요? 대화 내역이 삭제됩니다" > 확인/취소 |
| 검색 아이콘 탭 | 상단 검색 바 확장 > 닉네임/상품명으로 채팅방 필터링 |
| 필터 탭 전환 | 해당 거래 유형 채팅방만 표시 |
| 당겨서 새로고침 | 채팅 목록 갱신 |

## 비즈니스 로직

| 규칙 | 설명 |
|------|------|
| **채팅방 생성 조건** | 교환 제안 수락 시 자동 생성 / 공구 참여 시 자동 생성 |
| **채팅방 타입** | exchange (1:1 교환) / group_buy (공구 단체) |
| **교환 채팅방** | 1:1 대화, 교환 확정/운송장/완료 기능 포함 |
| **공구 채팅방** | 주최자+참여자 단체 대화, 공구 진행 상태 공유 |
| **메시지 미리보기** | 최신 메시지 1줄, 이미지는 "사진", 시스템 메시지는 원문 표시 |
| **안 읽은 수** | Firebase Realtime DB 기준, 사용자별 lastRead 타임스탬프와 비교 |
| **채팅방 정렬** | 최신 메시지 타임스탬프 내림차순 |
| **채팅방 나가기** | 교환 완료 후에만 나가기 가능, 진행 중인 거래는 나가기 차단 |
| **비로그인** | 채팅 목록 접근 시 AUTH01-D 로그인 유도 바텀시트 표시 |
| **실시간 업데이트** | Firebase onValue 리스너로 새 메시지 시 목록 자동 갱신 |
| **알림 연동** | FCM 푸시 알림, 채팅 탭 바텀 네비에 안 읽은 수 뱃지 반영 |

## 접근성

- 채팅방 리스트: `role="list"`, 각 아이템 `role="listitem"`
- 안 읽은 뱃지: `aria-label="안 읽은 메시지 N건"`
- 스와이프 액션: 키보드로도 접근 가능 (포커스 후 Delete 키)
- 빈 상태 CTA: Tab 키로 포커스 가능
