## 화면 개요

> 사용자가 등록한 교환글/공구/동행글, 완료된 거래 내역, 받은 후기, 찜 목록을 관리하는 활동 허브 화면. 각 탭으로 전환하며 자신의 활동 현황을 확인하고 관리할 수 있다.

---

## 진입 경로

- USER01 마이페이지 > 퀵 카운트 카드 탭
- USER01 마이페이지 > 메뉴 항목 (교환 내역, 받은 후기, 찜 목록) 탭
- 푸시 알림 > 내 활동 관련 알림 탭

---

## 화면 구성 (6개 탭)

| 탭 | 라우트 | 설명 |
|----|--------|------|
| **A. 내 교환글** | /my/exchanges | 등록한 교환글 목록 + 상태별 필터 |
| **B. 내 공구** | /my/group-buys | 참여/진행 공구 목록 + 상태별 필터 |
| **C. 내 동행글** | /my/companion | 등록한 동행글 목록 |
| **D. 교환 내역** | /my/history | 완료된 교환/공구 거래 기록 |
| **E. 받은 후기** | /my/reviews | 거래 상대가 남긴 후기 |
| **F. 찜 목록** | /my/wishlist | 찜한 교환글/공구/이벤트 |

## 공통 레이아웃

### 헤더

- 좌측: 뒤로가기 화살표 (마이페이지로 복귀)
- 중앙: "내 활동"

### 탭 바

- 가로 스크롤 탭: 교환글 / 공구 / 동행 / 내역 / 후기 / 찜
- 활성 탭: 파란색 하단 인디케이터 + Bold
- 각 탭 우측에 카운트 뱃지 (예: "교환글 3")

---

## A. 내 교환글 (/my/exchanges)

### A-1. 상태 필터 칩

- 가로 칩: 전체 / 교환중 / 제안 대기 / 교환 완료
- 선택된 칩: Primary 배경, 흰 텍스트

### A-2. 교환글 카드 리스트

| 요소 | 스펙 |
|------|------|
| 이미지 | 80x80px, border-radius 8px |
| 상품명 | 14px Bold, 최대 2줄 |
| 작품명 · 캐릭터명 | 12px Gray 500 |
| 상태 뱃지 | 교환중(Blue) / 제안 N건(Amber) / 교환완료(Green) / 삭제됨(Gray) |
| 등록일 | 12px Gray 400 |
| 조회수 · 찜 수 | 12px Gray 400 |

### A-3. 카드 액션

- 카드 탭: 해당 교환 상세(EXCH02) 이동
- 카드 롱프레스 또는 더보기: 끌어올리기 / 수정 / 삭제 액션시트
- 삭제 시 확인 팝업: "교환글을 삭제하시겠습니까?" (진행 중 교환 제안이 있으면 삭제 불가)

### A-4. 빈 상태

- 일러스트 + "등록한 교환글이 없어요"
- CTA: "교환글 등록하기" -- EXCH03 이동

---

## B. 내 공구 (/my/group-buys)

### B-1. 서브 탭

- 진행 중인 공구 / 참여한 공구 (역할 기준)
- 진행 중: 내가 진행자인 공구
- 참여한: 내가 참여자인 공구

### B-2. 상태 필터 칩

- 전체 / 모집중 / 진행 결정 대기 / 진행 확정 / 배송중 / 완료 / 무산

### B-3. 공구 카드 리스트

| 요소 | 스펙 |
|------|------|
| 이미지 | 80x80px |
| 상품명 | 14px Bold |
| 진행자 | 12px Gray 500 (참여한 공구에서만 표시) |
| 참여 현황 | 프로그레스 바 + "N/M명" |
| 상태 뱃지 | 모집중(Blue) / 진행확정(Green) / 무산(Red) / 배송중(Amber) / 완료(Gray) |
| 마감일 | D-N 또는 날짜 |

### B-4. 카드 액션

- 카드 탭: 공구 상세(GRPB02) 이동
- 진행자 역할일 때 더보기: 공구 관리(GRPB04) 이동

### B-5. 빈 상태

- "참여한 공구가 없어요"
- CTA: "공구 둘러보기" -- GRPB01 이동

---

## C. 내 동행글 (/my/companion)

### C-1. 동행글 카드 리스트

| 요소 | 스펙 |
|------|------|
| 동행 제목 | 14px Bold |
| 이벤트 정보 | 12px Gray 500 |
| 등록일 · 종료일 | 12px Gray 400, D-N 표시 |
| 매칭 알림 수 | "매칭 N건" 뱃지 (수신된 매칭이 있을 때) |
| 상태 | 모집중(Green) / 마감(Gray) / 취소(Red) |

### C-2. 카드 액션

- 카드 탭: 동행글 상세(WANT02) 이동
- 더보기: 수정 / 삭제 / 기간 연장 (이벤트 전까지 가능)

### C-3. 빈 상태

- "등록한 동행글이 없어요"
- CTA: "동행글 등록하기" -- WANT01 이동

---

## D. 교환 내역 (/my/history)

### D-1. 필터

- 기간 필터: 최근 1개월 / 3개월 / 6개월 / 1년 / 전체
- 유형 필터: 전체 / 교환 / 공구

### D-2. 거래 내역 카드

| 요소 | 스펙 |
|------|------|
| 상대방 프로필 | 36px 원형 + 닉네임 |
| 거래 유형 | 교환(Blue) / 공구(Purple) 뱃지 |
| 상품명 | 14px Bold |
| 완료일 | 12px Gray 400 |
| 후기 작성 여부 | "후기 작성 완료" 또는 "후기 작성하기" 링크 |

### D-3. 후기 미작성 시

- "후기 작성하기" 링크 탭 시 후기 작성 바텀시트
- 평가: 긍정 / 부정 선택 (필수)
- 텍스트 후기: 선택, 최대 200자
- 작성 기한: 완료 후 7일 이내, 기한 경과 시 작성 불가 (회색 처리)

### D-4. 빈 상태

- "아직 완료된 거래가 없어요"
- CTA: "교환 매물 둘러보기" -- EXCH01 이동

---

## E. 받은 후기 (/my/reviews)

### E-1. 후기 요약

- 상단 카드: 전체 후기 N건 / 긍정 N건 (N%) / 부정 N건 (N%)
- 긍정 비율 원형 게이지 (파란색)

### E-2. 후기 리스트

| 요소 | 스펙 |
|------|------|
| 평가 아이콘 | 긍정: 파란 엄지 위 / 부정: 빨간 엄지 아래 |
| 작성자 | 닉네임 (탭 시 타인 프로필 이동) |
| 텍스트 후기 | 14px, 최대 3줄 (더보기) |
| 작성일 | 12px Gray 400 |
| 관련 거래 | "교환: [상품명]" 회색 링크 |

### E-3. 이의 제기

- 부정 후기 카드에 "이의 제기" 텍스트 링크 (72시간 이내만 활성)
- 탭 시 바텀시트: 이의 사유 텍스트 입력 + 증거 첨부
- 접수 후 운영팀 72시간 내 검토

### E-4. 빈 상태

- "아직 받은 후기가 없어요"

---

## F. 찜 목록 (/my/wishlist)

### F-1. 카테고리 탭

- 가로 탭: 전체 / 교환글 / 공구 / 이벤트

### F-2. 찜 카드 리스트

- 교환글/공구: ProductCard와 동일 포맷
- 이벤트: 이벤트 카드 (이미지 + 제목 + 날짜 + 장소)
- 각 카드 우측 상단: 찜 해제 하트 아이콘 (채워진 빨간 하트)

### F-3. 찜 해제

- 하트 아이콘 탭 시 즉시 해제 (Optimistic UI)
- 하단 토스트: "찜이 해제되었습니다" + "되돌리기" 링크 (5초)

### F-4. 빈 상태

- "찜한 항목이 없어요"
- CTA: "매물 둘러보기" -- HOME01 이동
