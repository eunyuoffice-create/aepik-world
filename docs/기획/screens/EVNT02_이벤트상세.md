## 화면 개요

> 개별 팬 이벤트의 상세 정보(장소, 일시, 포스터, 지도 등)를 확인하고, 찜/공유/신고 등의 액션을 수행하는 화면. 관련 교환글이나 공구를 연결하여 이벤트 방문 시 굿즈 교환/공구 참여를 유도한다.

## 화면 구성 요소

### (1) Header
- 좌측: 뒤로가기(`ChevronLeft`) -- EVNT01 또는 이전 화면
- 중앙: 이벤트명 (스크롤 시 표시, 초기 투명)
- 우측: 공유(`Share2`) / 더보기(`MoreVertical`)
  - 더보기 메뉴: 신고하기 / 수정하기(본인 글) / 삭제하기(본인 글)

### (2) 대표 이미지 영역
- 풀 너비 이미지 (높이 200px, aspect-ratio 유지)
- 이미지 없는 경우: 카테고리 컬러 그라데이션 배경 + 이벤트 유형 아이콘
- 이미지 여러 장: 좌우 스와이프 캐러셀 + 페이지 인디케이터 도트

### (3) 이벤트 기본 정보
- **이벤트 유형 태그**: 카테고리 컬러 배경 (예: 생일카페, 지하철 광고 등)
- **이벤트명**: 20px Bold
- **대상**: 작품명 > 캐릭터명 (탭 시 해당 작품/캐릭터 검색)
- **카테고리 뱃지**: Purple/Pink/Amber/Cyan 중 해당 카테고리 색상
- **등록자**: 아바타(24px) + 닉네임 + 등록일
- **찜 수**: `Heart` 아이콘 + 찜 카운트

### (4) 일정 정보
- `Calendar` 아이콘 + 시작일 ~ 종료일 (D-day 표시)
  - 진행 중: 초록 "진행 중" 뱃지
  - 예정: 파란 "D-N" 뱃지
  - 종료: 회색 "종료" 뱃지
- `Clock` 아이콘 + 운영 시간 (없으면 비표시)
- [캘린더 추가] 버튼 -- 기기 캘린더에 일정 추가

### (5) 장소 정보
- `MapPin` 아이콘 + 주소 텍스트
- [주소 복사] 아이콘 버튼
- 카카오맵 미니맵 (높이 160px, 라운딩 12px)
  - 이벤트 위치 핀 표시
  - 탭 시 카카오맵 앱 또는 웹 열기 (길찾기)

### (6) 상세 설명
- 등록자가 작성한 자유 텍스트
- 접기/펼치기: 3줄 초과 시 "더보기" 링크
- 외부 링크: `ExternalLink` 아이콘 + URL (탭 시 인앱 브라우저 또는 외부 브라우저)

### (7) 관련 교환/공구 섹션
- "이 이벤트 관련 교환/공구" 헤더
- 같은 작품+캐릭터의 교환글/공구 최대 4개 표시 (가로 스크롤 카드)
- [더보기] -- EXCH01/GRPB01에 해당 작품+캐릭터 필터 적용 이동
- 관련 없으면 비표시

### (8) 액션 바 (하단 고정)
- 좌측: 찜 버튼 (`Heart` 빈/채움 + "찜" 텍스트)
- 우측: [공유하기] 버튼 (카카오톡, 트위터, 링크 복사)
- 본인 글인 경우: [수정] / [삭제] 버튼으로 교체
