## 화면 개요

> 알림 설정, 관심 카테고리 변경, 키워드 알림 관리, 계정 관리(비밀번호 변경, 소셜 연동, 탈퇴) 등 사용자 환경 설정을 관리하는 화면. 관심 카테고리 변경값은 홈 피드 개인화에도 즉시 반영된다.

## A. 설정 메인 (/my/settings)

### A-1. 헤더

- 좌측: 뒤로가기 화살표 (마이페이지로 복귀)
- 중앙: "설정"

### A-2. 알림 설정 섹션

| 항목 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| 푸시 알림 | 토글 스위치 | ON | 전체 알림 ON/OFF |
| 교환 매칭 알림 | 토글 스위치 | ON | 매칭 알림 수신 |
| 키워드 알림 | 토글 스위치 | ON | 키워드 매칭 알림 수신 |
| 공구 알림 | 토글 스위치 | ON | 공구 상태 변경 알림 |
| 채팅 알림 | 토글 스위치 | ON | 채팅 메시지 알림 |
| 이벤트/마케팅 알림 | 토글 스위치 | OFF | 이벤트/프로모션 |
| 방해금지 시간 | 시간 선택 | 없음 | 시작~종료 시간 설정 |

- 푸시 알림 OFF 시 하위 항목 비활성 (회색 처리)
- 방해금지: 시작/종료 시간 피커 (예: 23:00 ~ 08:00)

### A-3. 관심 카테고리

- 현재 선택된 카테고리 태그 나열 (Purple/Pink/Amber/Cyan)
- "변경" 텍스트 버튼 -- 탭 시 카테고리 선택 바텀시트 (AUTH02 Step 4 패턴)
- 선택 없음 시: "관심 카테고리를 설정하면 홈 추천과 맞춤 알림을 받을 수 있어요" 안내

### A-4. 계정 관리 섹션

리스트 형태:

| 메뉴 | 동작 | 비고 |
|------|------|------|
| 비밀번호 변경 | 바텀시트: 현재 비밀번호 + 새 비밀번호 + 확인 | 카카오 연동 전용 계정은 숨김 |
| 연결된 소셜 계정 | 카카오 연동 상태 표시 | 연동 해제/추가 (Phase 2) |
| 키워드 알림 설정 | 모드 B 이동 | 등록 키워드 N개 표시 |
| 로그아웃 | 확인 팝업 | USER01 로그아웃과 동일 |
| 회원 탈퇴 | 모드 C 탈퇴 프로세스 | Danger 텍스트 |

### A-5. 정보 섹션

| 메뉴 | 이동 대상 |
|------|-----------|
| 이용약관 | STAT01 약관 |
| 개인정보처리방침 | STAT01 정책 |
| 오픈소스 라이선스 | 라이선스 목록 |
| 앱 버전 | 현재 버전 표시 (1.0.0) |

## B. 키워드 알림 설정 (/my/keywords)

### B-1. 헤더

- 좌측: 뒤로가기
- 중앙: "키워드 알림"
- 우측: 없음

### B-2. 안내 문구

- "등록한 키워드와 매칭되는 교환글/공구가 올라오면 알림을 보내드려요"
- 12px Gray 500, 상단 패딩 16px

### B-3. 키워드 추가

| 요소 | 스펙 |
|------|------|
| 인풋 필드 | placeholder: "키워드 입력 (예: 하이큐 히나타)", 44px |
| 추가 버튼 | Primary Small, 인풋 우측 |
| 제한 | 최대 10개(무료 기준), 키워드당 2~30자 |

### B-4. 등록된 키워드 리스트

- 태그 칩 형태: 키워드 텍스트 + X 삭제 버튼
- 최신 등록순 정렬
- 각 키워드 탭 시 매칭된 매물 검색 결과로 이동 (SRCH01)

### B-5. 키워드 매칭 로직 안내

- 하단 안내 카드 (Gray 50 배경):
  - "Phase 1: 작품+캐릭터 수준 매칭"
  - "Phase 2: 버전 수준 정밀 매칭 (예정)"

### B-6. 빈 상태

- "등록된 키워드가 없어요"
- "키워드를 등록하면 새 매물 알림을 받을 수 있어요"

## C. 계정 관리 (탈퇴 프로세스)

### C-1. 비밀번호 변경 바텀시트

| 필드 | 스펙 |
|------|------|
| 현재 비밀번호 | 인풋 + 눈 아이콘 토글 |
| 새 비밀번호 | 인풋 + 강도 바 (AUTH02와 동일) |
| 새 비밀번호 확인 | 인풋 + 일치/불일치 표시 |
| 변경 버튼 | Primary Large 풀폭 |

- 현재 비밀번호 불일치 시: "현재 비밀번호가 올바르지 않습니다" 에러
- 성공 시: 토스트 "비밀번호가 변경되었습니다" + 바텀시트 닫힘

### C-2. 탈퇴 프로세스

**Step 1: 탈퇴 안내**

- "정말 탈퇴하시겠어요?" 제목
- 탈퇴 시 안내 사항:
  - 진행 중인 교환/공구가 완료되어야 탈퇴 가능합니다
  - 작성한 후기는 삭제되지 않습니다 (익명 처리)
  - 개인정보는 관련 법령에 따라 일정 기간 보관 후 파기됩니다
  - 탈퇴 후 동일 이메일로 재가입 시 30일 제한이 있습니다
- [다음] 버튼

**Step 2: 탈퇴 사유**

- 사유 선택 (라디오):
  - 더 이상 사용하지 않아요
  - 원하는 매물을 찾기 어려워요
  - 서비스 이용이 불편해요
  - 개인정보가 걱정돼요
  - 기타 (텍스트 입력)
- [탈퇴하기] Danger 버튼

**Step 3: 본인 확인**

- 비밀번호 입력 (이메일 가입) 또는 카카오 재인증 (소셜 가입)
- [최종 탈퇴] Danger 버튼

**Step 4: 탈퇴 완료**

- "탈퇴가 완료되었습니다"
- "그동안 이용해주셔서 감사합니다"
- 3초 후 홈 화면 이동

| 11 | 카테고리 변경 반영 | 관심 카테고리 저장 완료 시 HOME01 개인화 랭킹 캐시 무효화 후 즉시 재계산 |
