## 화면 개요

> 공동구매를 등록하는 폼 화면. 일반 공구와 세트 공구(캐릭터 선점) 두 유형을 선택하여 등록 가능한 페이지로, 진행자가 상품 정보, 가격, 모집 기간, 캐릭터 슬롯 설정 등을 입력하여 공구를 시작할 수 있다.

---

## 진입 경로

- 공구 목록(GRPB01) > FAB 탭
- 홈(HOME01) > FAB > 공구 등록 선택
- 공구 상세(GRPB02) > [또 다른 공구 시작하기] 버튼

---

## 진입 조건

- 로그인 필수
- 본인인증 필수
- 진행자 자격: 교환 거래 5회 이상 + 긍정 평가 90% 이상

---

## 화면 구성 요소

### ① Header
- 좌측: 뒤로가기 아이콘 (<)
- 중앙: 페이지 제목 "공구 등록"
- 우측: 닫기 아이콘 (X)

### ② 공구 유형 선택
- 형태: 큰 카드 2개 (2열 또는 모바일 시 수직 스택)
- 선택: 라디오 버튼 (하나만 선택 가능)
- 옵션:
  - **일반 공구**: "단일 상품 공동 구매" (기본 선택)
  - **세트 공구 (캐릭터 선점)**: "세트 구매 후 캐릭터 분배"
- 선택된 카드: Blue 테두리 + 배경 강조

### ③ 카테고리 선택
- 형태: 4개 칩 (라디오 그룹)
- 옵션: 애니/만화/소설 | 아이돌/아티스트 | 배우/무대 | 게임
- 기본 선택: 애니/만화/소설
- 선택된 칩: Blue 배경 + 흰색 텍스트
- 비선택 칩: Gray 배경 + Dark 텍스트

### ④ 기본 정보 입력
- **상품명** (필수)
  - Text input, 최대 100자
  - Placeholder: "상품명을 입력하세요"
  - 예시: "원신 카즈하 아크릴 스탠드 공구"
- **상품 이미지** (선택)
  - 최대 10장 업로드 (S3 + Sharp 리사이즈)
  - 포토 업로드 UI: 회색 점선 박스 + "+ 사진 추가" 텍스트
  - 업로드 후 썸네일 그리드 표시
  - 드래그로 순서 변경 가능
  - 각 이미지: 삭제 버튼(X) 오버레이
- **작품 검색/선택** (선택)
  - Text input with search icon
  - Autocomplete 드롭다운 (작품 DB 검색)
  - 세트 공구 선택 시 필수
- **굿즈 타입 선택** (선택)
  - Dropdown: 포토카드/아크릴/캔뱃지/피규어/인형/엽서/의류/기타

### ⑤ 가격 정보
- **일반 공구 (type: regular)**:
  - "1인당 가격": Number input with ₩ 표시
  - "배송비 포함": Toggle switch (기본: OFF)
  - 토글 ON 시 배송비는 이미 포함된 가격
- **세트 공구 (type: set)**:
  - "캐릭터당 가격": Number input with ₩ 표시
  - "배송비 포함": Toggle switch (기본: OFF)

### ⑥ 모집 정보
- **목표 인원** (필수)
  - Number input
  - 최소 2명 이상
- **최소 인원** (선택)
  - Number input
  - 목표 인원 이하
- **마감일** (필수)
  - Date picker
  - 최소: 현재 날짜 + 1일 이후
  - 예시: 2026-03-13
- **배송 예정일** (필수)
  - Date picker
  - 마감일 이후 설정 필수
  - 예시: 2026-04-01

### ⑦ 세트 공구 전용 - 캐릭터 슬롯 설정
*(type = "set" 선택 시에만 표시)*

- **캐릭터 목록** (자동 로드)
  - 작품 선택 시 해당 작품의 캐릭터 목록 자동 로드
  - 각 행: [체크박스(포함 여부)] + [캐릭터명] + [선점 가능 수량 input] + [선점된 수량] + [삭제 버튼]
  - 선택된 캐릭터만 공구에 포함
- **캐릭터 직접 추가**
  - 버튼: "[+] 캐릭터 직접 추가"
  - 클릭 시 입력폼: 캐릭터명 input + 가능 수량 input
- **진행자 본인 선점 설정**
  - 각 캐릭터 우측: "[선점]" 버튼 또는 체크박스
  - 선점한 캐릭터는 진행자가 미리 확보할 수량 설정
  - 진행자가 선택한 캐릭터에는 "(본인 선점)" 배지 표시
  - 세트 공구 최소 2개 캐릭터 필수

### ⑧ 상세 설명
- Textarea
- 마크다운 지원 (preview 버튼 옵션)
- Placeholder: "공구 상세 내용을 입력하세요. (배송 방법, 픽링 일정, 추가 금액 등)"
- 최대 3000자

### ⑨ 하단 CTA
- 버튼 2개 (좌우 배치):
  - **[미리보기]**: 등록 전 공구 정보 미리 확인 (outline 스타일)
  - **[등록하기]**: 공구 등록 제출 (Blue 배경, 흰색 텍스트)
- 등록 시 확인 모달: "공구를 등록하시겠습니까? [취소] [등록하기]"
- 진행자 자격 미달 시 안내 메시지 표시: "진행자 자격(교환 5회 이상, 긍정 90% 이상)이 없습니다."
- 등록 중: 버튼 비활성화 + 로딩 스피너

## 데이터 필드

### API: POST /api/groupbuy

**Request Body:**

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| type | string | O | "regular" 또는 "set" |
| category | string | O | 카테고리 (anime_manga/idol_artist/actor_stage/game) |
| title | string | O | 상품명 |
| images | string[] | X | 이미지 URL 배열 (S3) |
| work_id | string | X | 작품 ID (세트 공구 필수) |
| goods_type | string | X | 굿즈 타입 |
| price_per_person | number | O | 1인당 가격(일반) 또는 캐릭터당 가격(세트) |
| delivery_included | boolean | O | 배송비 포함 여부 |
| target_count | number | O | 목표 인원 |
| min_count | number | X | 최소 인원 |
| deadline | string | O | 마감일 (ISO 8601) |
| delivery_date | string | O | 배송 예정일 (ISO 8601) |
| description | string | X | 상세 설명 (마크다운) |
| character_slots | array | X | 캐릭터 슬롯 (세트 공구 필수) |
| character_slots[].character_id | string | O | 캐릭터 ID |
| character_slots[].quantity | number | O | 선점 가능 수량 |
| character_slots[].reserved_by_organizer | number | O | 진행자 선점 수량 |

**Response:**

| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 공구 ID |
| status | string | 상태 (active/pending/completed) |
| created_at | string | 생성 시간 (ISO 8601) |
| url | string | 공구 상세 페이지 URL (/groupbuy/[id]) |

## 인터랙션

| # | 트리거 | 동작 | 비고 |
|---|--------|------|------|
| 1 | 공구 유형 선택 | 라디오 토글 → 폼 필드 갱신 | 세트 공구 선택 시 캐릭터 슬롯 섹션 표시 |
| 2 | 카테고리 선택 | 칩 탭 → 해당 카테고리 선택 | 단일 선택 |
| 3 | 작품 검색 | Input 입력 → autocomplete 드롭다운 | 작품 DB API 호출 |
| 4 | 이미지 업로드 | "+ 사진 추가" 탭 → 파일 선택 | 다중 파일 지원 |
| 5 | 마감일 선택 | Date picker 탭 → 날짜 선택 | 현재일 + 1일 이후만 활성화 |
| 6 | 세트 공구 캐릭터 추가 | "[+] 캐릭터 직접 추가" → 팝업/모달 | 작품에 없는 캐릭터 수동 입력 |
| 7 | 진행자 선점 설정 | 캐릭터 우측 "[선점]" 버튼 → 수량 입력 모달 | 세트 공구 전용 |
| 8 | 미리보기 | "[미리보기]" 탭 → 공구 상세 프리뷰 | /groupbuy/[temp_id]/preview (임시) |
| 9 | 등록하기 | "[등록하기]" 탭 → 확인 모달 → API 호출 | 성공 시 GRPB02로 이동 |
| 10 | 뒤로가기 | 헤더 "<" 탭 또는 브라우저 뒤로 | 작성 중인 정보 유실 경고 모달 |
