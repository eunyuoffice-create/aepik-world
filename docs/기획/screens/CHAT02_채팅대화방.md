# CHAT02 채팅 대화방

## 메타 정보

| 항목 | 내용 |
|------|------|
| **화면코드** | CHAT02 |
| **화면명** | 채팅 대화방 |
| **라우트** | /chat/[roomId] |
| **이용자** | PC / Mobile |
| **버전** | 0.2 |
| **작성일** | 2026-03-06 |
| **상태** | 초안 |

---

## 화면 개요

> 교환/공구 상대와 실시간으로 대화하며, 거래 조건을 협의하고 교환 확정/운송장 입력/완료 등 거래 프로세스를 진행하는 1:1(또는 공구 단체) 채팅 화면. Firebase Realtime DB 기반 실시간 메시지 송수신.

---

## 진입 경로

- CHAT01 채팅 목록 > 채팅방 탭
- 교환 상세(EXCH02) > "채팅하기" 버튼
- 교환 제안 수락 시 자동 진입
- 공구 참여 완료 시 자동 진입
- 푸시 알림 탭 (채팅 메시지 알림)

| 상태 | 배경 | 아이콘 | 텍스트 |

| **교환 협의 중** | `#EFF6FF` | `MessageCircle` Blue | "교환 조건을 협의 중이에요" |
| **교환 확정 요청** | `#FEF3C7` | `AlertCircle` Amber | "교환 확정을 기다리고 있어요" (한쪽만 확정한 경우) |
| **교환 확정 완료** | `#D1FAE5` | `Check` Green | "교환이 확정되었어요! 상품을 발송해주세요" |
| **발송 완료 (내가)** | `#DBEAFE` | `Truck` Blue | "상대방의 발송을 기다리고 있어요" |

- 높이: auto (최소 44px), 패딩 12px 16px
- 탭 가능: 상세 상태 바텀시트 열기 (거래 타임라인 표시)

#### 보낸 메시지 (우측)

#### 받은 메시지 (좌측)

| 항목 | 스펙 |
|------|------|
| **배경** | `#F3F4F6` |
| **텍스트** | `#111827`, 14px |

| **타임스탬프** | 우측 하단, 11px, `#9CA3AF` |

#### 시스템 메시지

- 중앙 정렬, 배경 `#F3F4F6`, 라운딩 999px, 패딩 6px 16px
- 폰트 12px, `#6B7280`
- 용도: 채팅방 생성 안내, 거래 상태 변경, 교환 확정 알림, 직교환 시간 만료 안내

#### 직교환 시간 만료 시스템 메시지

- 트리거: 오늘 직교환(`instant_meetup`) 시간대가 종료된 시점에 자동 발송
- 스타일: 시스템 메시지 기본 + Amber 강조 (`Clock` 아이콘 `#F59E0B` + 텍스트)
- 메시지: "[이벤트명] 직교환 시간이 만료되었습니다. 일반 직거래 제안으로 전환되었어요."
- 하위 안내: "오프라인 만남 일정을 다시 조율하려면 채팅으로 협의해주세요."
- 배경: `#FEF3C7` (Amber 100), 라운딩 12px, 패딩 12px 16px
- 아이콘: `Clock` 16px Amber 좌측 배치

#### 이미지 메시지

- 썸네일: 최대 200px 너비, 라운딩 12px
- 탭 시: 전체화면 이미지 뷰어 (핀치 줌, 스와이프 닫기)
- S3 업로드: 원본 > Sharp 리사이즈 > WebP 변환 > CDN

#### 날짜 구분선

- 메시지 사이 날짜가 바뀔 때 표시
- 중앙 정렬: "2026년 3월 6일 목요일"
- 스타일: 12px `#9CA3AF`, 좌우 점선

#### 메시지 그룹핑

- 동일 발신자 연속 메시지: 첫 메시지만 아바타+닉네임, 이후는 indent + 타임스탬프만
- 5분 이내 연속 메시지: 마지막 메시지에만 타임스탬프 표시

## 비즈니스 로직

| 규칙 | 설명 |
|------|------|
| **실시간 채팅** | Firebase Realtime DB, onChildAdded 리스너 |
| **교환 확정 플로우** | 1. 한쪽 확정 요청 > 2. 양쪽 확정 > 3. 운송장 입력 > 4. 수취 확인 > 5. 완료 > 6. 후기 |
| **양쪽 확정** | 두 사용자 모두 "교환 확정" 클릭해야 확정 완료 |
| **운송장 입력** | 교환 확정 후에만 가능, 택배사+번호 필수 |
| **자동 완료** | 한쪽만 수취 확인 시 5일 후 자동 완료 처리 |
| **후기 요청** | 교환 완료 시 24시간 후 푸시 알림으로 후기 작성 요청 |
| **메시지 삭제** | 본인 메시지만, 전송 후 5분 이내, 상대에게는 "삭제된 메시지" 표시 |
| **이미지 전송** | S3 업로드 후 CDN URL 전송, 원본 + WebP 썸네일 |
| **채팅 보안** | 대화 기록 최소 6개월 암호화 보관 (분쟁 대비) |
| **계좌번호 감지** | Phase 1.5: 계좌번호 패턴 감지 시 경고 메시지 |
| **공구 채팅** | 주최자+참여자 단체 대화, 참여자 이름 목록 표시, 공구 상태 배너 연동 |
| **비로그인 차단** | 채팅방 직접 URL 접근 시 AUTH01-D 리다이렉트 |
| **직교환 시간 만료 처리** | `instant_meetup.time_window` 종료 시점에 해당 채팅방에 시스템 메시지 자동 발송 + FCM 알림(NOTI01 #16) + 제안 메타의 `instant_meetup` 표시를 "일반 직거래"로 전환 |

## 접근성

- 메시지 영역: `role="log"`, `aria-live="polite"` (새 메시지 알림)
- 전송 버튼: `aria-label="메시지 전송"`
- 이미지: `alt` 텍스트 "사용자가 보낸 이미지"
- 거래 카드: 버튼 포커스 순서 보장
- 타임스탬프: `aria-label="오후 3시 42분"`
