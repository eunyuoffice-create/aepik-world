## 화면 개요

공동구매 상세 정보를 확인하고 참여하는 화면입니다. 일반 공구는 참여 버튼을 제공하며, 세트 공구는 캐릭터 선점 UI를 제공합니다.

## 진입 경로

- 공구 목록 > 카드 탭
- 홈 > 인기 공구 카드 탭
- 알림 > 공구 알림 탭
- 채팅 > 공구 링크 탭

## 화면 구성 요소

### ① Header
- 뒤로가기 버튼
- 중앙: "공구 상세" 타이틀
- 우측: 공유(Share) 아이콘 + 신고(Report) 아이콘

### ② 이미지 갤러리
- 상품 이미지 스와이프 기능
- 페이지 인디케이터 (N/M)

### ③ 공구 정보 섹션

**태그**
- `[공구]` 또는 `[세트공구]` 태그 (파란색)
- 카테고리 태그 (애니/만화/소설, 아이돌/아티스트, 배우/무대, 게임)

**상품명**
- Bold, Large 텍스트

**가격**
- 일반 공구: `₩15,000/1인`
- 세트 공구: `₩8,000/캐릭터`

**배송비 표시**
- 배송비 포함 여부 명확히 표시

### ④ 진행자 정보
- 프로필 이미지 (원형)
- 닉네임
- 신뢰배지 (Lv.2 등)
- 거래횟수 (예: 거래 12회)
- 탭시 진행자 프로필로 이동

### ⑤ 참여 현황 (일반 공구)

**프로그레스 바**
- 큰 프로그레스 바 + `N/M명 (60%)`

**마감 정보**
- 최소 인원 표시 (예: 최소 20명)
- 마감일: `YYYY-MM-DD (D-N)` 형식
- 배송 예정일

### ⑤-B 캐릭터 선점 현황 (세트 공구)

**작품명 표시**
- 상단에 세트 공구 작품명 표시

**캐릭터 목록**
- 각 행 구성:
  - 캐릭터명
  - 선점 상태 (선점 가능 / 완료)
  - 프로그레스 바
  - 남은 수량 (예: 2/2 남음)

**마감된 캐릭터**
- "마감" 뱃지 표시
- 회색 처리 (비활성화)

**마감일/배송 예정일**
- 일반 공구와 동일

### ⑥ 상세 설명
- 진행자가 작성한 공구 상세 내용
- Markdown 또는 Plain Text 형식

### ⑦ 참여자 목록
- 현재 참여자 프로필 아이콘 나열 (원형)
- 최대 N명 표시
- "+N명 더보기" 링크

### ⑧ 하단 CTA 바 (Fixed)

**일반 공구**
- [찜 ♥️] + [참여하기] 버튼 (Full-width Blue)

**세트 공구**
- [찜 ♥️] + [캐릭터 선점하기] 버튼 (Full-width Blue)

**마감된 공구**
- 버튼 비활성 상태
- 텍스트: "마감된 공구입니다"

**이미 참여중**
- [찜 ♥️] + [참여 취소] 버튼 (Outline)

**비로그인 사용자**
- 로그인 유도 모달/페이지

**미인증 사용자**
- 본인인증 유도 모달/페이지

### ⑨ 참여 확인 바텀시트 (일반 공구)

**구성 요소**
- 상품명
- 가격
- 참여 수량 선택 (스피너)
- 총 결제금액 표시

**버튼**
- "참여 확정" 버튼

**안내문**
- "결제는 진행자와 직접 정산합니다"

### ⑨-B 캐릭터 선점 바텀시트 (세트 공구)

**캐릭터 선택**
- 캐릭터 목록 (체크박스 + 수량 조절)
- 선점 가능한 캐릭터만 활성화
- 비활성 캐릭터는 클릭 불가

**결제금액 계산**
- 총 결제금액 = 선점 수 × 단가

**버튼**
- "선점 확정" 버튼

## 비즈니스 로직

### 공구 상태 표시 (참여자 관점)

공구 상태에 따라 상세 화면의 표시가 달라집니다.

| 상태 | 뱃지 색상 | CTA 버튼 | 안내 메시지 |
|------|----------|---------|-----------|
| 모집중 | Green | [참여하기] / [캐릭터 선점하기] | - |
| 마감 (진행 결정 대기) | Amber | 비활성 | "진행자가 진행 여부를 결정 중입니다" |
| 진행 확정 | Blue | 비활성 | "공구가 확정되었습니다. 진행자 공지를 확인하세요" |
| 배송중 | Blue | 비활성 | "배송이 진행 중입니다" |
| 완료 | Gray | [후기 작성] | "공구가 완료되었습니다" |
| 무산 | Red | 비활성 | "공구가 무산되었습니다" + 사유 표시 |

### 무산 공구 표시
- 상단에 Red 배경 배너: "이 공구는 무산되었습니다"
- 진행자가 입력한 무산 사유 표시
- 입금자가 있는 경우: "환불은 진행자와 직접 정산해주세요" 안내
- CTA 영역: [진행자에게 채팅하기] 버튼 (환불 관련 문의용)

### 조기 마감 공구 표시
- 목표 인원 미달 상태에서 진행자가 조기 마감한 경우
- "목표 미달이지만 진행자가 진행을 결정했습니다" 안내

### 진행자 자격 검증
- 본인인증 완료 사용자만 공구 진행 가능
- 신뢰도 기준 충족 필요

### 마감일 자동 처리
- 마감일 도래 시 자동으로 "진행 결정 대기" 상태 진입
- 진행자가 "진행 확정" 또는 "무산 처리" 결정할 때까지 참여 불가
- 진행자 미응답 시 자동 무산 타이머 작동 (D+5 자동 무산)

### 선점 동시성 제어 (선착순)
- 세트 공구의 캐릭터 선점은 선착순 처리
- 동시 요청 시 트랜잭션 처리로 중복 방지

### 참여 취소
- 모집중 상태에서만 참여 취소 가능
- 마감 후 취소는 진행자와 채팅으로 협의

## 인터랙션

- 이미지 갤러리: 좌우 스와이프
- 진행자 정보: 탭시 프로필 페이지 이동
- 참여자 목록: "+N명 더보기" 탭시 전체 목록 모달
- 공유 버튼: 링크 공유 (카톡, URL 복사 등)
- 신고 버튼: 신고 모달 열기
- 찜 버튼: 찜 토글 (하트 채우기/해제)
- 참여하기/캐릭터 선점하기: 바텀시트 열기
