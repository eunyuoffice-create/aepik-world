# WANT01 동행 구함

## 메타 정보
| 항목 | 내용 |
|------|------|
| **화면코드** | WANT01 |
| **화면명** | 동행 구함 |
| **라우트** | /companion |
| **이용자** | PC / Mobile |
| **버전** | 1.4 |
| **작성일** | 2026-03-09 |
| **상태** | 초안 |

## 화면 개요
> 팬 이벤트(생일카페, 전시, 콘서트, 팝업스토어 등)에 함께 갈 동행을 찾는 게시판.
> 이벤트 캘린더(EVNT)와 연동되어, 특정 이벤트에 대한 동행을 쉽게 모집/신청할 수 있다.
> 교환/공구를 안 하는 유저도 동행 기능으로 유입 가능 -- 플랫폼 활성화 허브 역할.

## 진입 경로
- 홈 > 동행 구함 바로가기
- 이벤트 상세(EVNT02) > "동행 찾기" 버튼
- 마이페이지 > 내 동행글
- 알림 > 동행 관련 알림 탭
- 검색 결과 > 동행 탭

## 화면 구성 요소

### (1) Header
- 좌측: 페이지 제목 "동행 구함"
- 우측: 검색 아이콘 (`Search`) -- 탭 시 검색 페이지(SRCH01) 이동

### (2) 탭 바 (전체 / 내 동행)
- 2개 탭: **전체** / **내 동행**
- 전체: 모든 유저의 동행글 목록 (기본 활성)
- 내 동행: 본인 작성 동행글 + 신청한 동행글 (로그인 필요)
- 탭 하단 2px 인디케이터 (Blue #3B82F6)

### (3) 필터 바 (FilterChips)
- 가로 스크롤 칩 형태
- 필터 항목: 카테고리 `ChevronDown` / 작품 `ChevronDown` / 이벤트 유형 `ChevronDown` / 지역 `ChevronDown` / 상태 `ChevronDown`
- 상태 옵션: 모집중 / 마감 / 취소
- 이벤트 유형 옵션: 생일카페 / 전시 / 콘서트 / 팝업스토어 / 팬미팅 / 기타
- 칩 탭 시 바텀시트 열림 (COMM02 필터 바텀시트 패턴)
- 선택된 필터: 파란 배경(#3B82F6) + 흰 텍스트 + X 제거 버튼
- 필터 초기화: 우측 끝 "초기화" 텍스트 버튼

### (4) 정렬 + 결과 수
- 좌측: 전체 N개
- 우측: 정렬 드롭다운 -- 최신순(기본) / 동행일 임박순

### (5) 동행글 목록
- 카드 리스트 (세로 나열)
- **동행 카드** 구성:
  - 좌측: 대표 이미지 또는 이벤트 포스터 (80x80px, 라운딩 8px) / 없으면 카테고리 그라데이션 배경 + 아이콘
  - 우측 정보:
    - 상태 뱃지 (모집중: Green / 마감: Gray / 취소: Red)
    - 카테고리 태그 (카테고리 컬러)
    - 동행 제목 (14px, 600, 최대 2줄)
    - 이벤트명 (12px, #374151, 연결 이벤트 있으면 `Calendar` 아이콘 + 이벤트명)
    - 작품명 `dot` 캐릭터명 (12px, #6B7280)
    - 동행일 `dot` 장소 (10px, #9CA3AF, 예: "03.15(토) 오후 `dot` 홍대")
    - 모집 인원 표시 (10px, 예: "1/3명 신청")
  - 우측 끝: D-day 뱃지 (동행일 기준)
- **내 동행 탭**: 작성 글에 "끌어올리기" 버튼, 수정/삭제 메뉴 / 신청한 글에 신청 상태 표시
- 끌어올리기된 글: `Flame` 아이콘 + "UP" 뱃지
- 카드 탭 시: 동행 상세 페이지(WANT02)로 이동
- 무한 스크롤 (Intersection Observer)
- 스켈레톤 UI: 로딩 시 카드 형태 회색 박스 펄스 애니메이션

### (6) FAB (Floating Action Button)
- 우하단 고정 (bottom: 80px, right: 16px)
- 아이콘: `Edit` (+) -- 동행글 등록
- 비로그인 시: 로그인 유도 팝업
- 로그인 + 미인증 시: 본인인증 유도 바텀시트 (JIT)

### (7) 동행글 등록 바텀시트 (인라인)
- FAB 탭 시 바텀시트 슬라이드 업 (별도 페이지 이동 없음)
- 등록/수정 공용 폼 (mode=create | edit)
- 필드:
  - 연결 이벤트 (선택, 탭 시 이벤트 검색 피커 열림 -- EVNT 캘린더 연동)
    - 이벤트 선택 시 이벤트명/장소/일정 자동 입력
    - 없으면 직접 입력 모드
  - 동행 제목 (필수, placeholder: "함께 갈 동행을 구해요!")
  - 카테고리 (필수, 칩 선택: 애니/만화/소설 / 아이돌/아티스트 / 배우/무대 / 게임)
  - 작품 선택 (필수, 드롭다운/검색)
  - 캐릭터/멤버 선택 (필수, 드롭다운)
  - 동행 일시 (필수, 날짜 피커 + 시간대 선택: 오전/오후/저녁/종일)
  - 장소 (필수, 연결 이벤트 시 자동 입력, 없으면 텍스트 입력)
  - 모집 인원 (필수, 스테퍼 1~5명, 기본 1명)
  - 상세 설명 (선택, textarea: 동행 조건, 선호 사항 등)
  - 이미지 (선택, 최대 3장)
- CTA: "동행 등록" 버튼 (Primary)
- 수정 모드 CTA: "수정 저장" 버튼 (Primary)
- 수정 잠금:
  - 신청 이력 1건 이상(상태 무관)이면 핵심 조건(동행일/장소/모집인원) 수정 불가
  - 잠금 시 핵심 필드 비활성 + 안내 배너 노출
  - 안내 문구: "신청 이력이 있어 핵심 조건을 수정할 수 없어요. 새 글로 다시 모집해 주세요."
- 등록 즉시 목록 반영 + 매칭 알림 발송

### (8) 하단 네비게이션 (BottomNav)
- 5개 탭: 홈(`Home`) | 교환(`Repeat`) | 이벤트(`Calendar`) | 채팅(`MessageCircle`) | 마이(`User`)
- 현재: 별도 활성 탭 없음 (동행은 하단 탭에 포함되지 않음)
- 진입 경로에 따라 해당 탭 활성

## 데이터 필드

### API: GET /api/companion

**Request (Query Parameters):**

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| page | number | X | 페이지 번호 (기본 1) |
| limit | number | X | 페이지당 개수 (기본 20) |
| category | string | X | 카테고리 필터 |
| work_id | string | X | 작품 필터 |
| event_type | string | X | 이벤트 유형 필터 |
| region | string | X | 지역 필터 |
| status | string | X | 상태 (recruiting/closed/cancelled) |
| sort | string | X | 정렬 (latest/date_soon) |
| mine | boolean | X | 내 동행글만 (true/false) |
| event_id | string | X | 특정 이벤트의 동행글 필터 |

**Response:**

| 필드 | 타입 | 설명 |
|------|------|------|
| items | array | 동행글 목록 |
| items[].id | string | 동행글 ID |
| items[].title | string | 동행 제목 |
| items[].event | object/null | 연결 이벤트 정보 {id, name, type, date_range} |
| items[].work | object | 작품 정보 {id, name} |
| items[].character | object | 캐릭터 정보 {id, name} |
| items[].category | string | 카테고리 (anime_manga_novel/idol_artist/actor_stage/game) |
| items[].companion_date | string | 동행 일시 |
| items[].time_slot | string | 시간대 (morning/afternoon/evening/all_day) |
| items[].location | string | 장소 |
| items[].max_members | number | 모집 인원 |
| items[].current_members | number | 현재 신청 인원 |
| items[].description | string | 상세 설명 (nullable) |
| items[].images | string[] | 이미지 URL (nullable) |
| items[].status | string | 상태 (recruiting/closed/cancelled) |
| items[].user | object | 등록자 {id, nickname, trust_level, avatar} |
| items[].created_at | string | 등록일 |
| items[].is_bumped | boolean | 끌어올리기 여부 |
| total | number | 전체 개수 |
| has_next | boolean | 다음 페이지 여부 |

### API: POST /api/companion (동행글 등록)

**Request Body:**

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| title | string | O | 동행 제목 (max 50자) |
| event_id | string | X | 연결 이벤트 ID (EVNT 캘린더에서 선택) |
| event_name | string | O | 이벤트명 (event_id 있으면 자동, 없으면 직접 입력) |
| category | string | O | 카테고리 |
| work_id | string | O | 작품 ID |
| character_id | string | O | 캐릭터 ID |
| companion_date | string | O | 동행 일시 (ISO date) |
| time_slot | string | O | 시간대 (morning/afternoon/evening/all_day) |
| location | string | O | 장소 (event_id 있으면 자동) |
| max_members | number | O | 모집 인원 (1~5, 기본 1) |
| description | string | X | 상세 설명 (max 500자) |
| images | file[] | X | 이미지 (최대 3장) |

### API: PATCH /api/companion/:id (동행글 수정)

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| title | string | X | 동행 제목 |
| companion_date | string | X | 동행 일시 (잠금 시 수정 불가) |
| location | string | X | 장소 (잠금 시 수정 불가) |
| max_members | number | X | 모집 인원 (잠금 시 수정 불가) |
| description | string | X | 상세 설명 (잠금 후에도 수정 가능) |
| images | file[] | X | 이미지 (최대 3장) |

## 비즈니스 로직

| # | 규칙 | 설명 |
|---|------|------|
| 1 | 자동 마감 | 동행 일시 경과 시 자동으로 "마감" 상태 전환 |
| 2 | 인원 충족 마감 | 신청 수락 인원 = 모집 인원이면 자동 "마감" |
| 3 | 매칭 알림 | 동행글 등록 시 관심 작품/캐릭터 유저 + 찜 이벤트 유저에게 알림 |
| 4 | 끌어올리기 | 1일 1회 무료, 추가 시 유료 (500원/회, 1일 최대 3회) |
| 5 | 차단 사용자 | 차단한 사용자의 동행글 미노출 |
| 6 | 신고 제재 | 3건 이상 신고 시 관리자 검토 전까지 비공개 |
| 7 | 최대 등록 수 | 유저당 활성 동행글 최대 5건 |
| 8 | 신청 시 채팅 | 동행 신청 수락 시 1:1 채팅방 자동 생성 |
| 9 | 이벤트 연동 | 연결 이벤트 삭제 시 동행글 유지 (이벤트명 텍스트 보존) |
| 10 | 본인인증 게이팅 | 동행 등록/신청 시 로그인+본인인증 필수 (미인증은 인증 유도) |
| 11 | 수정 잠금 | 신청 이력 1건 이상(상태 무관)이면 핵심 조건 수정 불가, 상세 설명만 수정 가능 |
