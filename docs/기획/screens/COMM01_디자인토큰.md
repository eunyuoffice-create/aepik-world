# COMM01 디자인 토큰 (Design Tokens)

> **화면 ID**: COMM01
> **화면명**: 디자인 토큰
> **유형**: 공통 — 디자인 시스템 레퍼런스
> **버전**: v0.1
> **최종 수정**: 2026-03-06

---

## 1. 화면 개요

| 항목 | 내용 |
|------|------|
| **목적** | 서비스 전체에 적용되는 디자인 토큰(컬러, 타이포, 스페이싱, 라운딩, 그림자, 모션 등)을 시각적으로 정리한 레퍼런스 페이지 |
| **대상** | 개발자, 기획자 (프로덕트 내부 참조 문서) |
| **라우트** | 별도 라우트 없음 — 내부 문서 |
| **연관 문서** | 디자인 컨셉가이드 (`docs/애픽월드_디자인_컨셉가이드.md`) |

### 2.2 타이포그래피

| 레벨 | 크기 | 웨이트 | 행간 | 용도 |
|------|------|--------|------|------|
| Display | 28px | Bold 700 | 1.3 | 메인 배너 타이틀 |
| H1 | 24px | Bold 700 | 1.3 | 페이지 제목 |
| H2 | 20px | SemiBold 600 | 1.4 | 섹션 제목 |
| H3 | 16px | SemiBold 600 | 1.4 | 카드 제목, 상품명 |
| Body L | 16px | Regular 400 | 1.6 | 본문 |
| Body M | 14px | Regular 400 | 1.5 | 일반 텍스트 |
| Caption | 12px | Medium 500 | 1.4 | 보조 정보, 태그 |
| Micro | 10px | Medium 500 | 1.3 | 뱃지, 작은 레이블 |

폰트: **Pretendard** (대안 Noto Sans KR) / 영문: **Inter**

### 2.3 스페이싱 (4px 그리드)

4px, 8px, 12px, 16px, 20px, 24px, 32px, 48px

### 2.4 라운딩 (Border Radius)

| 요소 | 값 |
|------|------|
| 뱃지, 작은 태그 | 4px |
| 버튼, 인풋 | 8px |
| 카드, 이미지 | 12px |
| 모달 | 16px |
| 아바타 | 50% |
| Pill (태그, 칩) | 999px |

### 2.5 그림자 (Elevation)

| 레벨 | 값 | 용도 |
|------|-----|------|
| Level 0 | 없음 (border만) | 기본 카드, 인풋 |
| Level 1 | `0 1px 3px rgba(0,0,0,.08)` | 호버, 드롭다운 |
| Level 2 | `0 4px 12px rgba(0,0,0,.1)` | 모달, 바텀시트, FAB |
| Level 3 | `0 8px 24px rgba(0,0,0,.12)` | 토스트, 알림 팝업 |

### 2.6 반응형 브레이크포인트

| 이름 | 범위 | Tailwind | 레이아웃 |
|------|------|----------|---------|
| Mobile | 0~767px | 기본 | 1컬럼, 풀폭, BottomNav |
| Desktop | 768px~ | `md:` | 중앙 1200px, 사이드바 |

### 2.7 모션 & 트랜지션

| 요소 | 듀레이션 | 이징 | 속성 |
|------|----------|------|------|
| 버튼 호버 | 150ms | ease-out | background, transform |
| 카드 호버 | 200ms | ease-out | shadow, translateY |
| 페이지 전환 | 200ms | ease-in-out | opacity |
| 모달 | 250ms | ease-out | opacity, scale |
| 바텀시트 | 300ms | ease-out | translateY |
| 스켈레톤 | 1500ms | ease-in-out | pulse |
| 토스트 | 300ms | ease-out | translateY, opacity |
