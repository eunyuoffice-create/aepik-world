## 화면 개요

> 로그인 사용자의 프로필 요약과 활동 메뉴를 제공하는 마이페이지 메인 화면. 프로필 편집 모드와 타인 프로필 열람 기능을 포함한다.

---

## 진입 경로

- 하단 네비게이션 > 마이 탭
- 교환/공구 상세 > 작성자 프로필 영역 탭 (타인 프로필)
- 채팅방 > 상대방 프로필 탭 (타인 프로필)
- 검색 결과 > 카드 내 작성자 탭 (타인 프로필)

## A. 마이페이지 메인 (/my)

### A-1. 앱 헤더

- 좌측: 페이지 제목 "마이페이지"
- 우측: 설정 아이콘 (톱니바퀴) -- 탭 시 USER03 설정 이동

### A-2. 프로필 카드 영역

| 요소 | 스펙 |
|------|------|
| 프로필 이미지 | 72px 원형, 미등록 시 기본 아바타 |
| 닉네임 | 18px Bold |
| 인증 뱃지 | 본인인증 완료 시 닉네임 우측에 파란 체크 아이콘 (16px) |
| 신뢰 지표 행 | 교환 N회 · 긍정 N% · 가입일 YYYY.MM |
| 최애 표시 | 닉네임 하단에 "최애: [캐릭터/아이돌명]" (있을 때만 표시) |
| 프로필 편집 버튼 | Secondary Small 버튼 "프로필 편집" -- 탭 시 모드 B |

### A-3. 내 활동 퀵 카운트

- 가로 3등분 카드 (탭 가능)
- 항목: 내 교환글 N / 내 공구 N / 내 동행글 N
- 탭 시 USER02의 해당 탭으로 이동

### A-4. 메뉴 리스트

리스트 형태 (아이콘 + 라벨 + 우측 화살표)

| 그룹 | 메뉴 항목 | 이동 대상 | 비고 |
|------|-----------|-----------|------|
| **내 활동** | 교환 내역 | USER02 교환내역 탭 | 완료된 거래 |
| | 받은 후기 | USER02 후기 탭 | |
| | 찜 목록 | USER02 찜 탭 | 찜 카운트 뱃지 |
| | 보낸/받은 교환 제안 | EXCH05 교환신청함 | |
| **알림** | 키워드 알림 설정 | USER03 키워드 설정 | 등록 키워드 N개 |
| **지원** | 공지사항 | 공지 목록 | |
| | 고객센터 (FAQ / 1:1 문의) | STAT01 고객센터 | |
| **정보** | 이용약관 | STAT01 약관 | |
| | 개인정보처리방침 | STAT01 정책 | |
| | 앱 버전 | - | 현재 버전 표시 (탭 불가) |

### A-5. 로그아웃 버튼

- 메뉴 리스트 하단, Ghost 텍스트 버튼 "로그아웃"
- 탭 시 확인 팝업: "로그아웃 하시겠습니까?" / [취소] [로그아웃]

## B. 프로필 편집 (/my/edit)

### B-1. 헤더

- 좌측: 뒤로가기 화살표 (마이페이지로 복귀)
- 중앙: "프로필 편집"
- 우측: "저장" 텍스트 버튼 (변경 사항 있을 때만 활성)

### B-2. 프로필 이미지 변경

| 요소 | 스펙 |
|------|------|
| 이미지 영역 | 100px 원형, 중앙 정렬 |
| 카메라 오버레이 | 우하단 28px 원형 버튼 (카메라 아이콘) |
| 이미지 소스 | 탭 시 액션시트: 카메라 촬영 / 앨범에서 선택 / 기본 이미지로 변경 |
| 업로드 제한 | 5MB, JPG/PNG/WebP, 리사이즈 400x400 |

### B-3. 닉네임 변경

| 요소 | 스펙 |
|------|------|
| 인풋 필드 | 현재 닉네임 pre-fill, 44px 높이 |
| 글자 수 카운터 | 우측 하단 "N/12" |
| 실시간 중복 확인 | debounce 500ms, 사용 가능(초록) / 이미 사용 중(빨강) |
| 규칙 | 2~12자, 한글/영문/숫자, 특수문자/공백 불가 |
| 금칙어 검사 | 서버사이드 필터링 |

### B-4. 최애 표시 설정

| 요소 | 스펙 |
|------|------|
| 라벨 | "최애 표시 (프로필에 공개)" |
| 인풋 필드 | 자유 입력, 최대 20자 |
| 예시 | placeholder: "예: 영지 (블루아카이브)" |
| 삭제 | 인풋 우측 X 버튼으로 비우기 |

### B-5. 관심 카테고리 변경

- 현재 선택된 카테고리 태그 표시
- "변경" 버튼 탭 시 AUTH02 Step 4와 동일한 카테고리 선택 바텀시트
- 4개 카테고리 카드 (Purple/Pink/Amber/Cyan), 복수 선택 가능

## C. 타인 프로필 (/user/[userId])

### C-1. 헤더

- 좌측: 뒤로가기 화살표
- 중앙: 닉네임 표시
- 우측: 더보기 메뉴 (점 3개) -- 신고하기 옵션

### C-2. 프로필 영역

| 요소 | 스펙 |
|------|------|
| 프로필 이미지 | 80px 원형 |
| 닉네임 | 18px Bold + 인증 뱃지 |
| 최애 표시 | 닉네임 하단 |
| 신뢰 지표 | 교환 N회 · 긍정 N% · 가입일 YYYY.MM |
| 가입일 라벨 | "YYYY년 MM월 가입" |

### C-3. 등록 매물 탭

- 탭 바: 교환글 N / 공구 N / 동행글 N
- 각 탭 내 2열 그리드 카드 (ProductCard)
- 활성 매물만 표시 (완료/삭제 제외)
- 빈 상태: "등록된 [교환글/공구/동행글]이 없습니다"

### C-4. 받은 후기

- 프로필 영역 하단 또는 탭 바에 "후기 N" 탭 추가
- 후기 카드: 긍정/부정 아이콘 + 작성자 닉네임 + 텍스트 + 날짜
- 정렬: 최신순

### C-5. 신고하기

- 헤더 더보기 메뉴 > "신고하기" 선택 시 바텀시트
- 신고 사유 선택: 욕설/비하, 사진 도용, 사기 의심, 기타
- 증거 첨부 (선택, 최대 3장)
- 상세 설명 텍스트 (선택, 최대 500자)
- [신고 접수] Danger 버튼

| 11 | 금칙어 필터 | 닉네임/최애에 금칙어 포함 시 저장 차단 + 에러 메시지 |
| 12 | 무산 이력 표시 | 공구 무산 3회 이상 사용자는 타인 프로필에 "무산 이력 N회" 표시 |

## 화면 상태

| 상태 | 조건 | 표시 |
|------|------|------|
| 로딩 중 | 데이터 fetch 중 | 스켈레톤 UI (프로필 카드 + 메뉴) |
| 에러 | 네트워크 오류 | "연결이 불안정합니다" + 재시도 버튼 |
| 비로그인 | 미로그인 상태에서 /my 접근 | AUTH01-D 로그인 유도 바텀시트 |
| 타인 프로필 빈 상태 | 매물 0건 | 빈 상태 일러스트 + "등록된 매물이 없습니다" |
| 탈퇴 사용자 | 탈퇴된 userId 접근 | "탈퇴한 사용자입니다" 안내 |
| 정지 사용자 | 정지된 userId 접근 | "이용이 제한된 사용자입니다" 안내 |
