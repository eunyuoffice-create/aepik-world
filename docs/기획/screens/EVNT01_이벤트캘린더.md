## 화면 개요

> 팬들이 생일카페, 지하철 광고, LED 트럭, 전광판 등 다양한 팬 이벤트 일정을 캘린더/리스트/지도 뷰로 한 곳에서 확인하는 화면. 멀티 팬덤(애니/만화/소설 + 아이돌/아티스트 + 배우/무대 + 게임) 이벤트를 모두 커버하여 기존 K팝 전용 서비스와 차별화한다.

## 진입 경로

- 홈 > 다가오는 이벤트 > 더보기
- 검색 결과 > 이벤트 탭
- 알림 > 이벤트 관련 알림 탭

## 화면 구성 요소

### (1) Header
- 좌측: 페이지 제목 "이벤트"
- 우측: 검색 아이콘 (`Search`) -- 탭 시 검색 페이지(SRCH01) 이동

### (2) 곧 다가올 이벤트 배너 (NEW)
- **위치**: 헤더 바로 아래, 뷰 토글 탭 위
- **형태**: 수평 캐러셀 (자동 슬라이드 5초 + 수동 스와이프)
- **배너 카드 구성**:
  - 배경: 그라데이션 오버레이 + 이벤트 대표 이미지 (16:9 비율, 높이 ~140px)
  - 좌측 텍스트: D-day 뱃지 (`D-3`, `D-1`, `오늘`) + 이벤트명 (16px, Bold, White) + 장소 (12px, White 80%)
  - 인디케이터 도트: 하단 중앙
- **데이터 소스**: 오늘 기준 D-7 이내 시작하는 이벤트 중 찜 수 상위 5건
- **탭 시**: EVNT02 이벤트 상세 이동
- **빈 상태**: 다가올 이벤트 없으면 배너 영역 자체를 숨김

#### 배너/팝업 신청 스트립
- **위치**: 곧 다가올 이벤트 배너 바로 아래
- **구성**:
  - 좌측: 아이콘 (32px, Blue 배경 원형)
  - 안내 텍스트: "내 이벤트도 배너/팝업으로 홍보하기"
  - 슬롯 상태: 컬러 도트 + 잔여 슬롯 요약 (예: "메인 배너 3슬롯 / 팝업 2슬롯 남음")
    - Green 도트: 여유
    - Amber 도트: 거의 마감
    - Red 도트: 매진
  - [신청] 버튼 (Blue pill)
- **[신청] 탭 시**: BANR01 (배너/팝업 슬롯 현황) 이동
- **데이터 소스**: GET /api/banner/status (슬롯 요약)
- **배너 영역 숨김 시에도** 스트립은 독립 노출 (다가올 이벤트 없어도 스트립은 표시)

### (3) 뷰 토글 탭
- 3개 탭: 캘린더 (`Calendar`) / 리스트 (`List`) / 지도 (`Map`)
- 탭 하단 2px 인디케이터 (Blue #3B82F6)
- 기본 활성: 캘린더 뷰
- 뷰 전환 시 필터 상태 유지

### (4) 필터 바 (FilterChips)
- 가로 스크롤 칩 형태
- 필터 항목: 작품 ▾ / 캐릭터 ▾ / 이벤트 유형 ▾ / 지역 ▾
- 칩 탭 시 바텀시트 열림 (COMM02 필터 바텀시트 패턴)
- 선택된 필터: 파란 배경(#3B82F6) + 흰 텍스트 + X 제거 버튼
- 필터 초기화: 우측 끝 "초기화" 텍스트 버튼
- 이벤트 유형 옵션: 생일카페 / 지하철 광고 / LED 트럭 / 전광판 / 버스 광고 / 카페 이벤트 / 기타

### (5) 캘린더 뷰 — 모바일: 날짜 스크롤러 / PC: 월간 그리드

#### 모바일 (< 768px): 가로 날짜 스크롤러
- **월 네비게이션**: `ChevronLeft` / 현재 월(YYYY년 M월) / `ChevronRight` / "오늘" 뱃지 버튼
- **날짜 스크롤러** (가로 스크롤 바):
  - 셀 구성: 요일(월/화/..., 10px) + 날짜(숫자, 16px Bold) + 카테고리 도트(최대 4색)
  - 셀 크기: 48x64px, gap 4px
  - 오늘: 파란 원형 배경(#3B82F6, 흰 텍스트)
  - 선택된 날짜: 파란 테두리 링(2px)
  - 이벤트 없는 날짜: 도트 없음, 회색 텍스트
  - 과거 날짜: 연회색 텍스트(#D1D5DB)
  - 카테고리 도트 컬러: Purple(#8B5CF6), Pink(#EC4899), Amber(#F59E0B), Cyan(#06B6D4)
- **진입 시 기본**: 오늘 날짜 중앙 정렬, 오늘 선택 상태
- **스크롤 범위**: 과거 30일 ~ 미래 90일

#### PC (>= 768px): 풀 월간 캘린더 그리드
- 기존 월간 그리드 UI 유지
- 요일 헤더: 일 월 화 수 목 금 토 (일요일 빨간색)
- 날짜 셀에 카테고리 컬러 도트 + 이벤트 수 표시
- 날짜 클릭 시 우측 또는 하단에 이벤트 리스트 패널

### (6) 선택 날짜 이벤트 리스트 (개선)
- **위치**: 날짜 스크롤러(모바일) 또는 캘린더(PC) 아래
- **날짜 헤더**: "3월 15일 (토)" + 이벤트 수 뱃지 ("3건")
- **이벤트 카드** (리치 카드):
  - 좌측: 대표 이미지 썸네일 (72x72px, 라운딩 8px) — 없으면 유형별 기본 아이콘
  - 우측 정보:
    - 1행: 이벤트 유형 태그 (카테고리 컬러 소프트 배경) + D-day 뱃지 (선택)
    - 2행: 이벤트명 (14px, 600, 최대 2줄)
    - 3행: `MapPin` 장소명 (12px, #6B7280)
    - 4행: `Clock` 운영시간 (12px, #6B7280) — 예: "11:00 ~ 19:00"
  - 우측 끝: 찜 아이콘 (`Heart` 빈/채움)
- **카드 탭 시**: EVNT02 이벤트 상세 이동
- **빈 상태**: "이 날짜에는 이벤트가 없어요" + 이벤트 있는 가장 가까운 날짜 바로가기 링크
- **무한 스크롤**: 해당 날짜 이벤트 10건씩 페이징

### (7) 리스트 뷰
- 날짜별 그룹핑 (오늘 ~ 30일 후)
- **날짜 그룹 헤더**: "3월 7일 (금) - 오늘" / "3월 8일 (토) - 내일" / "3월 15일 (토)"
- **이벤트 카드** (리스트 아이템):
  - 좌측: 대표 이미지 (80x80px, 라운딩 8px) 또는 기본 아이콘
  - 우측 정보:
    - 이벤트 유형 태그 (카테고리 컬러 배경)
    - 이벤트명 (14px, 600, 최대 2줄)
    - 날짜 + 시간 (`Calendar` 아이콘, 12px, #6B7280)
    - 장소 (`MapPin` 아이콘, 12px, #6B7280, 1줄 ellipsis)
  - 우측 끝: 찜 아이콘 (`Heart` 빈/채움)
- 카드 탭 시 EVNT02 이벤트 상세 이동
- 이벤트 없는 날짜는 건너뜀

### (8) 지도 뷰
- 카카오맵 API 전체 화면 표시
- **이벤트 핀**: 카테고리 컬러 마커 (위치별)
  - 같은 위치 다수 이벤트: 클러스터 마커 (숫자 표시)
- **현재 위치 버튼**: 우하단 FAB (`Navigation` 아이콘)
- **핀 탭 시**: 하단 시트에 이벤트 정보 카드 표시
  - 카드 구성: 이미지 + 이벤트명 + 날짜 + 거리 + [상세 보기] 버튼
- **지도 이동 시**: "이 지역 검색" 버튼 표시 → 탭 시 해당 영역 이벤트 재검색
- 기본 범위: 서울 중심 / 위치 권한 허용 시 현재 위치 중심

### (9) FAB (이벤트 등록)
- 우하단 플로팅 버튼 (`Plus` 아이콘, 56px 원형, Blue #3B82F6)
- 탭 시:
  - 로그인 상태: **EVNT03 이벤트 등록 페이지로 이동** (별도 페이지)
  - 비로그인 상태: 로그인 필요 토스트 + AUTH01 이동
- 지도 뷰에서는 현재 위치 버튼과 겹치지 않도록 위치 조정 (현위치 버튼 위)

> [v0.2 변경] 이벤트 등록 폼을 EVNT01 인라인에서 **EVNT03 별도 페이지**로 분리. 상세 스펙은 EVNT03_이벤트등록.md 참조.

### (10) 하단 네비게이션
- 5탭: 홈(`Home`) / 교환(`Repeat`) / 이벤트(`Calendar`) / 채팅(`MessageCircle`) / 마이(`User`)
- 현재 활성: **이벤트** (Blue #3B82F6)
