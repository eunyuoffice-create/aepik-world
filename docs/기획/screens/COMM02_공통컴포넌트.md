## 2. 컴포넌트 목록

### 2.1 버튼 (Buttons)

#### 유형

| 유형 | 배경 | 텍스트 | 보더 | 용도 |
|------|------|--------|------|------|
| **Primary** | `#3B82F6` | `#FFF` | 없음 | 핵심 CTA (교환 제안, 공구 참여) |
| **Secondary** | `#FFF` | `#3B82F6` | 1px `#3B82F6` | 보조 액션 (찜, 공유) |
| **Ghost** | 투명 | `#4B5563` | 없음 | 텍스트 버튼 (더보기, 취소) |
| **Danger** | `#EF4444` | `#FFF` | 없음 | 삭제, 신고 |
| **Disabled** | `#F3F4F6` | `#9CA3AF` | 없음 | 비활성 |

#### 사이즈

| 사이즈 | 높이 | 좌우 패딩 | 폰트 크기 | 용도 |
|--------|------|-----------|-----------|------|
| Large | 48px | 24px | 16px | 메인 CTA, 폼 하단 |
| Medium | 40px | 16px | 14px | 일반 버튼 |
| Small | 32px | 12px | 12px | 카드 내, 태그 옆 |

#### 사용 원칙
- 화면당 Primary CTA는 최대 1~2개
- 파괴적 액션은 Danger 버튼 또는 확인 모달 필수
- 폼 하단 CTA는 풀폭(Full Width) Large
- 아이콘 버튼: 아이콘+텍스트 조합 또는 아이콘 단독 (최소 44x44px)

- 20x20px, 보더 2px `#D1D5DB`, 라운딩 4px
- 체크 시: 배경 `#3B82F6`, 체크마크 white

#### 라디오
- 20x20px, 보더 2px `#D1D5DB`, 원형
- 선택 시: 보더 `#3B82F6`, 내부 원 10px `#3B82F6`

#### 토글 (Toggle Switch)
- 트랙: 44x24px, 라운딩 12px
- OFF: 배경 `#D1D5DB`
- ON: 배경 `#3B82F6`
- 노브: 20x20px, white, shadow

#### 사용 원칙
- 필수 항목: 레이블 옆 붉은 별(*) 표시
- 에러 메시지: 인풋 바로 아래
- 글자 수 힌트: 하단 우측 또는 인풋 내 우측

### 2.4 팝업 & 오버레이

#### 확인 모달 (Confirm Modal)
- 용도: 되돌릴 수 없는 액션 (수락, 삭제, 탈퇴)
- 라운딩: 16px
- 그림자: Level 2
- 배경 딤: `#000` 50% opacity
- 진입: 250ms scale(0.95→1) + opacity
- 구조: 제목 + 본문 + 액션 버튼(취소/확인)
- ESC 키 또는 딤 탭으로 닫기

#### 바텀시트 (Bottom Sheet)
- 용도: 선택지 제공 (사유 선택, 정렬, 필터)
- 라운딩: 16px 16px 0 0
- 진입: 300ms translateY (하단→상단)
- 드래그 핸들: 40x4px, `#D1D5DB`, 중앙
- Desktop(768px+)에서는 중앙 모달로 전환
- 드래그 닫기: 아래로 100px+ 드래그 시

#### 토스트 (Toast)
- 용도: 성공/에러 피드백
- 배경: `#111827`, 텍스트 white
- 라운딩: 12px, 그림자 Level 3
- 위치: 하단 중앙
- 자동 닫힘: 3초
- 진입: 300ms translateY + opacity

#### 스낵바 (Snackbar)
- 용도: 되돌리기 가능한 액션
- 배경: `#374151`, 텍스트 white
- 라운딩: 8px
- 액션 버튼: `#93C5FD` 텍스트
- 유지: 5초 (액션 있을 때)

### 2.5 카드 (Product Card)

| 속성 | 값 |
|------|------|
| 보더 | 1px `#E5E7EB` |
| 라운딩 | 12px |
| 이미지 비율 | 1:1, object-fit: cover |
| 이미지 라운딩 | 12px 12px 0 0 |
| 호버 | shadow-md + translateY(-2px) 200ms |
| 내부 패딩 | 10px |

구조: 이미지 → 태그(거래유형+카테고리) → 상품명(최대 2줄) → 메타(작품·캐릭터) → 판매자 신뢰도

### 2.6 빈 상태 (Empty State)

- 라인 아이콘(48px, opacity 0.4) + 안내 메시지(14px, Gray 400) + CTA 버튼
- 메시지: 친근한 톤 ("아직 등록된 교환 상품이 없어요")
- CTA 버튼 항상 포함

### 2.7 스켈레톤 로딩 (Skeleton)

- shimmer 애니메이션: 1.5s ease-in-out
- 실제 콘텐츠와 동일한 크기/위치
- 데이터 로드 후 fade-in (200ms)

### 2.8 네비게이션

- 로고(좌측) + 검색/알림(우측)

- ← 뒤로가기 + 화면명
- 높이: ~48px

- 5탭: 홈, 교환, 이벤트, 채팅, 마이
- 비활성: Gray 400, 활성: Blue
- 768px 이상에서 숨김 (`flex md:hidden`)

#### 탭 바 (Tab Bar)
- 활성 탭: Blue 텍스트 + Blue 하단 보더 2px
- 비활성 탭: Gray 400

#### FAB (플로팅 액션 버튼)
- 56x56px, 라운딩 16px
- 배경 `#3B82F6`, 그림자 Level 2
- 우하단 고정 (bottom 80px, right 16px)

| 필터 바텀시트 칩 탭 | 선택 토글, 복수 선택 (정렬만 단일 선택) |

| 항목 | 스펙 |

#### 썸네일

| 항목 | 스펙 |
|------|------|
| **크기** | 100x100px |
| **라운딩** | 8px |
| **object-fit** | cover |

- 우측 하단 "N/M" 형식 (예: 3/10)
- 최대치 도달 시 빨간색 텍스트

#### 사용 규칙
- 교환 등록: 최대 10장
- 공구 등록: 최대 5장
- 최소 1장 필수 (업로드 전 CTA 비활성)
- 첫 번째 이미지 자동 대표 지정 (변경 가능)
- S3 업로드 파이프라인: 원본 → Sharp 리사이즈 → WebP 변환 → CloudFront CDN
