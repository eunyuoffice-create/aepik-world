## 화면 개요

> 보유 굿즈 정보, 희망 교환 조건, 거래 방식을 3단계로 입력하여 교환글을 등록하는 폼 화면. 수정 모드에서도 동일 화면을 사용하며, 기존 데이터를 프리필(prefill)한다.

---

## 진입 경로

- 교환 목록(EXCH01) > FAB 탭
- 교환 상세(EXCH02) > 본인 글 [수정] 탭

## 선행 조건

- 로그인 필수
- 본인인증 필수 (미인증 시 인증 유도 바텀시트)

## 화면 구성 요소

### ① Header
- 좌측: ← 뒤로가기 (작성 중 이탈 시 확인 다이얼로그)
- 중앙: "교환글 등록" (수정 모드: "교환글 수정")

### ② 스텝 인디케이터
- 3단계 표시: ① 보유 굿즈 → ② 원하는 굿즈 → ③ 거래 방식
- 현재 스텝: Blue 원 + Bold 텍스트
- 완료 스텝: Green 체크 ✓
- 미진행 스텝: Gray 원

### === Step 1: 보유 굿즈 ===

### ③ 사진 업로드
- 최대 5장 (JPG/PNG, 10MB 이하)
- 드래그 정렬 가능 (첫 번째 이미지 = 대표 이미지)
- + 버튼으로 추가, ✕ 버튼으로 삭제
- 카메라 / 앨범 선택 시트

### ④ 작품 선택
- 검색 인풋 + 자동완성 드롭다운
- 검색 결과 없을 시: "직접 추가" 버튼 → 관리자 승인 후 정식 등록
- 필수 항목 (*)

### ⑤ 캐릭터 선택
- 작품 선택 후 드롭다운 활성화
- 해당 작품의 등록된 캐릭터 목록 표시
- "직접 추가" 가능
- 필수 항목 (*)

### ⑥ 굿즈 타입 선택
- 드롭다운: 포토카드 / 아크릴 / 캔뱃지 / 피규어 / 인형 / 엽서/포스터 / 의류 / 기타
- 필수 항목 (*)

### ⑦ 상품 상태 + 수량
- 상태: S급/A급/B급/C급 라디오 버튼 (각 등급 설명 툴팁)
  - S급: 미개봉
  - A급: 거의 새 것 (1~2회 확인)
  - B급: 사용감 있음
  - C급: 하자 있음 (하자 설명 필수)
- 보유 수량: 숫자 스테퍼 (기본 1, 최대 99)
- 필수 항목 (*)

### ⑧ 상품명
- 텍스트 인풋 (최대 50자)
- 플레이스홀더: "예: 하이큐 히나타 아크릴 스탠드"
- 필수 항목 (*)

### [다음] 버튼

### === Step 2: 원하는 굿즈 ===

### ⑨ 원하는 작품/캐릭터
- 작품 검색 선택 (Step 1과 동일 UI)
- 캐릭터 선택
- "아무거나 괜찮아요" 체크박스 (선택 시 작품/캐릭터 비활성)
- 선택 항목

### ⑩ 원하는 굿즈 타입
- 드롭다운 (Step 1과 동일 옵션)
- "아무 타입 괜찮아요" 체크박스
- 선택 항목

### ⑪ 차액 설정
- 체크박스: "차액 협의 가능"
- 체크 시 활성화:
  - 방향: +받을게요 / -줄게요 토글
  - 금액 인풋 (1,000원 단위)
- 선택 항목

### [다음] 버튼

### === Step 3: 거래 방식 ===

### ⑫ 거래 방법
- 라디오: 택배 / 직거래 / 둘 다 가능
- 필수 항목 (*)

### ⑬ 직거래 위치 (조건부)
- "직거래" 또는 "둘 다" 선택 시 표시
- 카카오맵 위치 선택 (주소 검색 + 지도 핀)
- 선택한 주소 텍스트 표시

### ⑭ 추가 조건
- 텍스트에리어 (최대 500자)
- 플레이스홀더: "추가 교환 조건이나 참고사항을 적어주세요"
- 선택 항목

### ⑮ 등록하기 버튼
- 필수 항목 모두 입력 시 활성화
- 탭 시: 최종 확인 → 등록 완료
- 수정 모드: "수정하기" 텍스트

## 데이터 필드

### API: POST /api/exchange (등록) / PUT /api/exchange/[id] (수정)

**Request Body:**

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| images | File[] | ❌ | 상품 이미지 (최대 5장) |
| title | string | ✅ | 상품명 (최대 50자) |
| work_id | string | ✅ | 작품 ID |
| character_id | string | ✅ | 캐릭터 ID |
| goods_type | string | ✅ | 굿즈 타입 |
| condition | string | ✅ | 상품 상태 (S/A/B/C) |
| condition_note | string | ❌ | C급 시 하자 설명 |
| quantity | number | ✅ | 보유 수량 (기본 1) |
| wanted_work_id | string | ❌ | 원하는 작품 ID |
| wanted_character_id | string | ❌ | 원하는 캐릭터 ID |
| wanted_goods_type | string | ❌ | 원하는 굿즈 타입 |
| wanted_any | boolean | ❌ | 아무거나 가능 여부 |
| cash_diff_enabled | boolean | ❌ | 차액 협의 가능 여부 |
| cash_diff_amount | number | ❌ | 차액 금액 |
| cash_diff_direction | string | ❌ | 차액 방향 (plus/minus) |
| trade_method | string | ✅ | 거래 방법 |
| meetup_lat | number | ❌ | 직거래 위도 |
| meetup_lng | number | ❌ | 직거래 경도 |
| meetup_address | string | ❌ | 직거래 주소 |
| extra_condition | string | ❌ | 추가 조건 |
