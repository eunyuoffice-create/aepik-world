## 2. 회원가입 & 인증 플로우

### 2-1. 회원가입

```
[앱/웹 진입]
    │
    ▼
[둘러보기] ─── 비로그인 상태로 매물 검색, 이벤트 열람 가능
    │
    │ (교환글 등록/공구 참여/채팅 등 시도 시)
    ▼
[회원가입 화면]
    │
    ├── [카카오 로그인] ───→ OAuth 인증 ───→ [닉네임 설정]
    │                                            │
    └── [이메일 회원가입] → 이메일+비밀번호 입력 → [닉네임 설정]
                                                    │
                                                    ▼
                                            [관심 카테고리 선택] (선택사항)
                                                    │
                                                    ├── 애니·만화·소설
                                                    ├── 아이돌·아티스트
                                                    ├── 배우·무대
                                                    └── 게임
                                                    │
                                                    ▼
                                            [가입 완료 → 홈으로 이동]
```

### 2-2. 본인인증 (Just-in-time)

```
[교환글 등록 또는 공구 참여 첫 시도]
    │
    ▼
[본인인증 필요 안내 바텀시트]
    │
    ├── [인증하기] → PortOne 본인인증 팝업
    │                    │
    │                    ├── 성공 → [인증 완료 뱃지 부여] → 원래 작업 계속
    │                    └── 실패 → 재시도 안내
    │
    └── [나중에] → 원래 화면으로 복귀 (기능 제한 유지)
```

## 3. 메인 홈 플로우

```
[홈 화면]
    │
    ├── [검색바 탭] ──────────────────→ 3-1. 검색 플로우
    │
    ├── [배너 캐러셀 탭] ─────────────→ 배너 상세 (배너 주인공 관련 매물 목록)
    │
    ├── [다가오는 이벤트 섹션]
    │       ├── [이벤트 카드 탭] ────→ 이벤트 상세 페이지
    │       └── [더보기] ────────────→ 이벤트 캘린더 페이지
    │
    ├── [인기 교환 섹션]
    │       ├── [상품 카드 탭] ──────→ 교환 상세 페이지
    │       └── [더보기] ────────────→ 교환 목록 페이지
    │
    ├── [진행 중 공구 섹션]
    │       ├── [공구 카드 탭] ──────→ 공구 상세 페이지
    │       └── [더보기] ────────────→ 공구 목록 페이지
    │
    ├── [구함글 섹션]
    │       ├── [구함글 탭] ─────────→ 구함글 상세
    │       └── [더보기] ────────────→ 구함글 목록 페이지
    │
    ├── [🔔 알림 아이콘] ────────────→ 알림 목록 페이지
    │
    └── [☰ 메뉴 아이콘] ────────────→ 사이드 메뉴
```

### 3-1. 검색 플로우

```
[검색바 탭] (홈 상단 검색바 / 하단 네비 돋보기)
    │
    ▼
[검색 입력 모드 — SRCH01-A]
    │
    ├── [인기 검색어] → 탭 시 즉시 검색 실행
    │
    ├── [최근 검색어] → 탭 시 재검색 / [X] 개별 삭제 / [전체 삭제]
    │
    └── [텍스트 입력]
            │
            ├── 2글자 이상 → 실시간 자동완성 (작품명 > 캐릭터명 우선, 300ms 디바운스)
            │     └── 자동완성 항목 탭 → 즉시 검색 실행
            │
            └── [검색 버튼 / 엔터]
                    │
                    ▼
                [검색 결과 모드 — SRCH01-B]
                    │
                    ├── [탭 바] 전체 / 교환 / 공구 / 구함 / 이벤트
                    │
                    ├── [필터 칩 바] 작품 ▾ | 캐릭터 ▾ | 필터 (아이콘)
                    │     │
                    │     └── [필터 아이콘 탭]
                    │             │
                    │             ▼
                    │         [필터 바텀시트 — SRCH01-C]
                    │             ├── 카테고리 (4대 칩)
                    │             ├── 굿즈 타입 (다중 선택)
                    │             ├── 상품 상태 (S/A/B/C급)
                    │             ├── 거래 방법 (택배/직거래/둘 다)
                    │             ├── 정렬 (최신/인기/가까운)
                    │             ├── 추가 옵션 (차액 협의 가능만)
                    │             │
                    │             ├── [초기화] → 모든 필터 리셋
                    │             └── [N건 보기] → 필터 적용 + 바텀시트 닫기
                    │
                    ├── [정렬] (최신순 / 인기순 / 가까운순)
                    │
                    ├── [결과 카드 탭] → 해당 상세 페이지 (EXCH02/GRPB02/WANT01/EVNT02)
                    │
                    └── [검색 결과 없음] → 빈 상태 + 추천 검색어 제안
```

## 4. 교환 플로우

### 4-1. 교환글 등록

```
[FAB(+) 버튼 또는 등록하기 버튼]
    │
    ▼
[거래 유형 선택] ── 교환 / 공구 / 구함
    │
    │ (교환 선택)
    ▼
[교환글 등록 폼 — Step 1: 보유 굿즈]
    ├── 사진 업로드 (최대 5장)
    ├── 작품 검색/선택
    ├── 캐릭터 선택
    ├── 굿즈 타입 선택
    ├── 상품 상태 (S/A/B/C급)
    ├── 보유 수량
    └── [다음]
        │
        ▼
[교환글 등록 폼 — Step 2: 원하는 굿즈]
    ├── 원하는 작품/캐릭터/굿즈타입
    ├── 원하는 수량
    ├── 차액 협의 가능 여부 (체크박스)
    │     └── (체크 시) 희망 차액 금액 입력
    └── [다음]
        │
        ▼
[교환글 등록 폼 — Step 3: 거래 방식]
    ├── 택배 / 직거래 / 둘 다
    │     └── (직거래 선택 시) 지도에서 위치 선택 + 설명 입력
    ├── 추가 조건 (텍스트, 선택)
    └── [등록하기]
        │
        ▼
[본인인증 체크] ── 미인증 시 → 2-2. 본인인증 플로우
    │
    ▼
[등록 완료]
    ├── 교환글 상세 페이지로 이동
    └── (백그라운드) 매칭 알고리즘 실행 → 매칭 시 양쪽 푸시 알림
```

### 4-2. 교환 매칭 & 진행

```
[매칭 알림 수신] (FCM 푸시)
    │
    ▼
[알림 탭] → 매칭된 교환글 상세 확인
    │
    ▼
[교환 상세 페이지]
    │
    ├── [💬 채팅] → 채팅방 개설/입장
    │       │
    │       ▼
    │   [채팅에서 조건 협의]
    │       ├── 교환 수량 확인
    │       ├── 차액 금액 협의 (있는 경우)
    │       ├── 거래 방식 확인 (택배/직거래)
    │       └── 양쪽 합의
    │           │
    │           ▼
    │       [교환 확정 버튼 클릭] (양쪽 모두)
    │           │
    │           ▼
    │       ── 택배 거래 ──              ── 직거래 ──
    │           │                           │
    │           ▼                           ▼
    │       [차액 정산]               [일정/장소 확정]
    │       (직접 송금)                     │
    │           │                           ▼
    │           ▼                       [만남 & 교환]
    │       [각자 발송]                     │
    │       (운송장 번호 입력)               ▼
    │           │                       [교환 완료 버튼]
    │           ▼                       (양쪽 모두)
    │       [수취 확인]                     │
    │       (양쪽 모두)                     │
    │           │                           │
    │           └──────────┬────────────────┘
    │                      │
    │                      ▼
    │              [교환 완료 처리]
    │                      │
    │                      ▼
    │              [후기 작성 요청 알림]
    │                      │
    │                      ▼
    │              [후기 작성] (7일 이내)
    │                      │
    │                      ├── 긍정 / 부정
    │                      └── 텍스트 후기 (선택)
    │
    └── [🔄 교환 제안] → 내 보유 굿즈 선택 → 교환 제안 전송
            │
            └── 상대방에게 교환 제안 알림 → 채팅방 자동 생성
```

### 4-3. 한쪽만 완료 확인 시

```
[A가 교환 완료 클릭]
    │
    ▼
[B에게 완료 확인 요청 알림]
    │
    ├── B가 5일 내 확인 → 정상 완료
    └── B가 5일간 미응답 → 자동 완료 처리
```

## 5. 공구 플로우

### 5-1. 일반 공구 등록

```
[거래 유형: 공구 선택]
    │
    ▼
[공구 유형 선택] ── 일반 공구 / 세트 공구
    │
    │ (일반 공구)
    ▼
[일반 공구 등록 폼]
    ├── 상품명
    ├── 상품 사진 (최대 5장)
    ├── 1인당 가격 (배송비 포함 여부)
    ├── 목표 인원
    ├── 최소 인원 (선택)
    ├── 마감일
    ├── 배송 예정일
    ├── 상세 설명
    └── [등록하기]
        │
        ▼
    [진행자 자격 체크] ── 교환 5회+ & 긍정 90%+
        │
        ├── 자격 충족 → 등록 완료
        └── 미충족 → "공구 진행자 자격이 부족합니다" 안내
```

### 5-2. 세트 공구(캐릭터 선점) 등록

```
[공구 유형: 세트 공구 선택]
    │
    ▼
[세트 공구 등록 폼 — Step 1: 기본 정보]
    ├── 상품명 (예: "하이큐 아크릴 스탠드 풀세트")
    ├── 작품 선택 (DB에서)
    ├── 세트 구매 수량 (예: 2박스)
    ├── 상품 사진
    ├── 마감일 / 배송 예정일
    └── [다음]
        │
        ▼
[세트 공구 등록 폼 — Step 2: 캐릭터 설정]
    ├── 캐릭터 목록 자동 로드 (작품 DB 기반)
    ├── 캐릭터 추가/삭제 가능
    ├── 캐릭터별 선점 가능 수량 설정
    ├── 캐릭터당 가격 입력
    └── [다음]
        │
        ▼
[세트 공구 등록 폼 — Step 3: 확인]
    ├── 전체 정보 요약
    └── [등록하기]
        │
        ▼
    [등록 완료]
```

### 5-3. 공구 참여 (일반)

```
[공구 상세 페이지]
    │
    ▼
[정보 확인]
    ├── 상품명, 가격, 마감일
    ├── 현재 참여자 수 / 목표 인원
    └── 진행자 프로필 + 신뢰도
        │
        ▼
    [공구 참여하기 버튼]
        │
        ▼
    [본인인증 체크] ── 미인증 시 → 인증 플로우
        │
        ▼
    [참여 확인 팝업]
        │
        ├── 확인 → 참여 완료 + 채팅방 입장
        └── 취소 → 돌아가기
```

### 5-4. 세트 공구 참여 (캐릭터 선점)

```
[세트 공구 상세 페이지]
    │
    ▼
[정보 확인]
    ├── 상품명, 세트 가격, 인당 가격
    ├── 진행도 (선점 수 / 전체 수)
    └── 캐릭터별 선점 현황
        │
        ▼
    [캐릭터 선택]
        ├── 선점 완료된 캐릭터: 비활성 (이름 + 선점자 표시)
        └── 선점 가능한 캐릭터: [선점하기] 버튼
            │
            ├── 수량 선택 (남은 수량 범위 내)
            └── [선택 완료]
                │
                ▼
            [공구 참여하기 버튼]
                │
                ▼
            [본인인증 체크]
                │
                ▼
            [참여 확인 팝업]
                ├── 선택 캐릭터: OO
                ├── 결제 금액: O,OOO원
                └── [확인] → 선점 완료 + 실시간 수량 업데이트
```

### 5-5. 공구 진행 (진행자)

```
[마감일 도달 또는 목표 달성]
    │
    ▼
[진행자에게 알림]
    │
    ├── 일반 공구: 목표 달성 → 구매 진행
    │               미달 → 자동 취소 or 진행자 선택
    │
    └── 세트 공구: 전체 선점 완료 or 마감일
            │
            ▼
        [진행자: 구매 진행]
            │
            ▼
        [진행자: 공지 업데이트] (구매 완료, 배송 시작 등)
            │
            ▼
        [진행자: 각 참여자에게 발송]
            │
            ▼
        [참여자: 수령 확인]
            │
            ▼
        [후기 작성]
```

## 7. 이벤트 캘린더 플로우

### 7-1. 이벤트 열람

```
[이벤트 탭 (하단 네비)] 또는 [홈 > 다가오는 이벤트 > 더보기]
    │
    ▼
[이벤트 캘린더 페이지]
    │
```
