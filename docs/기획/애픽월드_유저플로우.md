## 2. 회원가입 & 인증 플로우

### 2-0. 로그인 [AUTH01]

```
[앱/웹 진입]
    │
    ▼
[AUTH01-A 로그인 메인]
    │
    ├── [카카오 로그인] → 카카오 OAuth 팝업/리다이렉트
    │       ├── 기존 회원 → JWT 발급 → HOME01
    │       └── 신규 회원 → AUTH02 (회원가입)
    │
    ├── [이메일 로그인] → AUTH01-B 이메일 로그인 모드
    │       ├── 이메일 + 비밀번호 입력
    │       ├── [로그인 버튼] → API 호출
    │       │     ├── 성공 → JWT 발급 → HOME01
    │       │     ├── 실패 → 인라인 에러 "이메일 또는 비밀번호가 일치하지 않습니다"
    │       │     └── 5회 초과 실패 → 30분 계정 잠금
    │       ├── [비밀번호를 잊으셨나요?] → AUTH01-C 비밀번호 찾기
    │       │     ├── 이메일 입력 → 재설정 링크 발송 (유효 1시간)
    │       │     └── 성공 → "메일이 전송되었습니다" 안내
    │       └── [회원가입] → AUTH02
    │
    ├── [로그인 없이 둘러보기] → 비로그인 세션 → HOME01 (검색/열람만 가능)
    │
    └── [약관 안내] → 이용약관/개인정보처리방침 링크 → STAT01
```

**비로그인 상태 기능 제한 시 → 로그인 유도 바텀시트 [AUTH01-D]**
```
[교환글 등록/공구 참여/채팅/찜/신고 시도]
    │
    ▼
[AUTH01-D 로그인 유도 바텀시트]
    ├── "로그인이 필요한 기능입니다" 안내
    ├── [카카오 로그인] → OAuth 진행
    ├── [이메일 로그인] → AUTH01-B
    └── [나중에] → 바텀시트 닫기, 현재 화면 유지
```

**특수 계정 상태 처리**
```
[로그인 시도]
    ├── 휴면 계정 (1년 미접속) → 휴면 해제 안내 팝업 → 본인인증 후 복구
    ├── 정지 계정 → 정지 안내 + 사유 + 해제일 표시, 기능 차단
    └── OAuth 에러 → 에러 페이지 + "다시 시도" / "이메일로 로그인" 안내
```

### 2-1. 회원가입 [AUTH02]

```
[AUTH01에서 진입]
    ├── 카카오 신규 → Step 1 → (Step 2 건너뜀) → Step 3 → Step 4
    └── "회원가입" 링크 → Step 1 → Step 2 → Step 3 → Step 4
        │
        ▼
[AUTH02-A Step 1: 약관 동의]
    ├── [전체 동의] → 모든 체크박스 일괄 선택/해제
    ├── [필수] 이용약관 동의 + "보기" (STAT01)
    ├── [필수] 개인정보 수집 및 이용 동의 + "보기"
    ├── [선택] 마케팅 정보 수신 동의
    ├── [필수] 만 14세 이상 확인
    └── [다음] → 필수 항목 모두 체크 시 활성화
        │
        ▼
[AUTH02-B Step 2: 계정 정보 입력] (이메일 가입만, 카카오 경유 시 건너뜀)
    ├── 이메일 입력 → 형식 검증 + 자동 중복 확인
    ├── 비밀번호 입력 (8자+, 영문+숫자+특수문자) → 강도 바 (약함/보통/강함)
    ├── 비밀번호 확인 → 일치 여부 실시간 표시
    └── [다음] → 모든 유효성 통과 시 활성화
        │
        ▼
[AUTH02-C Step 3: 프로필 설정]
    ├── 프로필 이미지 (선택, 80px 원형, 미등록 시 기본 아바타)
    ├── 닉네임 입력 (2~12자, 한글/영문/숫자)
    │     ├── debounce 500ms 후 중복 확인
    │     └── 금칙어 필터 (비속어, 관리자 사칭 차단)
    └── [다음] → 닉네임 유효 + 중복확인 통과 시 활성화
        │
        ▼
[AUTH02-D Step 4: 관심 카테고리 선택] (선택사항)
    ├── 애니/만화/소설 (Purple) | 아이돌/아티스트 (Pink)
    ├── 배우/무대 (Amber) | 게임 (Cyan)
    ├── 복수 선택 가능, 미선택도 허용
    ├── [시작하기] → 회원가입 완료
    └── [나중에 설정할게요] → 카테고리 미선택 상태로 완료
        │
        ▼
[AUTH02-E 가입 완료]
    ├── "{닉네임}님, 환영합니다!" 메시지
    ├── [시작하기] → JWT 발급 → HOME01
    └── (24시간 내 웰컴 푸시 알림)
```

### 2-2. 본인인증 (Just-in-time)

```
[교환글 등록 또는 공구 참여 첫 시도]
    │
    ▼
[본인인증 필요 안내 바텀시트]
    │
    ├── [인증하기] → PortOne 본인인증 팝업
    │                    │
    │                    ├── 성공 → [인증 완료 뱃지 부여] → 원래 작업 계속
    │                    └── 실패 → 재시도 안내
    │
    └── [나중에] → 원래 화면으로 복귀 (기능 제한 유지)
```

### 3-1. 검색 플로우

```
[검색바 탭] (홈 상단 검색바 / 하단 네비 돋보기)
    │
    ▼
[검색 입력 모드 — SRCH01-A]
    │
    ├── [인기 검색어] → 탭 시 즉시 검색 실행
    │
    ├── [최근 검색어] → 탭 시 재검색 / [X] 개별 삭제 / [전체 삭제]
    │
    └── [텍스트 입력]
            │
            ├── 2글자 이상 → 실시간 자동완성 (작품명 > 캐릭터명 우선, 300ms 디바운스)
            │     └── 자동완성 항목 탭 → 즉시 검색 실행
            │
            └── [검색 버튼 / 엔터]
                    │
                    ▼
                [검색 결과 모드 — SRCH01-B]
                    │
                    ├── [탭 바] 전체 / 교환 / 공구 / 구함 / 이벤트
                    │
                    ├── [필터 칩 바] 작품 ▾ | 캐릭터 ▾ | 필터 (아이콘)
                    │     │
                    │     └── [필터 아이콘 탭]
                    │             │
                    │             ▼
                    │         [필터 바텀시트 — SRCH01-C]
                    │             ├── 카테고리 (4대 칩)
                    │             ├── 굿즈 타입 (다중 선택)
                    │             ├── 상품 상태 (S/A/B/C급)
                    │             ├── 거래 방법 (택배/직거래/둘 다)
                    │             ├── 정렬 (최신/인기/가까운)
                    │             ├── 추가 옵션 (차액 협의 가능만)
                    │             │
                    │             ├── [초기화] → 모든 필터 리셋
                    │             └── [N건 보기] → 필터 적용 + 바텀시트 닫기
                    │
                    ├── [정렬] (최신순 / 인기순 / 가까운순)
                    │
                    ├── [결과 카드 탭] → 해당 상세 페이지 (EXCH02/GRPB02/WANT01/EVNT02)
                    │
                    └── [검색 결과 없음] → 빈 상태 + 추천 검색어 제안
```

## 4. 교환 플로우

### 4-1. 교환글 등록

```
[FAB(+) 버튼 또는 등록하기 버튼]
    │
    ▼
[거래 유형 선택] ── 교환 / 공구 / 구함
    │
    │ (교환 선택)
    ▼
[교환글 등록 폼 — Step 1: 보유 굿즈]
    ├── 사진 업로드 (최대 5장)
    ├── 작품 검색/선택
    ├── 캐릭터 선택
    ├── 굿즈 타입 선택
    ├── 상품 상태 (S/A/B/C급)
    ├── 보유 수량
    └── [다음]
        │
        ▼
[교환글 등록 폼 — Step 2: 원하는 굿즈]
    ├── 원하는 작품/캐릭터/굿즈타입
    ├── 원하는 수량
    ├── 차액 협의 가능 여부 (체크박스)
    │     └── (체크 시) 희망 차액 금액 입력
    └── [다음]
        │
        ▼
[교환글 등록 폼 — Step 3: 거래 방식]
    ├── 택배 / 직거래 / 둘 다
    │     └── (직거래 선택 시) 지도에서 위치 선택 + 설명 입력
    ├── 추가 조건 (텍스트, 선택)
    └── [등록하기]
        │
        ▼
[본인인증 체크] ── 미인증 시 → 2-2. 본인인증 플로우
    │
    ▼
[등록 완료]
    ├── 교환글 상세 페이지로 이동
    └── (백그라운드) 매칭 알고리즘 실행 → 매칭 시 양쪽 푸시 알림
```

    │       ▼

    │       ├── 교환 수량 확인
    │       ├── 차액 금액 협의 (있는 경우)
    │       ├── 거래 방식 확인 (택배/직거래)
    │       └── 양쪽 합의
    │           │
    │           ▼
    │       [교환 확정 버튼 클릭] (양쪽 모두)
    │           │
    │           ▼
    │       ── 택배 거래 ──              ── 직거래 ──
    │           │                           │
    │           ▼                           ▼
    │       [차액 정산]               [일정/장소 확정]
    │       (직접 송금)                     │
    │           │                           ▼
    │           ▼                       [만남 & 교환]
    │       [각자 발송]                     │
    │       (운송장 번호 입력)               ▼
    │           │                       [교환 완료 버튼]
    │           ▼                       (양쪽 모두)
    │       [수취 확인]                     │
    │       (양쪽 모두)                     │
    │           │                           │
    │           └──────────┬────────────────┘
    │                      │
    │                      ▼
    │              [교환 완료 처리]
    │                      │
    │                      ▼
    │              [후기 작성 요청 알림]
    │                      │
    │                      ▼
    │              [후기 작성] (7일 이내)
    │                      │
    │                      ├── 긍정 / 부정
    │                      └── 텍스트 후기 (선택)
    │
    └── [`Repeat` 교환 제안] → 내 보유 굿즈 선택 → 교환 제안 전송
            │
            └── 상대방에게 교환 제안 알림 → 채팅방 자동 생성

### 4-4. 교환 제안 관리 [EXCH05]

```
[교환 신청함 진입] (마이페이지 > 교환 신청함 / 알림 탭)
    │
    ▼
[EXCH05 교환 신청함 메인]
    │
    ├── [보낸 제안 탭]
    │     │
    │     ├── 제안 카드 목록 (상태별 필터: 전체/대기중/수락/거절/만료)
    │     │     ├── 대기중(pending) → [취소] 가능
    │     │     ├── 수락됨(accepted) → 채팅방 이동
    │     │     ├── 거절됨(rejected) → 거절 사유 확인
    │     │     ├── 취소됨(cancelled) → 상태 표시만
    │     │     └── 만료(expired) → "7일 미응답 자동 만료"
    │     │
    │     └── [제안 카드 탭] → 해당 교환글 상세(EXCH02)
    │
    └── [받은 제안 탭]
          │
          ├── 교환글별 그룹핑 (내 교환글 > 받은 제안 리스트)
          │     ├── 제안자 정보 (닉네임, 신뢰등급, 프로필)
          │     ├── 제안 굿즈 정보 (이미지, 상태, 조건)
          │     └── 제안 메시지 (있는 경우)
          │
          ├── [수락] 버튼
          │     │
          │     ▼
          │   [수락 확인 팝업] "이 제안을 수락하면 다른 대기 중인 제안은 자동 거절됩니다"
          │     │

          │     └── [취소] → 돌아가기
          │
          └── [거절] 버튼
                │
                ▼

    │     ├── 알림 설정 (7개 토글 + 방해금지 시간)
    │     │     └── 마스터 OFF 시 하위 비활성
    │     ├── 관심 카테고리 변경
    │     ├── 계정 관리
    │     │     ├── 비밀번호 변경 (바텀시트)
    │     │     ├── 소셜 연동 상태
    │     │     ├── 키워드 알림 설정 (/my/keywords) ─── USER03-B
    │     │     │     ├── 키워드 추가 (최대 20개, 2~30자)
    │     │     │     ├── 태그 칩 리스트 + X 삭제
    │     │     │     └── 칩 탭 → SRCH01 검색
    │     │     ├── 로그아웃 → 확인 팝업 → AUTH01
    │     │     └── 회원 탈퇴 ─── USER03-C
    │     │           ├── Step 1: 탈퇴 안내 (주의사항 4개)
    │     │           ├── Step 2: 탈퇴 사유 (라디오 5개)
    │     │           ├── Step 3: 본인 확인 (비밀번호/카카오)
    │     │           └── Step 4: 탈퇴 완료 → 3초 후 HOME01
    │     └── 정보 (약관/정책/라이선스/버전)
    │
    └── [고객센터]
          ├── FAQ
          ├── 1:1 문의
          └── 신고 내역 확인

[타인 프로필] ─── USER01-C (/user/[userId])
    │
    ├── 프로필 영역 (이미지+닉네임+인증뱃지+신뢰지표+가입일)
    ├── 등록 매물 탭 (교환글/공구/구함글) → 2열 그리드
    ├── 받은 후기
    └── [더보기] → 신고하기 바텀시트
```

## 12. 신고 플로우

```
[신고 대상 (교환글/공구/구함글/이벤트/채팅/사용자 프로필)]
    │

    │

[신고 바텀시트]
    ├── 신고 사유 선택
    │     ├── 교환 미발송
    │     ├── 공구 먹튀
    │     ├── 허위 매물
    │     ├── 욕설/비하
    │     ├── 사진 도용
    │     └── 기타
    ├── 증거 첨부 (스크린샷, 사진)
    ├── 상세 설명 (텍스트)
    └── [신고 접수]
        │
        ▼
    [접수 완료 안내]
        │
        └── 48시간 내 1차 검토 → 결과 알림
```

### 12-1. 신고 결과 처리

```
[관리자 검토 완료] (48시간 내)
    │
    ▼
[처리 결과 알림 발송]
    │
    ├── [제재 조치] → 신고 대상자에게 알림
    │     ├── 경고 → "커뮤니티 가이드 위반으로 경고가 부여되었습니다" + 사유
    │     ├── 매물 삭제/숨김 → "등록하신 매물이 정책 위반으로 삭제되었습니다"
    │     ├── 일시 정지 (7일/30일) → "계정이 N일간 이용 정지되었습니다" + 해제일
    │     ├── 공구 강제 종료 → 참여자 전원에게 "공구가 관리자에 의해 종료되었습니다" 알림
    │     └── 영구 정지 → "계정이 영구 정지되었습니다" + 동일 번호 재가입 차단
    │
    └── [반려 (정상)] → 신고자에게 "검토 결과, 위반 사항이 확인되지 않았습니다" 알림

[제재 단계 누적]
    ├── 교환 미발송: 경고 → 7일 정지 → 영구 정지
    ├── 공구 먹튀: 즉시 영구 정지 + 번호 차단
    ├── 허위 매물: 매물 삭제 + 경고
    ├── 욕설/비하: 경고 → 30일 정지
    └── 사진 도용: 매물 삭제 + 경고

[신고 내역 확인] (마이페이지 > 고객센터 > 신고 내역)
    ├── 내가 접수한 신고 목록
    ├── 각 신고 상태 (접수/검토중/완료/반려)
    └── 처리 결과 상세 확인
```

## 13. 후기 작성 & 이의제기 플로우

### 13-1. 후기 작성

```
[교환/공구 완료 확정]
    │
    ▼
[후기 작성 요청 알림] (완료 직후 + D+3 리마인더)
    │
    ├── 알림 탭 → 후기 작성 바텀시트
    └── 마이페이지 > 교환 내역 > [후기 작성하기]
          │
          ▼
      [후기 작성 바텀시트]
          ├── 거래 상대 정보 표시
          ├── 평가: [긍정] / [부정] 선택
          ├── 텍스트 후기 입력 (선택, 최대 200자)
          └── [작성 완료]
                │
                ├── 확인 → 후기 등록 완료 + 상대방에게 알림
                │          (작성 후 수정/삭제 불가 안내)
                └── 7일 이내 미작성 → 작성 기한 만료 (추후 작성 불가)
```

### 13-2. 이의 제기

```
[받은 후기 목록] (마이페이지 > 받은 후기 — USER02-E)
    │
    ├── [부정 후기 확인]
    │     └── [이의 제기] 버튼 (후기 작성 후 72시간 이내만 노출)
    │           │
    │           ▼
    │       [이의 제기 바텀시트]
    │           ├── 이의 사유 선택
    │           │     ├── 허위 내용
    │           │     ├── 거래와 무관한 내용
    │           │     ├── 욕설/비하
    │           │     ├── 광고성
    │           │     └── 직접 입력
    │           ├── 상세 설명 (필수, 최대 500자)
    │           └── [이의 제기 접수]
    │                 │
    │                 ▼
    │             [접수 완료] → "72시간 내 검토 후 결과를 알려드리겠습니다"
    │
    └── [이의 제기 결과 수신]
          ├── 후기 삭제 → "이의가 인정되어 해당 후기가 삭제되었습니다"
          └── 이의 반려 → "검토 결과, 정당한 후기로 판단되었습니다"
```
