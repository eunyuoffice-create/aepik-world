# 애픽 월드(Aepik World) 개인정보처리방침 (초안)

> **문서 상태**: 초안 — 법률 전문가 검토 필수
> **작성일**: 2026-03-05
> **적용 법률**: 개인정보보호법, 정보통신망 이용촉진 및 정보보호 등에 관한 법률

---

애픽 월드(이하 "회사")은 이용자의 개인정보를 중요시하며, 개인정보보호법 및 정보통신망법을 준수합니다. 본 개인정보처리방침을 통해 이용자의 개인정보가 어떻게 수집·이용·보관·파기되는지 안내합니다.

---

## 1. 수집하는 개인정보 항목

### 1-1. 회원가입 시 수집 항목

| 수집 항목 | 수집 목적 | 필수/선택 |
|-----------|----------|----------|
| 이메일 주소 | 계정 식별, 로그인, 공지 발송 | 필수 |
| 비밀번호 (이메일 가입 시) | 계정 보안 | 필수 |
| 카카오 계정 정보 (카카오 가입 시) | OAuth 로그인 | 필수 |
| 휴대폰 번호 | 본인인증, 1인 1계정 관리 | 조건부(첫 거래성 액션 시) |
| 본인인증 정보 (성명, 생년월일, 성별) | 본인 확인, 미성년자 보호 | 조건부(첫 거래성 액션 시) |
| 닉네임 | 서비스 내 표시 | 필수 |
| 프로필 이미지 | 서비스 내 표시 | 선택 |
| 관심 카테고리 (애니/아이돌/게임) | 맞춤 알림 제공 | 선택 |

### 1-2. 서비스 이용 과정에서 자동 수집되는 항목

| 수집 항목 | 수집 목적 |
|-----------|----------|
| IP 주소 | 부정 이용 방지, 접속 기록 관리 |
| 기기 정보 (OS, 브라우저, 기기 식별자) | 서비스 최적화, 푸시 알림 |
| 쿠키 | 로그인 유지, 서비스 이용 편의 |
| 서비스 이용 기록 (접속 일시, 검색어, 클릭 로그) | 서비스 개선, 통계 분석 |
| FCM 토큰 | 푸시 알림 발송 |

### 1-3. 교환·공구 이용 시 추가 수집 항목

| 수집 항목 | 수집 목적 | 비고 |
|-----------|----------|------|
| 교환/공구 내역 | 거래 이력 관리, 분쟁 해결 | 자동 기록 |
| 교환 제안 데이터 (제안 메시지, 상태, 거래 구조, 수량, 차액, manual_offer 이미지/상태/수량) | 제안 기능 제공, 이력 관리, 분쟁 해결 | 교환 제안 기능 이용 시 수집 |
| 오늘 직교환 메타 (이벤트명, 시간대, 대략 위치/랜드마크) | 당일 직교환 매칭 지원, 안전 가이드 준수 | 정확한 주소는 수집하지 않음 |
| 채팅 내용 | 분쟁 시 증거 확보 | 암호화 저장 |
| 배송 정보 (주소) | 교환 물품 배송 | 이용자 간 직접 공유, 회사 미보관 |

## 2. 개인정보의 수집·이용 목적

| 목적 | 상세 |
|------|------|
| 회원 관리 | 회원가입, 본인인증(거래성 액션 시), 1인 1계정 관리, 미성년자 보호 |
| 서비스 제공 | 교환 매칭, 공구 모집, 동행 구함 매칭, 키워드 알림 |

| 서비스 개선 | 이용 통계 분석, 서비스 품질 향상 |
| 분쟁 해결 | 이용자 간 분쟁 시 기록 제공, 수사기관 협조 |

회원 탈퇴 시까지 보유합니다.

### 3-3. 휴면 계정

1년 이상 미접속 시 휴면 전환되며, 개인정보는 별도 분리 보관됩니다. 휴면 전환 30일 전 이메일/푸시로 사전 안내합니다.

## 4. 개인정보의 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 다만, 다음의 경우에는 예외로 합니다.

| 제공 상황 | 제공 대상 | 제공 항목 | 근거 |
|-----------|----------|----------|------|
| 이용자의 동의가 있는 경우 | 교환 상대방 | 연락처 (채팅 내 직접 공유) | 이용자 동의 |
| 분쟁 해결 | 분쟁 상대방 | 연락처 (이용자 동의 하에) | 이용자 동의 |
| 법률상 의무 | 수사기관, 법원 | 법원 영장 등에 명시된 항목 | 관계 법령 |

## 5. 개인정보 처리 위탁

회사는 서비스 제공을 위해 다음과 같이 개인정보 처리를 위탁합니다.

| 수탁 업체 | 위탁 업무 | 보유 기간 |
|-----------|----------|----------|
| PortOne(아임포트) | 휴대폰 본인인증 | 인증 완료 즉시 파기 |
| Amazon Web Services (AWS) | 이미지 저장 (S3) | 회원 탈퇴 시 삭제 |
| Google Firebase | 푸시 알림 발송 (FCM), 채팅 (Realtime DB) | 서비스 이용 중 |
| 카카오 | 소셜 로그인 (OAuth) | 연동 해제 시 삭제 |

---

## 6. 개인정보의 파기

1. 보유 기간 경과 또는 처리 목적 달성 시 지체 없이 파기합니다.
2. 파기 방법:
   - 전자적 파일: 복구 불가능한 방법으로 영구 삭제
   - 출력물: 분쇄 또는 소각
3. 제재 이력의 경우, 재가입 방지를 위해 휴대폰 번호의 **해시값만** 보관하며 원본은 즉시 파기합니다.

## 7. 이용자의 권리와 행사 방법

이용자(또는 법정대리인)는 다음의 권리를 행사할 수 있습니다.

| 권리 | 행사 방법 | 처리 기한 |
|------|----------|----------|
| 개인정보 열람 요청 | 서비스 내 설정 또는 고객센터 | 10일 이내 |
| 개인정보 정정 요청 | 서비스 내 프로필 수정 | 즉시 |
| 개인정보 삭제 요청 | 회원 탈퇴 또는 고객센터 | 10일 이내 |
| 개인정보 처리 정지 요청 | 고객센터 | 10일 이내 |
| 동의 철회 | 서비스 내 설정 | 즉시 |

## 8. 개인정보 보호 조치

### 8-1. 기술적 조치

- 개인정보 암호화 저장 (비밀번호 bcrypt 해싱, 민감 정보 AES-256 암호화)
- SSL/TLS 통신 암호화
- 접근 권한 관리 및 접근 기록 관리
- 보안 프로그램 설치 및 주기적 업데이트

### 8-2. 관리적 조치

- 개인정보 취급자 최소화 (1인 운영 기준 운영자만 접근)
- 개인정보 보호 교육 실시
- 내부 관리계획 수립 및 시행

---

## 9. 쿠키 사용

1. 회사는 이용자의 편의를 위해 쿠키를 사용합니다.
2. 쿠키의 사용 목적: 로그인 유지, 방문 기록 분석
3. 이용자는 웹 브라우저 설정을 통해 쿠키 저장을 거부할 수 있으며, 이 경우 일부 서비스 이용에 제한이 있을 수 있습니다.

---

## 10. 만 14세 미만 아동의 개인정보

1. 회사는 만 14세 미만 아동의 회원가입 시 법정대리인의 동의를 받습니다.
2. 법정대리인은 아동의 개인정보에 대해 열람, 정정, 삭제를 요청할 수 있습니다.
